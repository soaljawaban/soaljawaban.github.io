17. Persilangan antara sapi jantan berambut hitam berbadan besar (HhBb)dengan sapi betina berambut putih berbadan besar (hhBb).a.Buatlah diagram persilangannya dan tentukan perbandingan fenotif pada keturunannya(F1)!b.Jika sapi berambut putih berbadan besar merupakan sapi unggul,berapa persen kemungkinan munculnya keturunan unggul tersebut?Tolongg dengan caranya ya:v)​


Jawaban:

a. bagan persilangan dan perbandingan fenotipe F1

P = HhBb x hhBb

G = HB, Hb, hB, hb x hB, hb

F1 = HhBB, HhBb, hhBB, hhBb, HhBb, Hhbb, hhBb, hhbb

Genotipe F1 = HhBB, HhBb, hhBB, hhBb, HhBb, Hhbb, hhBb, hhbb

Fenotipe F1 = rambut hitam badan besar (HhBB, HhBb, HhBb), rambut hitam badan kecil (Hhbb), rambut putih badan besar (hhBB, hhBb, hhBb), dan rambut putih badan kecil (hhbb).

Perbandingan Fenotipe = ambut hitam badan besar : rambut hitam badan kecil : rambut putih badan besar : dan rambut putih badan kecil = 3 : 1 : 3 : 1

b. persentase sapi rambut putih badan besar

% = jumlah perbandingan rambut putih badan besar/jumlah perbandingan seluruhnya x 100%

% = 3/8 x 100%

% = 37,5%

25. di silangkan sapi jantan berbulu hitam berbadan besar (HhBb) dengan sapi betina berbulu putih berbadan besar (hhBb) maka presentase genotif sapi berbulu putih berbadan besar hasil keturannya adalah? pakai cara!!!​


untuk persilangannya, ketahui dulu genotipenya

H=dominan (hitam), h=Resesif (Putih)

B=dominan (Besar), b=resesif (kecil)

jadi persilangannya seperti ini

HhBb >< hhBb              HhBB=hitam Besar         jumlah hitam besar=3

HB          hB                   HhBb=hitam besar         jumlah putih besar=3

Hb           hb                  hhBB=putih besar          jumlah hitam kecil=1

hB                                  hhBb=Putih Besar          jumlah putih kecil=1

hb                                   HhBb=Hitam besar

                                      Hhbb=Hitam Kecil

                                      hhBb=Putih besar

                                      hhbb=putih kecil

Jadi perbandingannya adalah 3:3:1:1 dengan presentase putih badan besar nantinya dikali dengan hasil bagi dari 100 % dengan jumlah total perbandingan yaitu

100% : 8 = 12,5 %

maka dikalikan perbandingannya masing-masing

Hitam besar ⇒ 3 X 12,5 = 37,5 %

Putih besar  ⇒ 3 X 12,5 = 37,5 %

hitam kecil ⇒ 1 X 12,5 = 12,5 %

putih kecil ⇒ 1 X 12,5 = 12,5 %

jadi jawabannya untuk presentasi putih badan besar adalah 37,5 %
