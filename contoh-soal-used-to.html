1. jelaskan tentang kalimat used to, be used to,get used to dan contoh kalimatnya.


Used to = dulu
Contoh = I used to go to my friends house, but now I dont
Be used to = kebiasaan
Contoh = I am used to this situation
Get used to = terbiasa
Contoh = I get used to be annoyed because of loud music1.used to digunakan untuk menunjukkan 
contohnya : dewi used to live in Malang
2.be used to di gunakan untuk mengucapkan sesuatu yang normal maupun sesuatu yang tidak biasanya.
contohnya : i'm used to living room
3. get used digunakan untuk mengungkapkan sesuatu dalam proses menjadi normal atau kini menjadi biasa.
contohnya : he doesn't like that big town, but he'll get used to it

22. apa arti dari used to dan bagaimana cara membedakan dengan be used to dan berikan contohnya


used to = kata bantu kerja yang digunakan untuk masa lampau dimana berbeda faktanya dengan sekarang.

formula = used to + infinitive

examples =
- I used to sleep with my mother when I was a baby
(fact : I dont sleep with my mother anymore because I am not a baby)

- she used to use an uniform when she was senior high school
( fact :she doesnt use uniform)kalau used to itu untuk lampau sedangkan be used to untuk mengekspresikan sesuatu yang tidak asing

39. perbedaan use to da used to? dan contoh kalimatnya dan rumusnya bagaimana?


use menggunakan dan used to pemggunaan yg bertujuan untuk sesuatuperbedaannya adalah pada bentuk kalimatnya (mempunyai arti yang sama yaitu terbiasa. pernah atau dulunya)
contoh bentuk kalimat positive: she used to listen to some music (dulunya dia mendengarkan musik atau dia terbiasa mendengar musik).sedang dalam bentuk kalimat negative menjadi: she didn't use to listen to some music (dulunya dia tidak mendengarkan musik dst)
rumus untuk kalimat positive: subject + used to + V1
rumus untuk kalimat negative: subject + didn't + use to + V1



40. apa pengertian used to,get used to,be used to,compliments,certenty and uncertenty,bargaining


used to adalah digunakan untuk, get used adalah untuk digunakan, be used adalah  digunakan, certainty adalah pemastian trhdap sesuatu, bergaining adalah tawar menawar

used to --> terbiasa
He used to eat vegetables since he was 5 years old
get used to --> menjadi terbiasa
He get used to the environment fastly.
be used to --> digunakan untuk
It will be used to make a dress.
compliments --> pujian
certainty --> kepastian
uncertainty --> ketidakpastian
bargaining --> menawar.
