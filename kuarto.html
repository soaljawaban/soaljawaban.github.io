2. Apa kegunaan kertas kuarto


Kertas kuarto atau sering dikenal dengan kertas a4 adalah kertas yang biasa digunakan di Indonesia untuk makalh, laporan, skripsi, karya ilmiah, dll.

Ukuran kertas ini yang paling banyak digunakan dibandingkan ukuran kertas lainnya. Di tempat fotocopy atau print, pasti menyediakan ukuran kertas ini. Ukuran kertas A4 sekitar 8,27 x 11,69 inchi atau 21 x 29,7 cm atau 210 x 297 mm.

Sebuah usb disk berkapasitas 16GB,jika 1 halaman kuarto dapat menampung 1500 huruf dan 1 hurif=8bit berapa halam kuarto dapat disimpan di usb disk tersebutJawabanPendahuluan

Ssistem satuan Standar Internasional (SI) untuk data.  

1 Giga Byte = 1000 Mega Byte  

1 Mega Byte = 1000 kilo Byte  

1 kilo Byte = 1000 Byte

1 Byte = 8 bit

Ada sistem ukuran yang berbeda yaitu

1 Mega Byte = 1024 kilo Byte  

1 kilo Byte = 1024 Byte

Namun sering kali produsen perangkat komputer tidak menggunakan sistem ukuran ini tapi menggunakan sistem pengukuran Internasional.


Pembahasan

1 halaman kuarto dapat menampung 1500 huruf.

Maka dapat kita simpulkan, bahwa 1 halaman kuarto maksimal memiliki ukutan

1500 hurus, 1 huruf 1 byte = 1500 Byte = 1,5 kilo Byte

Ukuran kapasitas USB disk adalah 16 Giga Byte

Maka ukuran USB disk dalam kilo Byte adalah 16 x 1000 x 1000 = 16.000.000 kB

Kita bisa hitung banyak halaman yang bisa disimpan dalam USB dengan cara sebagai berikut

Jumlah halaman maksimal = Ukuran USB / ukuran 1 halaman

= 16.000.000 / 1,5 = 10.666.666 halaman


Kesimpulan

Sebuah usb disk berkapasitas 16GB,jika 1 halaman kuarto dapat menampung 1500 huruf dan 1 hurif=8bit berapa halam kuarto dapat disimpan di usb disk tersebut : 10.666.666 halaman kuarto.
