3. Diketahui sudut PRS=3x° dan sudut QRS=(5x 20)°. tentukan nilai x , besar sudut QRS dan pelurus sudut QRS!

Nilai x = 20Sudut QRS = 120°Pelurus sudut QRS = 60°penjelasan dengan langkah-langkah

Diketahui :

Sudut PRS = 3x°Sudut QRS = (5x+20)°

Ditanya :

Nilai x ?

Sudut QRS ?

Pelurus sudut QRS ?

Jawab :

●Soal Pertama

Sudut pelurus 180°

[tex] \rm \angle PRS + \angle QRS = 180^o[/tex]

[tex] \rm 3x^o +(5x+20)^o=180^o[/tex]

[tex] \rm 3x +5x+20=180[/tex]

[tex] \rm 8x = 180-20[/tex]

[tex] \rm 8x = 160[/tex]

[tex] \rm x = 160\div 8[/tex]

[tex] \bf x = 20[/tex]

[tex]\\[/tex]

●Soal Kedua

[tex] \rm \angle QRS = (5x+20)^o[/tex]

[tex] \rm \angle QRS = (5(20)+20)^o[/tex]

[tex] \rm \angle QRS = (100+20)^o[/tex]

[tex] \bf \angle QRS = 120^o[/tex]

[tex]\\[/tex]

●Soal Ketiga

Pelurus sudut dari QRS adalah sudut PRS :

[tex] \rm \angle PRS = 3x^o[/tex]

[tex] \rm \angle PRS = 3(20)^o[/tex]

[tex] \bf \angle PRS=60^o[/tex]

21. Pada gambar berikut, besar QRS adalah ...


Pada gambar berikut, besar sudut QRS adalah A. 118°


Pembahasan

Segi empat tali busur dalam lingkaram memiliki ciri yang unik. Sudah jelas, karena berbentuk segiempat maka jumlah seluruh sudutnya adalah 360°. Eits, gak itu aja, sudut-sudut yang saling berhadapan jika dijumlahkan nilainya adalah 180°. Jika sesuai dengan soal, maka berlaku.

<PSR + <PQR = 180°<SPQ + <QRS = 180°

Naah, dari soal dapat kita ketahui bahwa

<PSR = 7x°

<PQR = 5x°

<SPQ = 3x° + 17°


Ditanya

Besar sudut QRS (<QRS)


Langkah cara

Kita gunakan persamaan pertama yang telah kita bahas diatas untuk menentukan nilai x nya.

<PSR + <PQR = 180°

180° = <PSR + <PQR

180° = 7x° + 5x°

180° = 12x°

x = 180/12

x = 15


Setelah nilai x nya diketahui, masukkan nilai x kedalam sudut SPQ.

<SPQ = 3x° + 17°

<SPQ = 3(15)° + 17°

<SPQ = 45° + 17°

<SPQ = 62°


Mencari besar sudut QRS

<SPQ + <QRS = 180°

<QRS = 180° - <SPQ

<QRS = 180° - 62°

<QRS = 118°


Jadi, besar sudut QRS adalah A. 118°
