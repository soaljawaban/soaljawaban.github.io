Soal terkait kalimat interogatif aktif ke pasif dalam berbagai tenses ya. Polanya adalah:

⁕ Simple Present

Is/Am/Are + S + V3 + ...?

⁕ Present Continuous

Is/Am/Are + S + being + V3 + ...?

⁕ Simple Past

Was/Were + S + V3 + ...?

⁕ Simple Present Menggunakan Modals

Modals (can/will) + S + be + V3 + ...?

⁕ Present Perfect

Have/Has + S + been + V3 + ...?

Pola kalimat positif:

⁕ Simple Present Menggunakan Modals

S + Modals + be + V3

⁕ Simple Past

S + Was/were + V3

⁕ Simple present

S + Is/Am/Are + V3

Note:

Jika kalimat interogatif diawali WH questions, Have, dan Modals, maka letaknya tetap di awal dan tidak berubah.Sesuaikan to be-nya dengan subjek yang baru (singular/plural.)Rumus di atas hanya yang digunakan pada soal. Untuk rumus lainnya silakan dicek sendiri ya.
