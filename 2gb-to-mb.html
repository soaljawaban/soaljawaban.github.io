Foto : 20 x 512 kb = 10.240 kb = 10.24 mb
Musik : 15 x 5 mb = 75 mb
Film : 2 x 400 mb = 800 mb
Total file : 10.24 + 75 + 800 = 885.24 mb

2Gb = 2000 mb

Jadi sisa Fd : 2000 - 885.24 = 1.114,76 mb = 1,11 GbDalam menghitung sisa FD bisa dengan cara biasa yaitu mengurangi seluruh file terhadap total FD berikut perhitungannya :
Foto : 20 file x 512 KB = 10,240 KB = 10.24 MB.
Musik : 15 file x 5 MB = 75 MB.
Film : 2 file x 400 MB = 800 MB.
Total file : 10.24 MB + 75 MB+ 800 Mb = 885.24 MB.

Flashdisk memiliki kapasitas sebesar 2GB => 2GB= 2,000 MB.

Jadi sisa Flashdisk adalah 2,000 MB - 885.24 MB = 1,114.76 MB.

1,114.76 MB => 1.11 GB.

Jawaban:

Gaet untuk genotipe MMBb adalah MB dan Mb (Opsi C)

Penjelasan:

Gamet

Gamet adalah sel kelamin yang berperan dalam penurunan sifat, Cara menghitung banyak gamet yang yang terbentuk yaitu dengan persamaan :

[tex]2^n[/tex]

dengan n adalah banyaknya alel HeteroZigot,.

Pada soal alel yang heterizigot 1 yaitu Bb jadi banyak gamet yang terbentuk adalah : 2¹ = 2

Menentukan jenis gamet dapat dilakukan dengan diagram berikut :

MM Bb  ⇒ M , B dan b

   ll    B   ll    b    ll

M ll  MB  ll  Mb   ll

Jadi Gamet yang terbentuk adalah MB dan Mb
