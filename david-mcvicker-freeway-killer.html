36. Apa kegunaan dari usb killer



Mengunci dan membuka Komputer 

Maksudnya adalah kita cukup mencolokkan flashdisk untuk membuka dan ketika flashdisk dicabut, maka seketika Komputer  atau PC kita otomatis terkunci. Untuk menggunakan fungsi tersebut, kita harus memasang aplikasi yang bernama Predator. 

Menghubungkan ke jaringan WiFi secara cepat

Hal tersebut memungkinkan karena Windows mempunyai fitur yang dapat menyimpan nama jaringan WiFi yang telah digunakan. Lalu data itu dapat kita simpan di flashdisk  dengan cara klik icon wireless lalu klik kanan WiFi yang baru saja digunakan > properties. Perhatikanlah Tab Connection, kemudian klik link Copy this network profile to a USB Flash drive, lalu klik Next, maka seketika itu, Windows akan menyalin konfigurasi dan pengaturan profile tersebut ke flashdisk. 

Meningkatkan kecepatan komputer 

Berlaku jika ada  bantuan dari aplikasi yang bernama readyboost. Aplikasi tersebut akan membaca cache file yang sering kali dibuka dari flashdisk, sehingga dapat mneingkatkan kecepatan dari komputer itu sendiri. 

Membuat Disk Reset Password

Dengan sebuah disk reset password yang bisa berguna jika kita lupa akan password kita untuk akun pengguna kita pada system Windows yang bukan merupakan bagan dari domain. 

Gunakan sebagai pemutar musicBagi kita yang memang menyukai music dan ingin mendengarnya dikomputer kator, tapi disaat yang sama tidak memiliki MP3 player, maka kita bisa menggunakan USB flash drive sebagai alat pemutar MP3 bersamaan dengan Windows Media Player dan juga satu set handphone.
