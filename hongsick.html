Pendapatan Nasional = 100.000.000 + 50.000.000 + 10.000.000 + 100.000.000 = 260.000.000

3. **Pendapatan Nasional:**

Pendapatan Nasional = Konsumsi Masyarakat + Investasi + Belanja Pemerintah + (Ekspor - Impor)

Pendapatan Nasional = 200.000.000 + 100.000.000 + 50.000.000 + (50.000.000 - 75.000.000) = 325.000.000

Jadi, Pendapatan Nasional pada masing-masing situasi adalah sebagai berikut:

1. Rp 76.176.500

2. Rp 260.000.000

3. Rp 325.000.000
