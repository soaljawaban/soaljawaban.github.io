8. cara penggunaan conditional if dengan jelas dan mendetail dan juga contohnya


type 1: It is possible and also very likely that the condition will be fulfilled.
Form: if + Simple Present, will-Future
Example: If you help me, I’ll help me too.
type 2 : It is possible but very unlikely, that the condition will be fulfilled.
Form: if + Simple Past, Conditional I (= would + Infinitive)
Example: If you helped me, I would help you too.

type 3: It is impossible that the condition will be fulfilled because it refers to the past.
Form: if + Past Perfect, Conditional II (= would + have + Past Participle)
Example: If I you had helped me, I would have helped you too.

11. Perbedaan mixed conditional dan If conditional


║MIXED CONDITIONAL VS IF CONDITIONAL║

[tex]Explanation[/tex]

Mixed Conditional dikatakan sebagai mixed time . Baik mixed conditional maupun if conditional, kedua nya sama-sama menjabarkan dampak/konsekuensi dalam suatu kondisi tertentu.

Dalam Mixed conditional, terdapat perbedaan struktur tenses (waktu ) yang ditempatkan pada –if clause –nya dengan tenses yang ada di main clause -nya (result clause). Ini merupakan hasil campuran  tenses ( mixed time ) yang diterapkan dari seluruh type pada  if conditional.  

→ If conditional terbagi dalam 3 kategori utama  terstruktur :

1. Conditional 1 => untuk menyatakan keinginan yang mungkin akan terwujud segera apabila syaratnya sudah terpenuhi saat ini { Real present conditional  }  

Pattern=> If+S+verb 1(s/es)+C , S+will+verb 1 + C

2. Conditional 2 => untuk menyatakan keinginan yang tidak mungkin  terwujud karena persyaratannya tidak  mampu untuk dipenuhi saat ini juga atau nanti-nanti { Unreal present conditional }

Pattern=>If+S+verb 2 + C , S+ would +verb 1 + C  

3. Conditional 3 => untuk menyatakan keinginan yang tidak terwujud di waktu lampau karena persyaratannya tidak terpenuhi { Unreal past conditional }  

Pattern=> If+S+had+verb 3+C , S+would +have+verb 3 + C

Sedangkan, dalam Mixed conditional  merupakan hasil campuran dari semua  kategori type –if conditional yang telah disebutkan di atas. Untuk mengetahui perbedaan mixed conditional dan –if conditional lebih jelas, dengan mencermati tenses yang digunakan dalam kondisi fakta nya.  

ANALYSIS DATA

1. Fact  : The students use Brainly application now  because they got the best mark yesterday

Conditional : If the students hadn’t gotten the best mark yesterday, they wouldn’t use Brainly application now.

Result ↑↓ :

Fakta pertama disimpulkan bahwa terdapat campuran tenses dalam makna kondisi faktanya.  

Maka,  verb “ hadn’t gotten “ merupakan penjabaran “got ” dan ini menggunakan  struktur –if clause dalam –If conditional tipe ke 3 untuk menyatakan kejadian lampau (past) tidak nyata. Sedangkan     “wouldn’t use” merupakan  penjabaran “ use”  dan ini menggunakan struktur –result clause  dalam –if conditional tipe ke 2 untuk menyatakan kejadian terkait  sekarang  (present  ) tidak nyata .

2. Fact  : The students got the best mark because they used Brainly application yesterday.

Conditional : If the students hadn’t gotten the best mark, they  wouldn’t have used Brainly application yesterday.

Result ↑↓ :

Fakta kedua disimpulkan bahwa tidak terdapat campuran tenses karena ini termasuk kategori terstruktur dari  type 3 conditional  untuk menyatakan kondisi fakta di waktu lampau (past).

“hadn’t gotten” merupakan penjabaran “got” dan “wouldn’t have used” merupakan penjabaran “used.” Bentuk kata kerja dalam If clause dan main clause sama-sama menyatakan kejadian terkait waktu lampau { past} tidak nyata,  jika dilihat tenses dari makna kalimat kondisi faktanya. Maka, ini termasuk kalimat type 3 conditional

Dari hasil analysis, terlihat jelas perbedaan antara mixed conditional dan –if conditional . Pada mixed conditional menggunakan pola tenses yang berbeda (campuran) dalam makna fakta nya dan terdapat kolaborasi  struktur kata kerja pada –if clause nya maupun main clause –nya yang diterapkan dari masing-masing  tipe conditional baik tipe 3 maupun tipe 2

Sedangkan –if conditional menggunakan pola tenses yang sama dalam makna faktanya dan terstruktur berdasarkan pola masing-masing tipe –if conditional.

20. 10 contoh if conditional type zero, 1 , 2 , 3


Kelas: SMA
Pelajaran: English
Kategori: conditional
Pembahasan: 

Zero conditional (to state the general / natural truth)
- If you heat the ice, it melts
- if you put the water in the fridge, it freezes.

Conditional 1 (to state something possible to happen)
- If I see her, I'll talk to her.
- If the police stops by, we'll hide in there. 
- If I can move to Scotland, I'll pay my uncle a visit. 

Conditional 2 (to state something imaginary)
- If the alien attacked earth, I'd choose to make them as my ally.
- If Israel invaded Iran now, the next World War would start.
- If I met Batman, I would ask him to train me to be the next superhero. 
 
Conditional 3 (to state a regret)
- If Neymar hadn't moved to PSG, MSN trio would still have existed.
- If Higuain had scored a goal in 2014 World Cup Final, Argentina would have won that highest FIFA trophy.

28. buatlah rumus Conditional if ( semua type) 1.2,3 beserta contoh kalimatnya


Jawaban:

0 Conditional

Form: If + present simple, .... present simple

Example: If you touch a fire, you get burned.

1st Conditional

Form: if + Simple Present, will-Future

Example: If I find her address, I’ll send her an invitation.

2nd Conditional

Form: if + Simple Past, Conditional I (= would + Infinitive)

Example: If I found her address, I would send her an invitation.

3rd Conditional

Form: if + Past Perfect, Conditional II (= would + have + Past Participle)

Example: If I had found her address, I would have sent her an invitation.
