Rumus:

past continous tense

(+) = Subject + Was/Were + Verb-ing + Object

(-) = Subject + Was/Were + not + Verb-ing + Object

(?) = Was/Were + Subject + Verb-ing + Object + ?

simple past tense

(+) = Subject + Verb 2 + Object

(-) = Subject + Did not + Verb 1 + Object

(?) = Did + Subject + Verb 1 + Object + ?
