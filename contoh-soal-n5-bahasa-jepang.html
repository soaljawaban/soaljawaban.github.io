Jawabanななせんさんびゃくにまんごせんはっぴゃくにじゅうよんまんさんぜんよんひゃくごじゅうまん

Penjelasan

Romaji + Terjemahan :

Nana sen sanbyaku (7.300)Ni man go sen happyaku (25.800)Ni juu yon man san zen (243.000)Yon hyaku go juu man (4.500.000)

Angka bahasa Jepang :

1 = 一 (ichi)  2 = 二 (ni)  3 = 三 (san)  4 = 四 (yon/shi)  5 = 五 (go)  6 = 六 (roku)  7 = 七 (nana/shichi)  8 = 八 (hachi)  9 = 九 (kyuu/ku)  10 = 十 (juu)  

-----------------------------------  

Untuk angka belasan, caranya "juu + (satuan)"  

11 = 十一 (juu ichi)  12 = 十二 (juu ni)  13 = 十三 (juu san)  14 = 十四 (juu yon)  15 = 十五 (juu go)  16 = 十六 (juu roku)  17 = 十七 (juu nana)  18 = 十八 (juu hachi)  19 = 十九 (juu kyuu)  

-----------------------------------  

Untuk angka puluhan, caranya "(angka pertama) + juu".  

Bila ada satuannya, tinggal ditambahkan di belakang.  

20 = 二十 (ni juu)  21 = 二十一 (ni juu ichi)  22 = 二十二 (ni juu ni)  30 = 三十 (san juu)  40 = 四十 (yon juu)  50 = 五十 (go juu)  60 = 六十 (roku juu)  70 = 七十 (nana juu)  80 = 八十 (hachi juu)  90 = 九十 (kyuu juu)  99 = 九十九 (kyuu juu kyuu)  

-----------------------------------  

Untuk angka ratusan diatas seratus, caranya "(angka pertama) + hyaku". *Pengecualian untuk angka 300, 600, dan 800.  

Bila ada puluhan atau satuannya, tinggal ditambahkan di belakang.  

100 = 百 (hyaku)  150 = 百五十 (hyaku go juu)  200 = 二百 (ni hyaku)  300 = 三百 (sanbyaku)  400 = 四百 (yon hyaku)  500 = 五百 (go hyaku)  600 = 六百 (roppyaku)  700 = 七百 (nana hyaku)  800 = 八百 (happyaku)  900 = 九百 (kyuu hyaku)  999 = 九百九十九 (kyuu hyaku kyuu juu kyuu)  

-----------------------------------  

Untuk angka ribuan diatas seribu, caranya "(angka pertama) + sen".  

*Pengecualian untuk angka 3000 dan 8000  

Bila ada ratusan, puluhan, atau satuannya, tinggal ditambahkan di belakang.  

1000 = 千 (sen)  2000 = 二千 (ni sen)  3000 = 三千 (sanzen)  4000 = 四千 (yon sen)  5000 = 五千 (go sen)  6000 = 六千 (roku sen)  7000 = 七千 (nana sen)  8000 = 八千 (hassen)  9000 = 九千 (kyuu sen)  9999 = 九千九百九十九  (kyuu sen kyuu hyaku kyuu juu kyuu)  

*Untuk angka 10.000 ke atas, 'Sen' ditulis 'Issen'  

-----------------------------------

Puluh ribu = 万 (man)10 ribu = 一万 (ichi man)100 ribu = 十万 (juu man)1 juta = 百万 (hyaku man)10 juta = 一千万 (issen man)100 juta = 一億 (ichi oku)1 miliar = 十億 (juu oku)10 miliar = 百億 (hyaku oku)100 miliar = 一千億 (issen oku)1 triliun = 一兆 (icchou)10 triliun = 十兆 (jucchou)100 triliun = 百兆 (hyacchou)

14. Bantu sebutkan 10 kanji N5Jangan asal thanks​


Semoga membantu Sudah ya Dek makasih poinnya

Jawaban & Penjelasan  

    Kanji merupakan salah satu jenis huruf Jepang, berupa karakter pinjaman dari Cina. Masing-masing huruf kanji memiliki beragam cara baca dan makna. Cara baca kanji dibagi menjadi Kunyomi (cara baca Jepang) dan Onyomi (cara baca Cina). Jumlah kanji yang biasa digunakan ada sekitar 2000 kanji. Kanji dibagi menjadi 5 level, dari yang paling rendah N5, sampai yang paling tinggi N1.

Berikut 10 contoh kanji level N5 :

========

1). 人

Arti = Orang

Kunyomi = Hito

Onyomi = Jin, Nin

   Contoh :

人間 (Ningen) = Manusia人生 (Jinsei) = Kehidupan

========

2). 山

Arti = Gunung

Kunyomi = Yama

Onyomi = San

   Contoh :

富士山 (Fujisan) = Gunung Fuji山田 (Yamada) = Yamada *nama

========

3). 川

Arti = Sungai

Kunyomi = Kawa

Onyomi = Sen

   Contoh :

川魚 (Kawazakana) = Ikan sungai鴨川 (Kamogawa) = Sungai Kamo

========

4). 生

Arti = Lahir, hidup, tumbuh

Kunyomi = I, U, Ha

Onyomi = Sei, Shou

   Contoh :

先生 (Sensei) = Pak/Bu (guru, dokter, dll)年生 (Nensei) = Kelas *tingkatan

========  

5). 日

Arti = Hari, matahari, minggu, Jepang

Kunyomi = Hi, Bi, Ka

Onyomi = Nichi, Jitsu

   Contoh :

日曜日 (Nichiyoubi) = Hari Minggu今日 (Kyou) = Hari ini

========

6). 月

Arti = Bulan, senin

Kunyomi = Tsuki

Onyomi = Getsu, Gatsu

   Contoh :

四月 (Shigatsu) = Bulan April今月 (Kongetsu) = Bulan ini

========

7). 火

Arti = Api, selasa

Kunyomi = Hi, Bi, Ho

Onyomi = Ka

   Contoh :

火曜日 (Kayoubi) = Hari Selasa花火 (Hanabi) = Kembang api

========

8). 水

Arti = Air, rabu

Kunyomi = Mizu

Onyomi = Sui

   Contoh :

水曜日 (Suiyoubi) = Hari Rabu鼻水 (Hanamizu) = Ingus

========

9). 土

Arti = Tanah, sabtu

Kunyomi = Tsuchi

Onyomi = To, Do

   Contoh :

土曜日 (Doyoubi) = Hari Sabtu土地 (Tochi) = Tanah

========

10). 花

Arti = Bunga

Kunyomi = Hana

Onyomi = Ka, Ke

   Contoh :

花火 (Hanabi) = Kembang api花びら (Hanabira) = Kelopak bunga

========
