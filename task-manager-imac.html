2. apa yang terjadi jika process penting di task manager di nonaktifkan (end task)??


Jawaban:

Apa yang terjadi jika process penting di task manager di nonaktifkan (end task)? JAWAB: Akan terjadi malfungsi sistem dan kesalahan sistem yang berakibat ringan hingga fatal, seperti not responding.

Penjelasan:

Ada 4 jenis proses penting yang kita tidak boleh end task di task manager begitu saja, yaitu:

Vital System Process.Windows Startup Application.Windows Explorer Application.Windows Logon Application.

Jika kita melakukan end task kepada salah satu dari 4 jenis proses penting tersebut, maka akan terjadi malfungsi sistem. Yaitu contohnya jika kita end task Windows Explorer Aplication (explorer.exe) maka GUI atau tampilan layar Windows akan stuck dan tidak merespon. Semua fitur sistem seperti Start Menu, Taskbar dan System Tray tidak akan bisa digunakan.

17. Tuliskan kegunaan dari task manager yang terdapat dari submenu pada tools


Penjelasan:

Sebenarnya ini pertanyaan untuk TI. Tapi Saya akan bantu menjawab :

Dalam submenu "Tools" pada beberapa aplikasi, task manager mungkin memiliki beberapa kegunaan, antara lain:

1. Memantau kinerja sistem: Task manager dapat menunjukkan penggunaan CPU, memori, dan penggunaan jaringan, sehingga pengguna dapat melihat apakah sistem mereka sedang berjalan dengan optimal atau terlalu banyak memakan sumber daya.

2. Menutup aplikasi yang macet: Jika sebuah aplikasi tidak merespons atau mengalami masalah, pengguna dapat menggunakan task manager untuk menutupnya. Task manager dapat memaksa penghentian aplikasi atau memaksa aplikasi untuk menutup dengan halus.

3. Mengatur startup: Task manager dapat membantu pengguna mengelola aplikasi yang berjalan saat sistem di-booting. Pengguna dapat menonaktifkan aplikasi yang tidak diperlukan, sehingga startup sistem menjadi lebih cepat.

4. Melihat penggunaan sumber daya oleh aplikasi: Task manager dapat menunjukkan aplikasi yang sedang menggunakan banyak sumber daya, sehingga pengguna dapat menentukan apakah aplikasi tersebut perlu dihentikan atau dikonfigurasi ulang.

5. Membuat snapshot: Task manager dapat membuat snapshot dari sistem, sehingga pengguna dapat memeriksa sistem pada waktu yang berbeda untuk melihat perbedaan dalam penggunaan sumber daya dan proses yang berjalan.

23. Tolong dibantuin yaa............di task manager CPU Usage saya selalu menunjukkan prosentase 100%. itu keanehan bagi saya karena membuat laptop saya menjadi lemot. Apa yang menyebabkan hal sepeti itu dan bagaimana cara mengatasinya?


CPU usage bisa mencapai 100% itu wajarnya karena Anda terlalu banyak menjalankan aplikasi.

Namun apabila Anda sedang tidak menjalankan banyak aplikasi tetap 100% juga, berarti ada sesuatu yang bekerja sendiri di belakang layar.
Bisa jadi itu malware, update system, registry error, startup program yg tidak perlu, kerusakan software dll.

Kalau dibiarin terus bisa-bisa CPU cepet panas, normalnya harus dijaga di 5-75%

Coba kembali lagi ke task manager, lalu urutkan berdasarkan CPU usage, lihat aplikasi mana yang paling besar memakan CPU.
Kalau itu sebuah svchost.exe, kill task aja yang terbesarnya.

Kalau di kemudian hari masih terus berulang, install ulang merupakan cara yang praktis, jangan lupa untuk membackup data sebelum install ulang.

Ini saya lampirkan CPU usage yg normal.
