Penjelasan:Imbuhan

Imbuhan adalah merupakan suatu kata yang merupakan sambungan suatu kata yang terdiri dari:

Prefiks (awal) Infiks (tengah) Sufiks (akhir) Konfliks/ Simulfiks (awalan/akhir)

1. Prefiks (awal)

Prefiks/awal adalah suatu kata imbuhan yang terletak pada awal suatu kata.

Contoh

meN+ber+di+ter+peN+per+se+ke+

contoh kata

meN+masak = memasak (masak) meN+ngaji = mengaji (ngaji) meN+bantu = membantu (bantu) meN+baca = membaca (baca) dll

2. Infiks (tengah)

Infiks/tengah adalah suatu kata imbuhan yang terletak ditengah pada suatu kata.

Contoh

+el++em++er++e++in+

contoh kata

m+el+aju = melaju (maju) c+er+it+er+a = ceritera (cerita) g+em+etar = gemetar (getar) dll

3. Sufiks (akhir)

Sufiks/akhir adalah suatu kata imbuhan yang terletak diakhir pada suatu kata.

Contoh

+kan+an+i+nya

contoh kata

buat+an = buatan (buat) cinta+i = cintai (cinta) rasa+i = rasai (rasa)awal+an = awalan (awal) akhir+an = akhiran (akhir) dll

4. Konfliks/ Simulfiks (awalan/akhir)

Konfliks/Simulfiks adalah suatu kata imbuhan yang terletak diawal/diakhir pada suatu kata.

Contoh

ke+anpe+anme+kanpeN+anber+anse+nya

contoh kata

me+maaf+kan = memaafkan (maaf) ber+dasar+kan = berdasarkan (dasar) pel+ajar+an = pelajaran (belajar)

Pendahuluan :

Imbuhan adalah sebuah kata tambahan pada sebuah kalimat, Imbuhan biasanya terdapat pada :

Awal KalimatTengah KalimatAkhir KalimatAwalan dan Akhir

• Awal Kalimat

Imbuhan biasanya terletak pada Awal Kalimat atau dapat disebut sebagai Prefiks.

Contoh Imbuhan Awal Kalimat :

Men...Me...Ter...Bi...Ke...Di...dst..

Contoh kata yang sudah diberi Imbuhan :

Me + masak = MemasakDi + kebun = DikebunKe + pada = Kepadadst..

• Tengah Kalimat

Imbuhan biasanya terletak pada Tengah Kalimat atau dapat disebut sebagai Infiks.

Contoh Imbuhan Tengah Kalimat :

nya...atur...ambil...tan...dst..

Contoh kata yang sudah diberi Imbuhan :

Me + nya + pa = MenyapaPer + atur + an = PeraturanMeng + ambil + kan = MengambilkanBer + tan + nya = Bertanyadst..

• Akhir Kalimat

Imbuhan biasanya terletak pada Akhir Kalimat atau dapat disebut sebagai Sufiks.

Contoh Imbuhan Akhir Kalimat :

an...kan...ngi...nya...dst..

Contoh kata yang sudah diberi Imbuhan :

Ber + papas + an = BerpapasanMem + buat + kan = MembuatkanMe + lindu + ngi = Melindungidst..

• Awalan dan Akhir

Imbuhan biasanya terletak pada Awal dan Akhir atau dapat disebut sebagai Konfliks dan Simulfiks

Contoh Imbuhan Awal dan Akhir Kalimat :

Me + pa...Me + panen...Di + tanya...Se + bagai...dst..

Contoh kata yang sudah diberi Imbuhan :

Me + nga + pa = MengapaMe + panen = MemanenDi + tanya = DitanyaSe + bagai = Sebagaidst..

1. meN- + takjub + kan = . . . .

Pembahasan :

1. meN- + takjub + kan

￫ Menakjubkan

Kata Men-ak ( Awal Kalimat ) + kata takjub, kata "tak" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

2. meN- + pasar + kan

￫ Memasarkan

Kata Men-ma ( Awal Kalimat ) + kata pasar, kata "p" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

3. meN- + karantina + kan

￫ Mengkarantinakan

Kata Men-g ( Awal Kalimat ) + kata karantina ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

4. meN- + tular + kan

￫ Menularkan

Kata Men ( Awal Kalimat ) + kata tukar, huruf "t" digantikan dengan huruf "n" ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

5. meN- + sanggup + i

￫ Menyanggupi

Kata Men-nya ( Awal Kalimat ) + kata sanggup, kata "s" dihapus ( Tengah Kalimat ) + kata i ( Akhir Kalimat )

6. meN- + saksi + kan

￫ Menyaksikan

Kata Men-ya ( Awal Kalimat ) + kata saksi, kata "sa" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

7. meN- + pangkas

￫ Memangkas

Kata Me ( Awal Kalimat ) + kata pangkas, kata "m" diubah menjadi kata "p" ( Akhir Kalimat )

8. meN- + tiru + kan

￫ Menirukan

Kata Me, kata "N" dihapus ( Awal Kalimat ) + kata tiru diubah menjadi kata "niru" ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

9. meN- + prakarsa + i

￫ Memperkarsai

Kata MeN, kata "N" diubah menjadi kata "m" ( Awal Kalimat ) + kata prakarsa ( Tengah Kalimat ) + kata i ( Akhir Kalimat )

10. meN- + seru + kan

￫ Menyerukan

Kata Men-y ( Awal Kalimat ) + kata seru, kata "s" dihapus ( Tengah Kalimat ) + kata kan ( Akhir Kalimat )

Penjelasan:

Kata berimbuhan adalah kata-kata dasar yang mendapatkan imbuhan yang berupa awalan, akhiran, sisipan, dan awalan-akhiran. Imbuhan sendiri berfungsi untuk menambahkan arti atau maksud dari kata-kata dasar yang diberi imbuhan tersebut.

Terdapat beberapa jenis kata berimbuhan, yaitu:

1. Prefiks (Awalan)

Prefiks adalah awalan yang merupakan penambahan pada bagian awal kata dasar. Imbuhan yang termasuk ke dalam kategori prefiks antara lain: ber-, me-, di-, ter-, ke-, se-, dan pe-.

Contoh:

BermainMeringkasDisamping

Kata asalnya:

Bermain = ber + mainMeringkas = me + ringkasDisamping = di + samping

2. Sufiks (Akhiran)

Sufiks merupakan akhiran yang merupakan penambahan imbuhan di bagian akhir kata dasar sehingga biasa disebut sebagai akhiran. Imbuhan yang termasuk ke dalam kategori sufiks antara lain: -an, -kan, dan -i.

Contoh:

MingguanPulangkan

Kata asalnya:

Mingguan = minggu + anPulangkan = pulang + kan

3. Infiks (Sisipan)

Imbuhan ini juga bisa disebut sebagai sisipan. Hal ini karena Infiks merupakan penambahan imbuhan yang disisipkan pada bagian tengah kata dasar. Beberapa contoh infiks antara lain: -el-, -em-, -in-, dan -er-. Contoh penggunaan infiks yaitu imbuhan -in- pada kata kerja sehingga menjadi kinerja.

4. Konfiks (Awalan dan Akhiran)

Konfiks adalah bentuk penambahan imbuhan pada bagian depan dan akhir kata dasar. Imbuhan yang termasuk ke dalam kategori konfiks antara lain: ke-an, ber-an, pe-an, per-an, dan se-nya.

Contoh:

KeberanianPermainanSebaiknya

Kata asalnya:

Keberanian = ke + berani + anPermainan = per + main + anSebaiknya = se + baik + nya

16. Bentuk kartel dan trust Bentuk trust dan kartel


Bentuk kartel dan trust antara lain :

1. Kartel

Kartel daerah yaitu sekelompok orang yang melarang orang lain yang bukan anggota kelompoknya untuk melakukan perdagangan di daerah yang mereka kuasai.Kartel kondisi, yaitu suatu kartel yang mensyaratkan kondisi-kondisi tertentu dalam perdagangan seperti diskon, letak penjualan dan lain sebagainya.Karte produksi, yaitu kartel yang berfokus pada pengendalian produksi barang tujuan mereka adalah membuat barang yang beredar dimasyarakat sedikit sehingga mereka dapat mengendalikan harga.Kartel harga, yaitu kartel yang digunakan untuk menentukan harga minimal suatu barang.Kartel pembagian laba, yaitu kartel yang digunakan untuk menentukan berapa besar laba yang harus diambil dalam menjual suatu barang.

2. Trust

Konsentrasi horizontal yaitu pemusatan beberapa Badan usaha yang sama konsentrasinya menjadi satuKonsentrasi vertikal yaitu badan usaha yang bekerjasama dan saling berurutan untuk menghasilkan suatu barang, misalnya untuk menghasilkan sebuah baju diperlukan petani kapas, pabrik pemintalan benang, pabrik tekstil, dan kemudian desainer dan penjahit.Konsentrasi paralel yaitu penggabungan dari beberapa badan usaha yang kegiatannya memproduksi atau menjual barang-barang yang berlainan, akan tetapi untuk pesanan yang samaPembahasan

Kartel merupakan gabungan dari beberapa perusahaan yang bertujuan untuk mengendalikan harga barang dipasaran, sehingga dari masing-masing perusahaan tersebut membuat perjanjian agar dapat mengendalikan pasar dan menghilangkan kompetitor.

Genre Percakapan Telepon

      Percakapan dapat terjadi antara dua orang atau lebih. Percakapan dapat dibagi berdasarkan dua kondisi yaitu formal dan informal. Percakapan formal akan terjadi jika berada di tempat formal atau dengan orang yang lebih tua. Adapun untuk percakapan informal dapat terjadi di mana saja dan kapan saja. Percakapan antara dua orang di telepon merupakan contoh dari percakapan formal dan bisa juga termasuk dalam percakapan tidak formal. Berkomunikasi melalui telepon termasuk dalam jenis komunikasi tidak langsung. Meski begitu, kalimat yang diucapkan melalui pesawat telepon harus mencerminkan etiket kesantunan dan juga keefektifan. Ada beberapa hal yang perlu diperhatikan dalam berkomunikasi melalui telepon, yaitu salam pembuka pembicaraan, menyatakan identitas diri, saat menerima telepon, menutup pembicaraan,waktu bertelepon, sikap bertelepon, dan efisien.

Dina     : Halo, Selamat siang, Bu. Saya Dina, bisa berbicara dengan Jelita?

Ibu       : Bisa, tunggu sebentar ya.

Dina     : Iya, Bu.

Jelita    : Halo, ada apa Dina?

Dina     : Maaf ya aku mengganggu jam istirahatmu. Aku hanya mau mengingatkan jangan lupa nanti sore untuk membawa semua nota pengeluaran kegiatan kemah kemarin. Karena, hari ini mau aku rekap.

Jelita    : Iya tidak apa Dina, nanti aku akan membawa semua nota dan catatan yang diperlukan saat rapat. Terima kasih ya sudah mengingatkan.

Jelita    : Iya, sama-sama. Selamat siang.
