Harga 2 celana panjang dan 5 kemeja adalah Rp. 410.000. Sedangkan harga 3 celana panjang dan 2 kemeja Rp. 340.000. Maa harga sebuah celana panjang dan 2 kemeja adalah Rp. 180.000.

Penyelesaian Soal :

Misalkan celana panjang = x

              Kemeja = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Harga 2 celana panjang dan 5 kemeja adalah Rp. 410.000" yaitu :

2x + 5y = 410.000   ...(Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Sedangkan harga 3 celana panjang dan 2 kemeja Rp. 340.000" yaitu :

3x + 2y = 340.000   ...(Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai x dengan menggunakan cara sebagai berikut :

2x + 5y = 410.000    ║×2║   4x + 10y = 820.000

3x + 2y = 340.000   ║×5║   15x + 10y = 1.700.000

_______________________________________ -

                                                      -11x = -880.000

                                                          x = [tex]\frac{880.000}{11}[/tex]

                                                          x = 80.000

LANGKAH KEEMPAT (IV)

Subtitusikan nilai x pada persamaan 2 sehingga diperoleh nilai y dengan menggunakan cara sebagai berikut :

3x + 2y = 340.000

3 (80.000) + 2y = 340.000

240.000 + 2y = 340.000

2y = 340.000 - 240.000

2y = 100.000

y = [tex]\frac{100.000}{2}[/tex]

y = 50.000

LANGKAH KELIMA (V)

Hitung harga 1 celana panjang dan 2 kemeja dengan menggunakan cara sebagai berikut :

x + 2y = 80.000 + 2 (50.000)

          = 80.000 + 100.000

          = 180.000

∴ Kesimpulan harga 1 celana panjang dan 2 kemeja adalah sebesar Rp. 180.000.
