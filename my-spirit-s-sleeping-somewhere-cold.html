PEMBAHASAN"Present Continuous Tense"

Rumus

(+) Subject + To be + (V1 + ing) + Object

(-) Subject + Don't / Doesn't + To be + (V1 + ing) + Object

(?) To be + Subject + (V1 + ing) + Object?

Keterangan

Untuk Subject --> I, to be nya "am"Untuk Subject --> You, They We, to be nya "are"Untuk Subject --> She, He, It, to be nya "is"

27. My brother listens to radio before sleeping(?) ​


Pola kalimat interogatif(?) dari Simple Present Tense adalah sebagai berikut.

(?) Do/Does + Subject + Verb 1 + ...

Jadi, JAWABANNYA adalah:

Does my brother listen to radio before sleeping?

-----------------------------------------------------------------------

*Mengapa menggunakan does? karena Subject-nya adalah kata benda tunggal/hanya satu/singular [brother]. Seandainya saja Subject-nya adalah kata benda jamak/lebih dari satu/plural [brothers], maka yang digunakan adalah do.

Pembahasan

Past continuous tense atau past progressive tense adalah sebuah kata kerja yang digunakan untuk menyatakan kejadian atau peristiwa yang secara jelas dilakukan di masa lampau.

Rumus past continuous(+) S + to be (was/were) + V-ing + O(-) S + to be (was/were) + not + V-ing + O(?) To be (was/were) + S + V-ing + O

Penggunaan was dan were

Was, digunakan untuk subjek I, she, he, itWere, digunakan untuk subjek you, we, they

Jawaban:

Arranging the words:

1. You mustn't forget to lock the door before you leave.

2. I think the key is somewhere in my room.

3. I wrote letters to ten people, but no one answered.

4. Ilham doesn't have to buy a new camera.

5. The homework is easy, you don't have to help me.

Penjelasan:

Jawaban dicetak tebal dan merupakan materi pembelajaran Arranging the words (menyusun kata kata) menjadi sebuah kalimat yang benar dalam Bahasa Inggris.

Menyusun kata-kata:

1. Jangan lupa mengunci pintu sebelum pergi.

2. Aku pikir kuncinya ada di suatu tempat di kamarku.

3. Aku menulis surat kepada sepuluh orang, tetapi tidak ada yang menjawab.

4. Ilham tidak harus membeli kamera baru.

5. Pekerjaan rumah itu mudah, kamu tidak perlu membantuku.

Jika susunannya salah atau kurang, maka terjemahannya pun tidak benar.

SImple Present Tense:

Verbal: (+) Subject + Verb-1 S/ES + Object

Nominal: (+) Subject + Tobe (is/am/are) + Adjective / Noun / Adverb

Modal Verbs (Must):

(-) Subject + must + not + Verb-1 + Object
