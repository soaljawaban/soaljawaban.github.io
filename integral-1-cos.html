2. 1. integral 1 / cos^6(3x + 2) = ?2. integral cos^2 3x dx = ?3. integral sin^2 dx = ?


1) int 1/cos^6 (3x+2) dx
= int sec^6 (3x+2) dx
misal :
u = 3x + 2
du = 3 dx ----> dx = 1/3  du
so,
=1/3  int sec^6 (u) du
= 1/3 int (sec^2(u))^2 sec^2(u) du
= 1/3 int (1+tan^2(u))^2 sec^2(u) du
= 1/3 int (1 + 2 tan^2(u) + tan^4(u)) sec^2u) du
split menjadi 3 integral :
= 1/3 int sec^2(u) du + 2/3 int tan^2(u) sec^2(u) du + 1/3 int tan^4(u) sec^2(u) du

jelas, untuk 1/3 int sec^2(u) du = 1/3 tan(u)
untuk yg lainnya gunakan int u-subs,
misal y = tan(u)
dy = sec^2(u) du
shg, diperoleh :
2/3 int tan^2(u)sec^2(u)du + 1/3 int tan^4(u)sec^2(u) du
= 2/3 int y^2 dy + 1/3 y^4 dy
= 2/3 * 1/3 y^3 + 1/3 * 1/5 y^5 + C
= 2/9 y^3 + 1/15 y^5 + c
substitusikan kembali bahwa y = tan(u), maka
= 2/9 tan^3 (u) + 1/15 tan^5 (u)

shg totalnya,
1/3 tan(u) + 2/9 tan^3 (u) + 1/15 tan^5 (u) + C
finally, substitusikan kembali u = (3x + 2)
so,
= 1/3 tan(3x+2) + 2/9 tan^3 (3x+2) + 1/15 tan^5 (3x+2) + C

2) int cos^2 (3x) dx
= int (1 + cos(6x))/2 dx
= int 1/2 dx + 1/2 cos(6x) dx
= 1/2 x + 1/12 sin(6x) + C

4. Integral dx/1 + cos x =


Hasil integral dari [tex]\int {\frac{1}{1 \:+\: cos \: x}} \, dx[/tex] adalah [tex]tan \: (\frac{x}{2}) \:+\: c[/tex]. Bentuk trigonometri dapat diubah menjadi bentuk lainnya lalu diintegralkan.

INTEGRAL TRIGONOMETRI

Bentuk umum trigonometri yang harus diingat

[tex]sin \: 2A \:=\: 2 \: sin \: A \: cos \: A[/tex] [tex]cos \: 2A \:=\: cos^2 \: A \:-\: sin^2 \: A[/tex]
[tex]cos \: 2A \:=\: 2 \: cos^2 \: A \:-\: 1[/tex]
[tex]cos \: 2A \:=\: 1 \:-\: 2 \: sin^2 \: A[/tex] [tex]tan \: 2A \:=\: \frac{2 \: tan \: A}{1 \:-\: tan^2 \: A}[/tex] [tex]sec \: A \:=\: \frac{1}{cos \: A}[/tex] [tex]cos \: A \:=\: \frac{1}{sin \: A}[/tex]

Integral trigonometri untuk bentuk

[tex]\int {cos \: x} \, dx \:=\: sin \: x \:+\: c[/tex] [tex]\int {sin \: x} \, dx \:=\: - cos \: x \:+\: c[/tex] [tex]\int {sec^2 \: x} \, dx \:=\: tan \: x \:+\: c[/tex] [tex]\int {cosec^2 \: x} \, dx \:=\: - cot \: x \:+\: c[/tex] [tex]\int {tan \: x \: sec \: x} \, dx \:=\: sec \: x \:+\: c[/tex] [tex]\int {cotan \: x \: cosec \: x} \, dx \:=\: - cosec \: x \:+\: c[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

[tex]\int {\frac{1}{1 \:+\: cos \: x}} \, dx[/tex]

Ditanyakan:

Hasil integral?

Jawaban:

Perhatikan

[tex]cos \: 2x \:=\: 2 \: cos^2 \: x \:-\: 1[/tex]

[tex]cos \: x \:=\: 2 \: cos^2 \: \frac{x}{2} \:-\: 1[/tex]

[tex]1 \:+\: cos \: x \:=\: 1 \:+\: 2 \: cos^2 \: \frac{x}{2} \:-\: 1[/tex]

[tex]1 \:+\: cos \: x \:=\: 2 \: cos^2 \: \frac{x}{2}[/tex]

Bentuk dari soal dapat diubah menjadi

[tex]\int {\frac{1}{1 \:+\: cos \: x}} \, dx \:=\: \int {\frac{1}{2 \: cos^2 \: \frac{x}{2}}} \, dx[/tex]

= [tex]\frac{1}{2} \int \frac{1}{cos^2 \: \frac{x}{2}}} \, dx[/tex]

= [tex]\frac{1}{2} \int \sec^2 \: \frac{x}{2}}} \, dx[/tex]

Misalkan

u = [tex]\frac{x}{2}[/tex] [tex]\frac{du}{dx} \:=\: \frac{1}{2}[/tex]
[tex]du \:=\: \frac{1}{2} \: dx[/tex]
[tex]2 \: du \:=\: dx[/tex]
dx = 2 du

Maka

= [tex]\frac{1}{2} \int \sec^2 \: u \, (2 \: du)[/tex]

= [tex]\frac{1}{2} \times 2 \int \sec^2 \: u \, du[/tex]

= [tex]\int \sec^2 \: u \, du[/tex]

= [tex]tan \: u \:+\: c[/tex]

= [tex]tan \: (\frac{x}{2}) \:+\: c[/tex]
