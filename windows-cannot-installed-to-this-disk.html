1. Bagaimana cara Windows menamai suatu drive (seperti hard disk atau floppy disk drive)?


Jawaban:

Windows menamai suatu drive dengan huruf alfabet.

Penjelasan:

Untuk floppy disk drive diawali A dan B

Untuk hard disk diawali C, D dan seterusnya (menyesuaikan).

Salah 1 contohnya ada di gambar berikut ini (pada Windows 10), dalam jendela Disk Management:

- tak ada A & B menunjukkan komputer tak dipasang floppy disk drive

- C dan D menunjukkan drive pada harddisk

- E adalah drive untuk CD ROM

8. untuk melakukan perawatan perangkat lunak menggunakan disk defragmenter yang ada pada Windows partisi dan file yang ada didalamnya dapat dirapikan dengan cara​


cara melakukan disk defragmenter pada hardisk defragmenterisk menggunakan windows XP dan juga windows 7:

Langkah pertama, klik start – accessories – pilih disk defragmentation (pada penggunan Windows XP)

Untuk pengguna Windows 7, pilh start, lalu search disk defragmentation, lalu jalankan programnya.

Setelah programnya berjalan, maka akan keluar jendela disk defragmenter di layar komputer anda. ketika muncul, aka nada pilihan drive akan diproses, yaitu drive D, C, dan juga system reverse. (Jumlah ini tergantung dari jumlah drive yang anda gunakan sebagai partisi pada hardisk defragmenterisk yang anda gunakan di dalam komputer)

Pilih drive C terlebih dahulu (sebagai lokasi drive sistem operasi)

Klik pada drive C, lalu pilih analyze disk terlebih dahulu. Analyze disk digunakan untuk melakukan analisa kebutuhan defragmentisasi pada drive tersebut.

Setelah selesai dilakukan proses analyze, maka saatnya anda melakukan proses defragmentasi. Klik pada drive C, lalu pilih defragment disk. Ketika sudah dipilih, maka sistem operasi anda akan secara otomatis bekerja dengan cara mengurutkan dan juga merapihkan file dan juga data – data anda yang tidak teratur, sehingga menjadi lebih teratur.

Tunggu proses defragmentasi hingga selesai. Setelah selesai, pindah ke drive berikutnya, yaitu drive D, untuk melakukan analisa dan juga defragmentasi pada drive D di hardisk defragmenterisk anda.

Lakukan langkah yang sama seperti ketika anda akan memproses disk defragmenter pada drive C hardisk defragmenterisk anda.
