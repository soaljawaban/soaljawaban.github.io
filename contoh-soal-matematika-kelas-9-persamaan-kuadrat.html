5. matematika kelas 9 Mtsn persamaan kuadrat 2x-3 x-3​


Jawaban:

Untuk menyelesaikan persamaan kuadrat 2x - 3x - 3 = 0, kita akan menggunakan metode faktorisasi.

Langkah-langkahnya adalah sebagai berikut:

1. Tulis persamaan kuadrat dalam bentuk standar: ax^2 + bx + c = 0. Dalam kasus ini, a = 2, b = -3, dan c = -3.

2. Faktor persekutuan terbesar (FPB) dari koefisien a, b, dan c adalah 1 (karena tidak ada faktor yang sama bagi mereka). Oleh karena itu, kita harus mencari faktor-faktor dari c yang menghasilkan b saat dijumlahkan.

Faktor dari -3 adalah -1, 1, -3, dan 3.

Kita mencari dua angka yang saat dikalikan menghasilkan -3 dan saat dijumlahkan menghasilkan -3. Nah, angka-angka itu adalah -3 dan 1.

3. Sekarang kita akan membagi persamaan menjadi dua bagian berdasarkan faktor yang telah kita temukan:

2x - 3x - 3 = (2x - 3)(x + 1) = 0

4. Persamaan ini menjadi dua bagian yang dapat diselesaikan secara terpisah:

2x - 3 = 0 atau x + 1 = 0

5. Selesaikan setiap bagian persamaan:

a. 2x - 3 = 0:

Pertama, kita tambahkan 3 ke kedua sisi persamaan:

2x = 3

Kemudian, bagi kedua sisi dengan 2:

x = 3/2 = 1.5

b. x + 1 = 0:

Pertama, kurangi 1 dari kedua sisi persamaan:

x = -1

Jadi, solusi persamaan kuadrat 2x - 3x - 3 = 0 adalah x = 1.5 dan x = -1.

18. Algebra persamaan kuadrat matematika kelas 9​


Jawaban:

[tex]HP = {2, - 8}[/tex]

Penjelasan dengan langkah-langkah:

[tex] \frac{1}{4 - x} - \frac{1}{2 + x} - \frac{1}{4} = 0[/tex]

Tuliskan setiap pecahan dengan penyebut yang sama yaitu, 4(4 - x) (2 + x)

[tex]\frac{4(2 + x) \times1}{4(2 + x)(4 - x)} - \frac{4(4 - x) \times 1}{4(4 - x)(2 + x)} - \frac{(4 - x)(2 + x) \times 1}{(4 - x)(2 + x) \times 4} = 0[/tex]

[tex] \frac{4(2 + x)}{4(4 - x)(2 + x)} - \frac{4(4 - x)}{4(4 - x)(2 + x)} - \frac{(4 - x)(2 + x)}{4(4 - x)(2 + x)} = 0[/tex]

[tex] \frac{4(2 + x) - 4(4 - x) - (4 - x)(2 + x)}{4(4 - x)(2 + x)} = 0[/tex]

[tex] \frac{8 + 4x - 16 + 4x - (8 + 2x - {x}^{2}) }{4(4 - x)(2 + x)} = 0[/tex]

[tex] \frac{6x - 16 + {x}^{2} }{4(4 - x)(2 + x)} = 0 \\ [/tex]

[tex]6x - 16 + {x}^{2} = 0 \\ {x}^{2} + 6x - 16 = 0 \\ {x}^{2} + 8x - 2x - 16 = 0 \\ x(x + 8) - 2(x + 8) = 0 \\ (x + 8)(x - 2) = 0[/tex]

[tex] x + 8 = 0 \\ x = - 8 \\ x - 2 = 0 \\ x = 2[/tex]

[tex]HP = {2, - 8}[/tex]

Bentuk umum persamaan kuadrat adalah ax² + bx + c = 0. Dalam hal ini, a ≠ 0 dan a, b, dan c merupakan bilangan riil. Lebih lanjut, a ≠ 0 sebab jika a = 0, maka tidak akan tebentuk persamaan kuadrat.

Penjelasan:

Poin a.

[tex]\[\begin{gathered} 3x - {x^2} = - 2 \hfill \\ \Leftrightarrow 3x - {x^2} + 2 = 0 \hfill \\ \Leftrightarrow - {x^2} + 3x + 2 = 0 \hfill \\ \end{gathered} \][/tex]

Poin b.

[tex]\[\begin{gathered} 2{x^2} = - 5 + x \hfill \\ \Leftrightarrow 2{x^2} + 5 - x = 0 \hfill \\ \Leftrightarrow 2{x^2} - x + 5 = 0 \hfill \\ \end{gathered} \][/tex]

Poin c.

[tex]\[\begin{gathered} 2x\left( {2x + 2} \right) = 2x\left( {x - 3} \right) \hfill \\ \Leftrightarrow 4{x^2} + 4x = 2{x^2} - 6x \hfill \\ \Leftrightarrow 4{x^2} - 2{x^2} + 4x + 6x = 0 \hfill \\ \Leftrightarrow 2{x^2} + 10x = 0 \hfill \\ \end{gathered} \][/tex]

Poin d.

[tex]\[\begin{gathered} 2{x^2} - 5 = 3x \hfill \\ \Leftrightarrow 2{x^2} - 5 - 3x = 0 \hfill \\ \Leftrightarrow 2{x^2} - 3x - 5 = 0 \hfill \\ \end{gathered} \][/tex]

Poin e.

[tex]\[\begin{gathered} 2\left( {2{x^2} - 3} \right) = x\left( {x + 1} \right) \hfill \\ \Leftrightarrow 4{x^2} - 6 = {x^2} + x \hfill \\ \Leftrightarrow 4{x^2} - {x^2} - x - 6 = 0 \hfill \\ \Leftrightarrow 3{x^2} - x - 6 = 0 \hfill \\ \end{gathered} \][/tex]

Poin f.
