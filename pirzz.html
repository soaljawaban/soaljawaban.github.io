34. Tolong jawab ya pilzz yg cepat


Mapel : Matematika
Bab : Busur dan Juring Lingkaran

Diketahui :
∠AOB = 120°
∠BOC = 60°
Panjang busur AB = 40 cm
Luas juring BOC = 64 cm²

Ditanyakan :
A] Panjang busur BC = .. ?
B] Luas juring AOB = ... ?

Penyelesaian :
A] Panjang busur AB/Panjang busur BC = ∠AOB/∠BOC
                      40 cm / Panjang busur BC = 120°/60°
                                   Panjang busur BC = 40 x 60 / 120
                                   Panjang busur BC = 20 cm

B] Luas juring AOB/Luas juring BOC = ∠AOB/∠BOC
                    Luas juring AOB/64 cm² = 120°/60°
                                Luas juring AOB = 64 x 120 : 60
                                Luas juring AOB = 128 cm²
