19. nyatakan besar sudut sudut dalam bentuk desimal B.110°36'


Nyatakan besar sudut berikut dalam bentuk desimal:

a. 110°36'  = 110,6°b. 197°22'30"​ = 197,375°Pendahuluan

Untuk sudut yang lebih kecil dari derajat adalah menit, dilambangkan dengan ( ' ), dan detik dilambangkan dengan ( " ).

Konversi dari ukuran sudut-sudut itu adalah :

1 derajat = 60 menit, ditulis 1° = 60'1 menit = 60 detik, ditulis 1 ' = 60"1 derajat = 3600 detik, ditulis 1° = 3600''[tex]\displaystyle 1' = (\frac{1}{60})^{0}[/tex][tex]\displaystyle 1'' = (\frac{1}{60})~'[/tex][tex]\displaystyle 1'' = (\frac{1}{3600})^{0}[/tex]

Pembahasan

a. 110°36'

110°36'  = 110° + 36 '

           = 110° + ([tex]\displaystyle \frac{36}{60}[/tex])°

           = 110° + 0,6°

           = 110,6°

b. 197°22'30"​

197°22'30"​ = 197° + 22' + 30"

                  = 197° + 22' + ([tex]\displaystyle \frac{30}{60}[/tex])'

                  = 197° + 22' + 0.5'

                  = 197° + 22.5'

                  = 197° + ([tex]\displaystyle \frac{22,5}{60}[/tex])°

                  = 197° + 0,375°

                  = 197,375°

3x + 2y ≤ 18, x + 3y ≥ 12, x ≥ 0, y ≥ 0. Daerah penyelesaiannya adalah dapat dilihat di lampiran.

Untuk menentukan daerah himpunan penyelesaian pada pertidaksamaan linear yaitu: ax + by ≤ c dan ax + by ≥ c, dengan syarat koefisien y harus positif (b > 0), maka

Daerahnya diarsir ke atas jika ax + by ≥ c

Daerahnya diarsir ke bawah jika ax + by ≤ c

Pembahasan

3x + 2y ≤ 18 ⇒ diarsir ke bawah karena ≤

Jika x = 0 maka y = 9 ⇒ (0, 9)

Jika y = 0 maka x = 6 ⇒ (6, 0)

Hubungkan titik (0, 9) dan (6, 0), kemudian perpanjang dan arsir ke bawah maka diperoleh grafik 3x + 2y ≤ 18

x + 3y ≥ 12 ⇒ diarsir ke atas karena ≥

Jika x = 0 maka y = 4 ⇒ (0, 4)

Jika y = 0 maka x = 12 ⇒ (12, 0)

Hubungkan titik (0, 4) dan (12, 0), kemudian perpanjang dan arsir ke atas maka diperoleh grafik x + 3y ≥ 12

x ≥ 0

daerah arsirnya disebelah kanan sumbu y (diarsir ke kanan)

y ≥ 0

daerah arsirnya disebelah atas sumbu x (diarsir ke atas)

Daerah penyelesaian dari pertidaksamaan tersebut dapat dilihat di lampiran dengan titik-titik sudutnya adalah (0, 9), (0, 6) dan titik potong kedua garis
