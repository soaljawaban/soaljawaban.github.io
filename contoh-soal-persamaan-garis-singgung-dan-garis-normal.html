Penjelasan dengan langkah-langkah:

Persamaan Kurva

Diberikan :

y = 5x³-3x²+2x+1 dan

titik (-1,3) → x1 = -1 , y1 = 3

a) Persamaan garis singgung

Cari y' ( turunan dari persamaan kurva tersebut )

y = 5x³-3x²+2x+1

y' = 5(3x²) - 3(2x¹)+2(1x⁰)

y' = 15x²-6x+2

Gradien garis singgung

m = 15x²-6x+2 ...... subtitusi nilai x1

m = 15(-1)²-6(-1)+2

m = 15+6+2

m = 23

maka persamaan garis singgung

y - y1 = m(x-x1)

y - 3 = 23(x+1)

y - 3 = 23x+23

23x - y + 26 = 0 ✓

b) Persamaan garis normal

gradien garis normal

m.singgung × m.normal = -1

23 × m.normal = -1

m.normal = -1/23

maka persamaan garis normalnya

y - 3 = -1/23(x+1)

y - 3 = -1/23x - 1/23

1/23 x+y -3 + 1/23 = 0

1/23 x+y -68/23 = 0 ....... kedua ruas kali 23

x + 23y - 68 = 0 ✓

aplikasi turunan

SOAL :

Tentukan persamaan garis singgung dan persamaan garis normal dari kurva 3xy + y² = 2x - y di y = 0

PENYELESAIAN :

3xy + y² = 2x - y

3xy + y² - 2x + y = 0

saat y = 0 → x = 0

m = dy/dx = -Fx / Fy

m = dy/dx = - { ∂z/∂x } / { ∂z/∂y }

m = dy/dx = - { 3y - 2 } / { 3x + 2y + 1 }

m = dy/dx = - {-2} / {1}

m = dy/dx = 2

PGS :

y - y₁ = m (x - x₁)

y - 0 = 2(x - 0)

     y = 2x

PGN :

y - y₁ = -1/m (x - x₁)

y - 0 = -1/2 (x - 0)

     y = -1/2 x

KESIMPULAN :

Diperoleh pergasing y = 2x dan perganor y = -1/2 x

6. Tentukan persamaan garis singgung dan garis normal di titik (1,6) pada Kurva : y=3x²+2x-5 Tentukan Persamaan garis singgung dan garis normal di titik (0.-2) pada Kurva: x² - 2xy + y² - x +3y + 2 = 0


Persamaan garis singgung

y = 3x² + 2x - 5

m=y' = 6x + 2

y' = m = 8

y - y1 = m (x - x1)

y - 6 = 8 ( x - 1)

y = 8x - 8 + 6

y = 8x - 2

y - 8x + 2 = 0

8x - y -2 = 0

Saya bantu soal yang kedua, tapi garis singgung saja.

Untuk mencari persamaan garis singgung kurva di suatu titik, pertama kali yang harus kita lakukan yaitu memeriksa titik tersebut apakah benar pada kurva atau tidak.

Langkah 1 : (Pengujian)

Uji titik (0, -2) dengan cara mensubstitusikan kepada kurva sehingga :

x² - 2xy + y² - x + 3y + 2 = 0

0² - 2(0)(-2) + (-2)² - 0 + 3(-2) + 2 = 0

4 - 6 + 2 = 0

-2 + 2 = 0

0 = 0 (benar)

Langkah 2 : (Menentukan Gradien Garis Singgung Kurva)

Perlu dipahami, gradien garis singgung ditentukan oleh :

m = y'

Dengan menurunkan persamaan secara implisit, akan diperoleh :

2x-(2y+2xy')+2yy'-1+3y'=0

Langsung substitusi saja titiknya :

[tex]2(0)-(2(-2)+2(0)y')+2(-2)y'-1+3y'=0\\4-4y'+3y'-1=0\\-y'=-3\\y'=3[/tex]

Jadi, m = y' = 3.

Langkah 3 : (Menentukan Persamaan Garis Singgungnya)

Gunakan persamaan :

[tex]y-y_1=m(x-x_1)[/tex]

Substitusikan (0, -2) dan m = 3 ke persamaan tersebut, diperolehlah :

y - (-2)=3(x - 0)

y + 2 = 3x

y = 3x - 2

Langkah 4 : (Kesimpulan)

Jadi, persamaan garis singgungnya adalah y = 3x - 2.

Materi : Aplikasi Turunan

Tinjau Garis Singgung

Gradien garis singgungs suatu kurva dapat dicari menggunakan turunan pertama fungsi tersebut.

[tex]y=x^2-1\\\frac{dy}{dx}=x^2-1\\\frac{dy}{dx}_{x=2}=m=2^2-1=3\\\\y-y_1=m(x-x_1)\\y-3=3(x-2)\\y=3x-3\\3x-y-3=0[/tex]

Tinjau Garis Normal

Garis normal adalah garis yang tegak lurus dengan garis singgung kurva serta melewati titik dimana garis singgung menyinggung kurva.

Gradien yang tegak lurus dengan suatu garis tercapai dengan

[tex]m_1m_2=-1[/tex]

Karena [tex]m_1=3[/tex], maka

[tex]m_1m_2=-1\\3m_2=-1\\m_2=-\frac{1}{3}\\\\y-y_1=m_2(x-x_1)\\y-3=-\frac{1}{3}(x-2)\\3y-9=-x+2\\x+3y-11=0[/tex]

5y = -x + 31
x + 5y - 31 = 0 (bentuk implisit)

23. Diketahui f (x)=x2-1. Tentukan persamaan garis singgung dan persamaan garis normal ?


Untuk menentukan persamaan garis singgung dan persamaan garis normal dari fungsi f(x) = x^2 - 1, kita perlu menggunakan konsep turunan.

Langkah pertama adalah mencari turunan dari fungsi f(x):

f'(x) = 2x

Kita dapat menggunakan turunan ini untuk menentukan gradien (slope) dari garis singgung dan garis normal pada suatu titik pada kurva.

1. Persamaan Garis Singgung:
Garis singgung pada suatu titik (x, f(x)) pada kurva adalah garis yang memiliki gradien yang sama dengan turunan dari fungsi tersebut pada titik tersebut.

Maka, gradien garis singgung adalah f'(x) = 2x.
Pada titik (x, f(x)), gradien garis singgung adalah 2x.

Untuk menentukan titik potong garis singgung dengan kurva, kita perlu menyelesaikan persamaan berikut:
2x = f(x) = x^2 - 1

Dengan menggabungkan kedua persamaan tersebut, kita dapat menyelesaikannya untuk mendapatkan nilai x dan kemudian mencari nilai f(x) yang sesuai. Dalam hal ini, kita memerlukan metode numerik atau metode grafis untuk menentukan nilai x dengan lebih akurat.

2. Persamaan Garis Normal:
Garis normal pada suatu titik (x, f(x)) pada kurva adalah garis yang tegak lurus terhadap garis singgung pada titik tersebut. Gradien garis normal adalah negatif dari inversi gradien garis singgung.

Maka, gradien garis normal adalah -1/(2x).

Persamaan garis normal yang melalui titik (x, f(x)) adalah:
y - f(x) = -1/(2x)(x - x)

Sederhanakan persamaan tersebut sesuai kebutuhan.

Sekali lagi, untuk menentukan nilai x dan f(x) pada titik potong garis normal dengan kurva, kita perlu menggunakan metode numerik atau metode grafis.

Harap dicatat bahwa persamaan garis singgung dan garis normal yang dihasilkan dalam bentuk umum ini tergantung pada titik spesifik pada kurva.

30. pers garis singgung dan garis normal​


Persamaan garis singgung dan garis normal kurva [tex]y=2x^3-8[/tex] di titik (2,8) adalah [tex]\boldsymbol{y=24x-40}[/tex] dan [tex]\boldsymbol{24y=-x+194}[/tex].

PEMBAHASAN

Persamaan garis singgung adalah suatu persamaan garis yang menyinggung kurva di suatu titik. Sedangkan persamaan garis normal adalah suatu persamaan garis yang kedudukannya tegak lurus dengan persamaan garis singgung suatu kurva.

Karena saling tegak lurus maka perkalian gradien antara garis normal dan garis singgung bernilai -1

[tex]m_n\times m_s=-1[/tex]

dimana

[tex]m_s=\frac{d}{dx}[f(x)][/tex]

dengan :

[tex]m_n=[/tex] gradien garis normal

[tex]m_s=[/tex] gradien garis singgung

.

DIKETAHUI

Kurva [tex]y=2x^3-8[/tex]

.

DITANYA

Tentukan persamaan garis singgung dan persamaan garis normalnya di titik (2,8).

.

PENYELESAIAN

> Mencari persamaan garis singgung.

Kita cari dahulu gradiennya.

[tex]y=2x^3-8[/tex]

[tex]y'=3\times2x^2-0[/tex]

[tex]y'=6x^2[/tex]

.

Maka gradien garis singgung :

[tex]m_s=y'(2)[/tex]

[tex]m_s=6(2)^2[/tex]

[tex]m_s=24[/tex]

.

Sehingga persamaan garis singgungnya :

[tex]y-b=m(x-a)[/tex]

[tex]y-8=24(x-2)[/tex]

[tex]y=24x-40[/tex]

.

.

> Mencari persamaan garis normal.

Kita cari dahulu gradiennya.

[tex]m_n\times m_s=-1[/tex]

[tex]m_n\times24=-1[/tex]

[tex]m_n=-\frac{1}{24}[/tex]

.

Sehingga persamaan garis nomralnya :

[tex]y-b=m(x-a)[/tex]

[tex]y-8=-\frac{1}{24}(x-2)~~~~~...kedua~ruas~dikali~24[/tex]

[tex]24y-192=-x+2[/tex]

[tex]24y=-x+194[/tex]

.

KESIMPULAN

Persamaan garis singgung dan garis normal kurva [tex]y=2x^3-8[/tex] di titik (2,8) adalah [tex]\boldsymbol{y=24x-40}[/tex] dan [tex]\boldsymbol{24y=-x+194}[/tex].
