31. sebut dan jelaskan submenu dari menu favorites


menu favorites memiliki beberapa submenu yang berfungsi untuk menyimpan alamat suatu homepage yang kita inginkan ke harddisk dan mengaturnya sesuai keinginan kita. Fungsi submenu favorites sebagai berikut :

add and favorites untuk menyimpan alamat suatu web atau homepage yang kita inginkan ke dalam folder favorites untuk diakses kembali pada waktu lain Organize Favorites untuk mengatur alamat web atau homepage yang terdapat dalam folder favorites agar lebih rapih dan mudah mengaksesnya kembali Item lain, selain kedua submenu diatas bagian ini berisi dftar alamat yang pernah kita simpan dalam folder favorites. Jumlahnya tergantung dari alamat yang pernah disimpanPembahasan

Internet Explorer (IE) merupakan program web yang mudah digunakan. Melalui browser ini dapat memperoleh berbagai informasi dari berbagai alamat situs web di internet. Dengan mengetikan alamat situs tertentu browser akan mengantarkan ke situs tersebut ddan menampilkan halaman utama (homepage) di layar komputer.

Berikut adalah elemen-elemen browser IE

menu bar

menu bar disebut juga pulldown menu merupakan sederetan menu yang jika diklik akan memnunculkan sub menu dibawahnya. Menu bar terdiri dari menu File, Edit, View, Favorites, Tools, dan Help

menu File : menu file merupakan menu yang berhubungan dengan penanganan file. Isi submenu file antara lain :

New terdiri dari :

Windows untuk menampilkan jendela internet explorer yang baru Massage untuk membuat pesan mail baru Post untuk mengirim pesan baru Contact untuk membuat data baru di Address Book Internet Call untuk mengaktifkan NetMeeting

Open untuk membuka dokumen / file di hard disk atau mengakses alamat tertentu dari internet

Edit untuk menampilkan halaman ke program editor yang dapat mengedit file HTML seperti frontpage dan word

Save untuk menyimpan perubahan yang dilakukan pada konfigurasi dai internet explorer

Save as untuk menyimpan dokumen yang aktif menjadi file HTML atau TXT

Page Setup untuk mengubah ukuran kertas header dan footer, orientasi percetakan serta margin halaman dari dokumen aktif hendak dicetak

Print untuk mencetak dokumen yang aktif ke printer

Print Preview untuk menampilkan hasil cetakan pada layar kompute

Send untuk mengirim link atau halaman aktif melalui e-mail dan membuat shortcut untuk alamat dari dokumen yang aktif.

Import dan export untuk mengekspor dan mengimport file dari folder favorites dan  

bookmarks dari dan ke internet explorer.

Properties untuk menampilkan inforamasi mengenai dokumen yang aktif saat itu

Work offline untuk menampilkan halaman web tanpa terhubung ke internet (tanpa mendownloadnya)

Close untuk keluar dari internet explorer

Menu View

menu View terbagi dalam sub menu-submenu yang berfungsi untuk melakukan memanupulasi tampilan yang terlihat di layar monitor.secara rinci fungsi tersebut adaalah :

Toolbar untuk menampilkan atau menyembunyikan toolbar pada halaman internet explorer yang tampak dim layar monitor. Bila terdapat tanda (v ) didepan pilihan ini maka toolbar pilihan tersebut sedang aktif Status bar untuk menampilkan atau menghilangkan baris status pada halaman internet explorrer yang tampak dilayar monitor. Bila terdapat tanda (v ) didepan pilihan ini maka status bar tersebut sedang aktif Explorer bar untuk menampilkan atau menyembunyikan explorer bar pada halaman internet explorer. Bila terdapat tanda (v ) didepan pilihan ini maka explorer bar sedang aktif Go to untuk menampilkan halaman sebelumnya, berikutnya, atau alamat yang sudah pernah diakses sebelumnya Stop untuk menghentikan proses penampilan suatu halaman web yang sedang kita akses Refresh untuk menampilkan atau memproses kembali isi halaman web yang sedang aktif untuk memperbaharui isinya Text Size untuk mengubah ukuran teks atau huruf teks menjadi lebih kecil sedang besaar dari dokumen yang sedang kita aktifkan Encording untuk menentukan bahasa yang digunakan pada halaman web yang kita aktifkan Source untuk menentukan bentuk asli atau format asli pada halaman yang sedang aktif . format tampilanya biasanya berformat HTML Privacy Report untuk menampilkan kotak dialog privacy report yang berisi tentang halaman web yang sudah dibuka Full Screen untuk menampilkan jendela IE satu layar penuh tanpa baris judul, baris status, dan baris menu.

Menu Tool

menu tool terdiri atas submenu-submenu yang berfungsi untuk mengakses Outlook Express, melakukan singkronisasi semua isi data yang bisa ditampilkan secara offline, menga update komponen windows dari internet, dan sebagainya.
