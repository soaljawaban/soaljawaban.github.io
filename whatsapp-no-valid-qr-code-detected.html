4. yang bisa jawab cakep Valid No debat :)​


Jawab:

1. 48° R dan 52°R

2. 52°R dan 147 °F

3. 194° F dan 90°C

4. 212 F dan  80°R

Penjelasan dengan langkah-langkah:

1. Diketahui : °F = 140°,

  Ditanya : °R, °C

  Jawab :

  140° Fahrenheit = ….   °Reamur

                             = 4/9  x (°F-32)°  

                             = 4/9  x (140-32)°  

                             = 4/9  x 108°

                             = 48° R

  140° Fahrenheit = ….   °Celcius

                             = (°F − 32) × 5/9

                             = (140 °F − 32) × 5/9

                             = 60 °C

2. Diketahui : °C = 65°,

  Ditanya : °R, °F

  Jawab :

  65° Celcius = ….   °Reamur

                      = 4/5 x °C

                      = 4/5 x 65

                      = 260/5

                      = 52°R

  65° Celcius = ….   °Fahrenheit

                      = 9/5 x °C + 32

                      = 9/5 x 65 + 32

                      = 585/5+32

                      = 117 + 32

                      = 147 °F

3. Diketahui : °R = 72°,

  Ditanya : °C, °F

  Jawab :

  72° Reamur = ….   °Fahrenheit

                      = (9/4 x °R) + 32°F

                      = (9/4 x 72) + 32°F

                      = (9 x 18) + 32°F

                      = 162 + 32°F

                      = 194° F

 72° Reamur = ….   °Celcius

                     = 5/4 x °R

                     = 5/4 x 72

                     = 5 x 18

                     = 90°C

4. Diketahui : °C = 100°,

  Ditanya : °R, °F

  Jawab :

 100° Celcius = ….   °Fahrenheit

                      = (9/5 x °C) + 32 F

                      = (9/5 x 100) + 32 F

                      = 180 + 32

                      = 212 F

 100° Celcius = ….   °Reamur

                      = 4/5 x °C

                      = 4/5 X 100  

                      = 80°R
