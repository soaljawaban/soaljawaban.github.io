5. change the sentence into interrogative "the black cat has attacked my cat twice"


Kalimat Interrogative adalah ➡️ Has the black cat attacked my cat twice?

PEMBAHASANPresent Perfect

Digunakan untuk menyatakan Kegiatan atau Peristiwa yg sudah selesai tapi masih ada kaitannya di masa sekarang.

Rumus Present Perfect(+) S + has/have + V3 + O/C(-) S + has/have + not + V3 + O/C(?) has/have + S + V3 + O/C?

Maka kalimatnya(+) the black cat has attacked my cat twice(-) the black cat has not attacked my cat twice(?) has the black cat attacked my cat twice?

Perintah soal

Change the sentence into interrogative

=> Ubahlah kalimat berikut menjadi kalimat tanya (?)

"the black cat has attacked my cat twice"Keterangan:Subject => The black catHas + Verb-3 => has attacked Object => my catComplement => twice

Maka kalimat Interrogative

=> Has the black cat attacked my cat twice? ✔️
