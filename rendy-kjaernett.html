NILAI ADI 87 DAN NILAI RENDI 83

Penjelasan dengan langkah-langkah:

misalkan :

adi = x

rendi = y

[tex] 73 = \frac{(24 \times 72) + x + y}{24 + 2} \\ 73 \times 26 = 1728 + x + y \\ 1898 - 1728 = x + y \\ x + y = 170 \: ... \: (1)[/tex]

y = x - 4 (masukkan ke persamaan 1)

x + (x - 4) = 170

2x = 174

x = 87

y = 87 - 4

= 83
