Cara-Cara

1. Buka Control Panel dengan cara mengetikkan “Control Panel” di kolom pencarian di samping Start Menu.

2. Kemudian, pilih System and Security dan tekan menu Administrative Tools.

3. Akan muncul halaman baru yang berisikan berbagam macam fitur dengan fungsi yang berbeda-beda. Di sini, Anda pilih Services saja.

4. Scroll ke bawah sampai Anda menemukan Windows Update, dan klik dua kali. Muncul halaman Windows Update Properties.

5. Sekarang, ganti opsi pada Startup Type menjadi Disabled, dan klik tombol Stop pada Service Status. Kemudian tekan Ok.

17. Bukan Tugas atau QuizBagaimana caranya Update dari Windows XP ke Windows 7 ?[ Jika tidak tahu, jangan jawab pertanyaan ini dan laporkan saja pertanyaan ini ]​


Jawaban:

1. Pertama mount Installasi Windows 7 dengan menggunakan software seperti Daemon tools. kemudian jalankan “setup.exe” dan pilih “Install Now“.

2. Pada tahap Get Important Update, jika pada saat installasi komputer kamu terkoneksi ke internet, kamu bisa memilih “Go online to get the latest update for installation“. Tetapi jika tidak terkoneksi dengan internet kamu sebaiknya memilih opsi “Do not get latest updates for installation“.

3. Ikuti terus tahap installasi windows 7 seperti biasa. dan pada tahap “Which type of installation do you want?” Pastikan kamu memilih “Custom“.

4. Kemudian pilihan partisi yang berisi sistem Windpws XP. dan nantinya jika ada notifikasi Windows.old, klik OK.

5. Setelah itu kamu bisa lanjutkan proses installasi Windows 7 hingga selesai

6. Setelah proses upgrade dan instalasi diatas selesai, sebenarnya Windows 7 tersebut sudah siap untuk digunakan. Namun ada satu hal cukup penting yang masih harus kamu lakukan, terutama jika ingin mendapatkan tambahan free space storage di HDD kamu.

Ketika kamu sudah selesai menginstall Windows 7, cek dan pastikan bahwa semua bisa berjalan baik tanpa masalah. Jika kamu tidak menemukan masalah apapun di Windows 7 tersebut, maka kamu bisa menghapus Windows.old

26. jelaskan manfaat windows update!


Memahami Pembaruan Windows adalah salah satu layanan yang disediakan oleh Microsoft untuk sistem operasi Windows bagi pengguna untuk memperbarui sistem Windows itu sendiri.

Pembahasan :

Fungsi Pembaruan Windows:

1. Memeriksa Kebijakan Pembaruan: Layanan Pembaruan Windows ini memungkinkan pengguna untuk memeriksa atau memeriksa sebelumnya terkait dengan menyetujui atau memperbarui sistem operasi Windows.

2. Mengatasi Kesalahan:ada prinsipnya, sebuah sistem tidak terlepas dari yang namanya bug atau error.  

3. Meningkatkan Keamanan:Fitur Windows Update ini akan men-download database terkait keamanan sehingga sistem keamanan lebih up so far terhadap rentannya virus yang terus berkembang.  

4. Memperbarui dan Menambah Sistem: Sistem yang diperbarui atau terdapat penambahan sistem yang penting dapat mengatasi error atau bug. Hal yang paling penting ialah perangkat tetap selalu up to date terhadap pembaruan atau pemutakhiran terhadap berbagai jenis sistem yang tersedia.

5. Membantu Perangkat Lunak Secara Efisien: Menggunakan, fitur ini sangat berguna untuk sistem mutakhir yang akan membantu perangkat berjalan sesuai fungsinya, yaitu menjaganya tetap efisien.

windows 7

Fitur di Windows 7 kurang complete.

windows  95

Kinerja lebih baik dari Windows 95

Kecepatan waktu start up dan shutdown yang lebih cepat dari Windows 95

Windows 98 juga memiliki kemampuan mendownload sendiri file-file update Windows 98.

Kemampuan multi display card yang dimiliki Windows 98

Internet Explorer 4.0 yang terintegrasi dengan desktop sehingga memudahkan browsing internetUtility baru untuk tune up dan repair Windows,peningkatan dalam menjalankan  

kemampuan multimedia baik dalam kualitas grafis maupun kualitas suara

Dukungan yang lebih baik bagi device baru seperti USB,DVD, dll

Bugs di Windows 98 semakin sedikit, dan komputer yang menggunakan sistem operasi ini semakin jarang hang dibandingkan  

Windows 95Lebih cepat jika digunakan untuk mengakses internet Cocok untuk maen game-game ringan atau game playstation 1 dan jalannya pun stabil

KEKURANGAN WINDOWS 95

Tampilannya kurang menarik dan masih terlalu kaku.

Kestabilan sistem kurang  

Windows 98 lebih boros memori dan ruang harddisk daripada Windows 95

Sering terjadinya blue screen

windows 98

Kekurangan :

* Dukungan support sudah dicabut oleh produsennya (microsoft)

* Susah cari driver, karena produsen hardware menarik dukungan untuk win 98

* Tidak mendukung dan tidak bisa diinstall di perangkat terbaru (bila dipaksakan akan blue screen)

* Tidak ada versi 64bit

* Ringkih, mudah crash

* Kemampuan multi tasking yang sangat buruk

* Kemampuan multi thread yang sangat buruk

* Penanganan device yang buruk, DMA sering konflik

* Manajemen resource cpu yang kacau balau

* Tidak bisa membuat file dg ukuran lebih dari 2GB

* Tidak bisa mendeteksi memory besar (1GB masih stabil, dipaksa 2GB-hasil BlueScreen)

* Utility sangat terbatas, apalagi tool network

* Gampang terserang virus, gampang di hack, di crack, gampang di acak-acak

* Terlalu beresiko melakukan transaksi perbankan dari browser win 98, karena fitur networknya minim

* Tidak mendukung applikasi keluaran terbaru

* Tampilan terlalu sederhana (meskipun sempat dibanggakan di era nya)

Kelebihan :

* Cocok untuk hardware lama (misalnya pentium I,II,III)

* Ringan

* Ukuran installer kecil, bisa diinstal dari disket

* Bisa diinstall dari dos

* Bisa booting ke dos

* Cocok untuk development applikasi under dos terutama applikasi recovery

* Cocok untuk main game tempoe doeloe

* System (boot) mudah ditransfer ke media lain seperti cd dan disket

* Dos protected mode (DPMI) yang sempurna, baik untuk applikasi DOS32 dan DOS16

* Sangat cocok untuk development bagi programmer BIOS, atau programmer bahasa mesin

5. Jelas lebih secure di banding 98 dan stabil.

1. Biaya Operasi relatif mahal.

2. Bila server keseluruhan tidak akan berjalan dan megalami gangguan.

3. Jika tidak ada server Windows 2000 Server client tidak bisa mangoneksikan ke server begitupun sebaliknnya.

5. Terlalu banyak service yang tidak berguna.

39. jelaskan manfaat dari windows update


Windows Update adalah perangkat lunak bawaan Microsoft untuk sistem operasinya Microsoft Windows. Sesuai dengan nama dan artinya, update artinya pembaruan


Manfaat dari Windows Update antara lain :

1. Memperbaiki celah keamanan yang ada di sistem operasimu

2. Menambah dan menghapus fitur yang ada dikomputermu

3. Melakukan penginstalan pembaruan program dan driver secara menyeluruh

4. Meningkatkan build Windows. Misalnya dari build 7600 menjadi build 7601
