Pedagang tersebut mempunyai modal Rp400.000.000,00 dan mengharapkan keuntungan Rp500.000,00 dari setiap penjualan laptop. Tentukan model matematika untuk menunjukkan keuntungan maksimum yang dapat diperoleh pedagang tersebut.


Jawab:

keuntungan max = 30.000.000

Penjelasan dengan langkah-langkah:

dik x = i3, ->> 5000.000

y = i5, ->>7500.000

max laptop = 60

modal = 400.000.000

untung per unit = 500.000

ok pertama tama, marilah kita mengucapkan syukur kepada Tuhan YME.haha

5000.000x +7500.000y ≤ 400.000.000 (sederhanakan dibagi 1000.000)

1.--> 5x +7.5 y  ≤ 400

so , 5x +7.5 y  ≤ 400

jika x = 0 maka,

7.5 y  ≤ 400

y ≤ 400/7.5

y ≤ 53

jika y = 0 maka,

5x ≤ 400

x ≤400/5

x = 80

2.-->x+y  ≤  60

jika x = 0 maka,

y ≤60

jika y = 0 maka,

x ≤60

3. --> selanjutnya

Eliminasi x

5x + 7.5 y ≤ 400  ║ x1

x +y≤60               ║ x5

5x + 7.5 y ≤ 400

5x +5y  ≤  300

-----------------------------   -

2.5 y ≤  100

y ≤ 40

terus cari nilai x dimana y = 40

x +y ≤ 60

x + 40 ≤  60

x ≤ 20

dari ketigak persamaan diatas cari keuntungan maksimal yang di dapat

persamaan pertama x,y = 80,53

persamaan kedua x,y = 60,60

persamaan ketiga x,y = 20,40

dari ketiga persamaan diatas didapat nilai x dan y. kemudian cari keuntungan masing masing x dan y nya.

titik 1 x,y = 0,53

keuntungan = 500.000x +500.000y

keuntungan = 0 + 500.000* 53

keuntungan = 2650.0000

titik 2 x,y = 20,40

keuntungan = 500.000x +500.000y

keuntungan = 500.000*20 +500.000*40

keuntungan = 30.000.000

titik 3 x,y = 60,0

keuntungan = 500.000x +500.000y

keuntungan = 500.000*60 +0

keuntungan = 30.000.000

jadi keuntungan maksimal bisa kita lihat pada data diatas. yaitu 30.000.000
