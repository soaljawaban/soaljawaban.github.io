1. Apakah aman bagi saya untuk meng - update Windows 7 ke Windows 10 menggunakan Media Creation Tool? Yang jago IT tolong jawab ya :3


Mungkin lebih tepatnya "upgrade". Untuk beberapa alasan media creation tool adalah cara paling aman dan mudah untuk melakukan upgrade dari versi Windows lama (Windows 7 & 8) ke versi terbaru yaitu Windows 10, terlebih untuk pengguna Windows yang awam.

Pembahasan:

Media creation tool adalah sebuah program yang didesain oleh MIcrosoft untuk mempermudah aktivitas yang berkaitan dengan instalasi Windows yang diantaranya:

Melakukan upgrade ke versi Windows terbaru secara langsungMembuat media instalasi pada USB atau DVDMengunduh file (ISO) instalasi Windows saja

Dalam hubungannya dengan pertanyaan yang kamu ajukan upgrade dari Windows versi lama ke Windows 10 menggunakan media creation tool akan memberikan keuntungan seperti

Bisa memilih antara arsitektur 32 dan 64 bitProses/langkahnya yang mudah dimengertiAman karena jika yang dipilih adalah upgrade maka kamu akan memiliki data cadangan dari Windows terdahulu. Jadi kalau ternyata Windows 10 dirasa tidak cocok dengan sumber daya komputer kamu (lemot dll) maka kamu bisa kembali ke windows 7 melalui data cadangan tadi

Sebagai catatan pastikan ruang penyimpanan yang tersedia lebih dari 3GB, dan koneksi internetnya harus stabil.

13. fungsi Artistic media tool pada CorelDraw?​


Jawaban:

Penjelasan:

FUNGSI BAGIAN ARTISTIC MEDIA TOOL

Preset : adalah bentuk seni menggunakan �Preset� yang menekankan bidang tipis dan tebal yang bisa dikreasikan sedemikian rupa sehingga mendapatkan bidang yang menarik. Pilihan preset ini sangat beragam seperti yang sudah di preview di bagian Corel Draw Originals.

Brush : menggoreskan kuas dengan menggunakan kurva yang menyerupai outline. Didalam mode brush ini ternyata masih terdapat tipe kuas yang bisa dipakai, seperti : Artistic, Calligraphy, Object, Scroll, Sensual, Splatter, Symbol, Texture yang kesemuanya memiliki karakteristik yang berbeda.

Spray : menghasilkan objek yang menyerupai semprotan. Dalam kategorinya juga terdapat bermacam-macam jenis objek artistic nya. Alat media tool yang memiliki banyak kategori ini juga sangat berguna untuk mendapatkan clip art yang menarik seperti daun, buah, permen, jejak kaki dan lain sebagainya.

Calligraphic : goresan bullpen bisa didapatkan dengan menggunakan media jenis ini. Ketebalan serta kemiringannya pun bisa disesuaikan sebelum, sesaat atau sesudah digunakan. Alat ini sangat cocok untuk digunakan pada pembuatan tulisan indah baik secara Arabic ataupun alphabet biasa. Biasanya pada kenyataannya untuk mendapatkan tulisan yang indah maka alat tulis yang digunakan harus di peges ujungnya untuk mendapatkan kemiringan yang diinginkan.

Pressure : Menghasilkan goresan seperti boxy pen dengan ketebalan yang bisa disesuaikan sesuai keinginan.

15. Bagaimana cara menghilangkan windows error recovery


mapel : TI
kelas : 12

Windows Error Recovery dapat dihilangkan jika kamu mengetahui permasalahan yang terjadi pada komputermu baik hardware maupun software. ketika terjadi hal ini, mungkin komputermu mengingatkanmu untuk mengganti file atau memperbaiki hardware yang bersangkutan

Windows Error Recovery ada 2
a. dikarenakan masalah kecil
b. dikarenakan kerusakan file Windows/hardware

untuk permasalahan pertama:
coba ganti PSU atau masalah pada kestabilan listrik

kalau masalah kedua:
kalau kamu sudah pernah kena Blue Screen dan muncul Windows Error Recovery, coba startup repair (perbaiki hal yang salah dengan komputer)

tetapi kalau kamu belum pernah kena Blue Screen tetapi muncul Windows Error Recovery, kerusakan ada di hardware (layaknya hardisk). lakukan startup repair, cari kode erroronya di internet dan lakukan tindakan

tanya jika tidak jelas
