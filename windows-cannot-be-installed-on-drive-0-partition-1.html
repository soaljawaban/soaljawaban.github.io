18. Jelaskan arti dan manfaat dari istilah berikut. Kemudian tunjukkan wujud implementasi dari penjelasan anda : a. Shrink Hardiks Drive b. Extend Partition c. Sharing d. Permision / hak akses


Jawaban:

a. Shrink Hard Disk Drive:

Shrink Hard Disk Drive adalah istilah yang digunakan dalam administrasi sistem komputer untuk mengurangi ukuran partisi pada hard disk drive. Proses ini memungkinkan pengguna untuk mengambil ruang yang ada di dalam partisi dan mengalokasikannya ke partisi lain atau membuat partisi baru. Hal ini bermanfaat ketika pengguna ingin mengoptimalkan penggunaan ruang disk atau membagi ruang disk menjadi partisi yang lebih kecil.

Implementasi: Misalnya, jika pengguna memiliki partisi hard disk drive dengan ukuran 100 GB dan ingin mengurangi ukurannya menjadi 80 GB, mereka dapat menggunakan utilitas manajemen disk seperti "Disk Management" di Windows untuk melakukan proses shrink pada partisi tersebut.

b. Extend Partition:

Extend Partition adalah istilah yang digunakan dalam administrasi sistem komputer untuk memperluas ukuran partisi pada hard disk drive. Proses ini memungkinkan pengguna untuk mengambil ruang yang tidak teralokasikan atau diambil dari partisi lain dan menambahkannya ke partisi yang ada. Hal ini bermanfaat ketika pengguna ingin memperluas kapasitas penyimpanan pada partisi tertentu yang hampir penuh.

Implementasi: Sebagai contoh, jika pengguna memiliki partisi hard disk drive dengan ukuran 50 GB dan ruang tidak teralokasikan sebesar 20 GB, mereka dapat menggunakan utilitas manajemen disk untuk memperluas partisi tersebut menjadi 70 GB dengan menambahkan ruang tidak teralokasikan ke partisi tersebut.

c. Sharing:

Sharing adalah istilah yang merujuk pada proses berbagi atau membagikan sumber daya, seperti file, folder, atau printer, antara pengguna atau perangkat yang berbeda dalam jaringan komputer. Hal ini memungkinkan pengguna untuk mengakses dan menggunakan sumber daya tersebut dari perangkat lain dalam jaringan.

Implementasi: Sebagai contoh, dalam sebuah jaringan rumah atau kantor, pengguna dapat membagikan folder berisi file dokumen kepada pengguna lain dalam jaringan. Dengan mengatur izin atau hak akses yang tepat, pengguna lain dapat mengakses dan membuka file-file tersebut melalui jaringan.

d. Permission / Hak Akses:

Permission atau hak akses merujuk pada izin yang diberikan kepada pengguna atau grup pengguna untuk mengakses, memodifikasi, atau melakukan tindakan tertentu terhadap suatu sumber daya, seperti file, folder, atau aplikasi. Hak akses ini digunakan untuk mengontrol tingkat akses dan keamanan dalam sistem komputer.

Implementasi: Misalnya, dalam sistem operasi seperti Windows, pengguna dapat diberikan hak akses tertentu pada file atau folder. Hak akses ini bisa berupa hak baca, tulis, atau eksekusi. Dengan memberikan hak akses yang sesuai, administrator sistem dapat membatasi atau mengatur apa yang dapat dilakukan oleh pengguna terhadap suatu sumber daya.

39. Jelaskan bagaimana cara mengunci folder di drive windows 7 ?


Jadi, di Windows 7,8 dan 10 sudah terdapat sistem enkripsi folder yang bisa kamu gunakan untuk mengamankan folder dan isinya.

Berikut cara menggunakannya:

1. Arahkan ke file atau folder yang ingin kamu enkripsi, pada contoh ini saya akan memilih sebuah folder.

2. Klik kanan di file atau folder tersebut lalu klik Properties.

3. Pada menu General, klik tombol Advanced.

4. Centang bagian Encrypt contents to secure data, kemudian klik OK lalu Apply.

5. Akan muncul pop up Windows yang memintamu untuk memilih apakah ingin melakukan enkripsi di folder only atau folder, subfolder dan file didalamnya. Lalu klik OK.

6. Kamu akan melihat tanda gembok kecil berwarna kuning di folder atau file yang terenkripsi sebagai tanda. Metode enkripsi ini akan melindungi file penting milikmu dari tangan yang tidak bertanggungjawab. Enkripsi ini tersambung dengan akun Windows komputermu. Jadi saat kamu log in menggunakan akun Windows-mu, file tersebut akan terdekripsi otomatis. Simpelnya, cara ini hanya akan melindungin file terenkripsi jika kamu log out dari akun Windows-mu dan orang lain log in dengan akun Windows-nya lalu mencoba membuka file yang kamu enkripsi.
