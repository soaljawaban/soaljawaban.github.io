39. smga yang bantu dapet pahala​


Jawaban:

C

Penjelasan dengan langkah-langkah:

Diketahui f: x → 1/2 (x - 3) dengan Df {x | x ≤ 7,x ∈ R}. Daerah hasil fungsi f adalah ...

A. {y | y ≤ 7, y ∈ R}

B. {y | y ≥ 7, y ∈ R}

C. {y | y ≤ 2, y ∈ R}

D. {y | y ≥ 2, y ∈ R}

E. {y | y ≤ -2, y ∈ R}

Pembahasan

Diketahui f : x → (x - 3), maka f(x) = (x - 3)

Menentukan dua titik yg dilalui f(x) = (x - 3)

Daerah asal Df = {x | x ≤ 7, x ∈ R}, maka daerah asal = {1, 2, 3, 4, 5, 6, 7}

Titik potong

untuk x = 3

f(x) = (x - 3)

= (3 - 3)

= 0

Titik (3, 0)

Untuk x = 7

f(x) = (7 - 3)

= (4)

= 2

titik (7, 2)

Daerah asal Df = {x | x ≤ 7, x ∈ R} sehingga titik ujung paling kiri tak hingga dan titik ujung paling kanan (7, 2) tidak berlubang. Gambar sketsa grafik fungsinya bisa dilihat pada lampiran.

Dari gambar diperoleh nilai y = f(x) terendah adalah negatif tak hingga sehingga y > -∞.

Nilai y = f(x) tertinggi adalah y = 2 dan titik ujung (7, 2) tidak berlubang sehingga y ≤ 2.

Jadi daerah hasil fungsi f adalah {y | y ≤ 2, y ∈ R}.

Jawaban : C
