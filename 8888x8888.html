12. Dibawah ini terdapat 10 sampel data nilai Statistika dan data nilai Matematika yang diambil dari siswa kelas Z sebagai berikut: NO 1234 g 6 7 8 9 10 Nilai Statistik (X) 65 60 70 60 65 80 65 70 70 75 Nilai Matematika (M) 80 86 80 83 8888888 84 90 78 87 80 Apakah terdapat hubungan antara nilai Statistika dengan nilai Matematika kelas Z disamping: a. Buatlah hipotesis hubungan dari soal diatas? b. Seberapa besar koefisien korelasinya? C. Diketahui tabel r dengan n=10 derajat kepercayaan 95% / a= 5%, r = 0,497, buatlah kesimpulannya?​


Jawaban:

a. Hipotesis hubungan dari soal di atas dapat dirumuskan sebagai berikut:

- Hipotesis Nol (H0): Tidak ada hubungan antara nilai Statistik (X) dengan nilai Matematika (M) di kelas Z.

- Hipotesis Alternatif (H1): Terdapat hubungan antara nilai Statistik (X) dengan nilai Matematika (M) di kelas Z.

b. Untuk menghitung koefisien korelasi antara nilai Statistik (X) dengan nilai Matematika (M), kita dapat menggunakan rumus koefisien korelasi Pearson. Koefisien korelasi Pearson (r) adalah ukuran statistik yang mengukur sejauh mana hubungan linear antara dua variabel. Rumusnya adalah sebagai berikut:

r = (Σ((X - X_mean) * (M - M_mean))) / (sqrt(Σ(X - X_mean)^2) * sqrt(Σ(M - M_mean)^2))

Dengan X adalah variabel nilai Statistik, M adalah variabel nilai Matematika, X_mean adalah rata-rata nilai Statistik, dan M_mean adalah rata-rata nilai Matematika.

c. Dalam tabel r dengan n=10 dan derajat kepercayaan 95% (α=0,05), tidak diberikan nilai kritis r yang spesifik untuk r = 0,497. Namun, kita dapat membuat kesimpulan berdasarkan hipotesis dan koefisien korelasi yang ditemukan.

Jika nilai p-value (nilai signifikansi) dari uji statistik untuk koefisien korelasi lebih kecil dari α (0,05), maka kita dapat menolak hipotesis nol (H0) dan menyimpulkan bahwa terdapat hubungan yang signifikan antara nilai Statistik dengan nilai Matematika. Sebaliknya, jika nilai p-value lebih besar dari α, maka kita gagal menolak hipotesis nol dan tidak dapat menyimpulkan adanya hubungan yang signifikan antara kedua variabel.

Dalam kasus ini, karena tidak diberikan nilai p-value, kita tidak dapat membuat kesimpulan yang pasti tentang hubungan antara nilai Statistik dan Matematika berdasarkan tabel r. Untuk membuat kesimpulan yang lebih akurat, diperlukan analisis statistik lanjutan dengan menggunakan perangkat lunak statistik yang sesuai atau bantuan dari pakar statistik.

Jawaban

Baik, berikut adalah perhitungan Total Cost (TC), Average Fixed Cost (AFC), Average Variable Cost (AVC), Average Cost (AC), dan Marginal Cost (MC) berdasarkan data yang diberikan:

- Fixed Cost (FC) = Rp. 60.000 ribu

1. Untuk Produksi (Q) = 0:

  - Total Cost (TC) = FC + VC = 60 + 0 = Rp. 60.000 ribu

2. Untuk Produksi (Q) = 1:

  - Variable Cost (VC) = 6 ribu

  - Total Cost (TC) = FC + VC = 60 + 6 = Rp. 66.000 ribu

3. Untuk Produksi (Q) = 3:

  - Variable Cost (VC) = 15 ribu

  - Total Cost (TC) = FC + VC = 60 + 15 = Rp. 75.000 ribu

4. Untuk Produksi (Q) = 6:

  - Variable Cost (VC) = 30 ribu

  - Total Cost (TC) = FC + VC = 60 + 30 = Rp. 90.000 ribu

5. Untuk Produksi (Q) = 10:

  - Variable Cost (VC) = 50 ribu

  - Total Cost (TC) = FC + VC = 60 + 50 = Rp. 110.000 ribu

6. Untuk Produksi (Q) = 13:

  - Variable Cost (VC) = 65 ribu

  - Total Cost (TC) = FC + VC = 60 + 65 = Rp. 125.000 ribu

Selanjutnya, kita dapat menghitung Average Fixed Cost (AFC), Average Variable Cost (AVC), Average Cost (AC), dan Marginal Cost (MC) menggunakan rumus yang telah disebutkan sebelumnya:

- Average Fixed Cost (AFC) = FC / Q

- Average Variable Cost (AVC) = VC / Q

- Average Cost (AC) = TC / Q

- Marginal Cost (MC) = ΔTC / ΔQ

Berikut adalah hasil perhitungannya:

a. Total Cost (TC) pada masing-masing tingkat produksi:

  - TC(Q=0) = Rp. 60.000 ribu

  - TC(Q=1) = Rp. 66.000 ribu

  - TC(Q=3) = Rp. 75.000 ribu

  - TC(Q=6) = Rp. 90.000 ribu

  - TC(Q=10) = Rp. 110.000 ribu

  - TC(Q=13) = Rp. 125.000 ribu

b. Average Fixed Cost (AFC) pada masing-masing tingkat produksi:

  - AFC(Q=0) = FC / Q = Rp. 60.000 ribu / 0 = Tidak dapat ditentukan (divisi dengan nol)

  - AFC(Q=1) = FC / Q = Rp. 60.000 ribu / 1 = Rp. 60.000 ribu

  - AFC(Q=3) = FC / Q = Rp. 60.000 ribu / 3 = Rp. 20.000 ribu

  - AFC(Q=6) = FC / Q = Rp. 60.000 ribu / 6 = Rp. 10.000 ribu

  - AFC(Q=10) = FC / Q = Rp.

60.000 ribu / 10 = Rp. 6.000 ribu

  - AFC(Q=13) = FC / Q = Rp. 60.000 ribu / 13 = Rp. 4.615,38 ribu

c. Average Variable Cost (AVC) pada masing-masing tingkat produksi:

  - AVC(Q=0) = VC / Q = Tidak dapat ditentukan (divisi dengan nol)

  - AVC(Q=1) = VC / Q = 6 ribu / 1 = Rp. 6.000 ribu

  - AVC(Q=3) = VC / Q = 15 ribu / 3 = Rp. 5.000 ribu

  - AVC(Q=6) = VC / Q = 30 ribu / 6 = Rp. 5.000 ribu

  - AVC(Q=10) = VC / Q = 50 ribu / 10 = Rp. 5.000 ribu

  - AVC(Q=13) = VC / Q = 65 ribu / 13 = Rp. 5.000 ribu

d. Average Cost (AC) pada masing-masing tingkat produksi:

  - AC(Q=0) = Tidak dapat ditentukan (TC(Q=0) / Q=0)

  - AC(Q=1) = TC(Q=1) / Q=1 = Rp. 66.000 ribu / 1 = Rp. 66.000 ribu

  - AC(Q=3) = TC(Q=3) / Q=3 = Rp. 75.000 ribu / 3 = Rp. 25.000 ribu

  - AC(Q=6) = TC(Q=6) / Q=6 = Rp. 90.000 ribu / 6 = Rp. 15.000 ribu

  - AC(Q=10) = TC(Q=10) / Q=10 = Rp. 110.000 ribu / 10 = Rp. 11.000 ribu

  - AC(Q=13) = TC(Q=13) / Q=13 = Rp. 125.000 ribu / 13 = Rp. 9.615,38 ribu

e. Marginal Cost (MC) pada masing-masing tingkat produksi:

  - MC(Q=0) = Tidak dapat ditentukan (ΔTC(Q=0) / ΔQ=0)
