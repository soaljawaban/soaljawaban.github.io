Penjelasan:

Mencari panjang AE :

AB² = AE² + BE²

AE² = AB² - BE²

AE² = 7² - 1²

AE = √49 - 1

AE = √48

AE = 4√3 cm

Mencari AF :

DF = AE - FC

DF = 4√3 - 2

AF² = AD² + DF²

AF² = 4² + (4√3 - 2)²

AF² = 16 + 48 - 16√3 + 4

AF² = 68 - 16√3

Mencari resultan menggunakan sumbu x dan y :

AFx = DF

AFx = 4√3 - 2 cm

AFy = AD

AFy = 4cm

AEx = 4√3 cm

AEy = 0 cm

R = √Rx² + Ry²

R = √(AFx + AEx)² + (AFy + AEy)²

R = √(4√3 - 2 + 4√3)² + (4 + 0)²

R = √(8√3 - 2)² + 4²

R = √192 - 32√3 + 4 + 16

R = √212 - 32√3 cm
