2. hitunglah jarak episentrum gempa berikut ini​


1. Diketahui :

Gelombang sekunder = 21. 36'50"Gelombang primer = 21. 35'20"

Ditanya : Jarak episentrum ?

Rumus : delta (segitiga) = (S-P)-1'×1.000 km

delta = jarak episentrumS = gelombang sekunderP = gelombang primer1' = satu menit1.000 km

Dijawab = (21. 36'50" - 21. 35'20") - 1' × 1.000 km

= (1,5 - 1) × 1.000 km

= 0,5 × 1.000 km

= 500 km

10. Sebuah stasiun pencatat gempa mencatat gelombang primer suatu gempa terjadi pada pukul 19.30’20"" dan gelombang sekunder tercatat pada pukul 19.35’50"". hitunglah jarak episentrum gempa.


✏Jawaban :

4.500 km

♻ Penjelasan :

Diketahui :

→ Gelombang Primer ( P ) = 19°30’20""

→ Gelombang Sekunder ( S ) = 19°35’50""

Ditanya :

→ Jarak episentrum gempa ( Δ ) = ??

Penyelesaian :

Δ = ( ( S - P ) - 1' ) × 1.000 km )

= ( ( 19°35’50"" - 19°30’20"" ) - 1' ) × 1.000 km )

= ( 5'30" - 1' ) × 1.000 km )

= 4'30" × 1.000

= ( 4 × 1.000 km ) + ( 30/60 × 1.000 km )

= 4.000 km + 500 km

= 4.500 km

Jadi, jarak episentrum gempa dengan stasiun pemantau tersebut adalah 4.500 km

27. jelaskan klasifikasi gempa berdasarkan jarak dan episentrumnya


Berdasarkan bentuk episentrum

(a) Gempa linear, yaitu gempa yang episentrumnya berbentuk garis. Gempa tektonik merupakan gempa linear. Salah satu akibat tektonisme adalah patahan.
(b) Gempa sentral, yaitu gempa yang episentrumnya berupa titik. Gunung api pada erupsi sentral adalah sebuah titik letusan, demikian juga runtuhan retak bumi.

Gempa setempat: < 10.000 km.
Gempa jauh: sekitar 10.000 km.
Gempa sangat jauh: > 10.000 km

[tex]\boxed{\sf{\Delta =( (S - P) - 1') \times 1000 }}[/tex]

[tex]\Delta[/tex] = Jarak Episentrum

S = Gelombang sekunder

P = Gelombang primer

⠀ ⠀ ⠀

Diketahui

S = 10.22'50"

P = 10.20'30"

⠀ ⠀ ⠀

Ditanyakan

[tex]\Delta[/tex] = ...?

⠀ ⠀ ⠀

Penjelasan

[tex]\sf{\Delta =( (S - P) - 1') \times 1000 }[/tex]

[tex]\sf{\Delta =( (22'50" - 20'30") - 1') \times 1000 }[/tex]

[tex]\sf{\Delta =( (2'20") - 1') \times 1000 }[/tex]

[tex]\sf{\Delta =( 2,333' - 1') \times 1000 }[/tex]

[tex]\sf{\Delta = 1,333' \times 1000 }[/tex]

[tex]\sf{\Delta = 1.333 }[/tex]

⠀ ⠀ ⠀

Jarak Episentrumnya 1.333 km

38. sebutkan contoh gempa hiposentrum dan episentrum


Hiposentrum merupakan pusat gempa yang berada di kedalaman bumi. Titik pusat gempa yang berada di permukaan bumi dan tegak lurus dengan hiposentrum disebut episentrum. Episentrum merupakan permukaan bumi atau dasar laut yang menjadi pusat merambatnya getaran bumi.

39. carilah jarak episentrum dari soal berikit :stasiun pencatat gempa b ,mencatat gelombang primer suatu gempa pukul 15.40'.20",dan gelombang scunder pukul 15.37'.30",tentukan jarak episentrum gempa dari stasiun pencatat gempa.


Mata pelajaran: Geografi
Kelas: 10 SMA
Kategori: Tenaga Endogen
Kode kategori berdasarkan KTSP: -
Kata kunci: epicentrum gempa, menghitung jarak epicentrum gempa

Jawaban:
Diketahui:
gelombang primer pukul 15.40'.20",
gelombang sekunder pukul 15.37'.30",
Ditanya:
tentukan jarak episentrum gempa dari stasiun pencatat gempa?
Jawab:
Delta = ((S-P) – 1’) X 1.000 Kilometer)
= (( 15.40'.20" - 15.37'.30") - 1') x 1000 km
= ((2'50") - 1') x 1000 km
= 1'50" x 1000 km
= 1.500 km

Jadi jarak epicentrum gempa tersebut dari stasiun pencatat gempa adalah 1.500 km.

Penjelasan:
Episentrum gempa adalah gelombang gempa yang berada di permukaan bumi dan tegak lurus dengan hiposentrum kemudian menyebar ke berbagai arah. Untuk menghitung jarak episentrum gempa dapat menggunakan metode episentral berikut:

METODE EPISENTRAL
Episentral ialah jarak episentrum atau pusat gempa di stasiun pencatat gempa. Untuk menentukan episentrum dengan menggunakan metode episentral diperlukan minimal tiga stasiun pengamat yang mencatat kejadian gempa, sehingga dapat dihitung jarak episentral masing-masing stasiun. Untuk menghitung jarak episentral digunakan rumus LASKA, sebagai berikut:

Delta = ((S-P) – 1’) X 1.000 Kilometer)

Ket:
Delta = jarak episentral dari stasiun pengamat dalam satuan kilometer
S = gelombang sekunder (dalam menit)
P = gelombang primer (dalam menit)
1’ = 1 menit = 60 detik (AS)
