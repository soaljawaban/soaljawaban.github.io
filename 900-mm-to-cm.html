14. 0,5 km² berapa hm²? , 9,753 dam² berapa dm²? , 900 mm² berapa cm? , 3m² berapa dam²? , 400.000dm² berapa hm²plis bantu jawab mau di kumpul besok​


Jawaban:

Untuk mengonversi satuan luas, kita perlu memperhatikan faktor konversi antara satuan tersebut. Berikut adalah konversi yang diperlukan untuk masing-masing pertanyaan:

1. 1 km² = 100 hm²

Untuk mengonversi 0,5 km² menjadi hm², kita dapat menggunakan rumus berikut:

0,5 km² x 100 = 50 hm²

Jadi, 0,5 km² adalah sama dengan 50 hm².

2. 1 dam² = 100 dm²

Untuk mengonversi 9,753 dam² menjadi dm², kita dapat menggunakan rumus berikut:

9,753 dam² x 100 = 975,3 dm²

Jadi, 9,753 dam² adalah sama dengan 975,3 dm².

3. 1 mm² = 0,01 cm²

Untuk mengonversi 900 mm² menjadi cm², kita dapat menggunakan rumus berikut:

900 mm² ÷ 100 = 9 cm²

Jadi, 900 mm² adalah sama dengan 9 cm².

4. 1 m² = 0,01 dam²

Untuk mengonversi 3 m² menjadi dam², kita dapat menggunakan rumus berikut:

3 m² ÷ 0,01 = 300 dam²

Jadi, 3 m² adalah sama dengan 300 dam².

5. 1 dm² = 0,0001 hm²

Untuk mengonversi 400.000 dm² menjadi hm², kita dapat menggunakan rumus berikut:

400.000 dm² x 0,0001 = 40 hm²

Jadi, 400.000 dm² adalah sama dengan 40 hm².

28. QUIZ (896/900)➠ Keliling persegi panjang yang memiliki panjang 57 mm dan lebar 28 mm ?Pakai cara!​​​​​​​​​​​​​​​​​​​​​​​​​​​​​​​​


[tex]\LARGE {\purple { diketahui } } [/tex]

Panjang = 57 mm

Lebar = 28 mm

[tex]\LARGE {\purple { ditanya } } [/tex]

Keliling persegi panjang?

[tex]\LARGE {\purple { jawaban } } [/tex]

Cara 1

Keliling = Panjang + Lebar + Panjang + Lebar

Keliling = 57 mm + 28 mm + 57 mm + 28 mm

Keliling = 85 mm + 57 mm + 28 mm

Keliling = 142 mm + 28 mm

Keliling = 170 mm

Cara 2

Keliling = 2 × ( Panjang + Lebar )

Keliling = 2 × ( 57 mm + 28 mm )

Keliling = 2 × 85 mm

Keliling = 170 mm

Cara 3

Keliling = ( 2 × Panjang ) + ( 2 × Lebar )

Keliling = ( 2 × 57 mm ) + ( 2 × 28 mm )

Keliling = 114 mm + 56 mm

Keliling = 170 mm

[tex]\LARGE {\purple { kesimpulan } } [/tex]

Maka keliling persegi panjang yang memiliki panjang 57 mm dan lebar 28 mm adalah 170 mm

♢ Pertanyaan :

Keliling persegi panjang yang memiliki panjang 57 mm dan lebar 28 mm ? ... .

♢ Penyelesaian :

Keliling = 2 × (P + L)tentukan keliling

Maka,

= 2 × (P + L)

= 2 × (57 + 28)mm

= 2 × 85 mm

= [tex]\boxed{\green{\sf{170 \: mm}}}[/tex]

♢ Kesimpulan :

Jadi, Hasil dari keliling persegi P.57 mm dan L.28 mm adalah [tex]\colorbox{aqua}{170 mm}[/tex]
