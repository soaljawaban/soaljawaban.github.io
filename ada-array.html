1. definisi array dan konsep dasar array dan macam-macam array??​


Array adalah tipe data terstruktur yang berfungsi untuk menyimpan sejumlah data yang tipenya sama.

Bagian yang menyusun array disebut dengan isi atau elemen array, masing-masing elemen bisa diakses melalui indeks array.

Pembahasan

Array dan variabel biasa memiliki perbedaan, yakni :

Array :

Dapat memiliki sebuah nilai Misalnya ada 4 data, maka hanya cukup menggunakan 1 nama variabel : x[4]

Variable biasa :

Hanya dihubungkan melalui sebuah nilai sajaMisalkan ada 4 data, maka menggunakan nama variable : x1, x2, x3, x4.

Mendeklarasikan variable array ada 2 cara, yakni :

Mendeklarasikan variablel array tanpa memasukan nilai ke dalam variable tersebut. Penulisannya : int barang [3];Mendeklarasikan variable array dengan memasukkan nilainya sekaligus ke dalam variable tersebut. Penulisannya  : int barang [6] = {22,12,8,10};

Pendahuluan

Matriks adalah susunan bilangan atau simbol yang disusun dalam bentuk baris dan kolom sehingga membentuk suatu bangun persegi.

Jenis –jenis Matriks

• Matriks Baris

adalah matriks yang hanya terdiri satu baris.

• Matriks Kolom

adalah matriks yang hanya terdiri satu kolom.

• Matriks Nol

adalah matriks yang semua elemennya berupa bilangan nol.

• Matriks Persegi

Matriks ini biasanya terjadi ketika jumlah baris sama dengan jumlah kolom.

• Matriks Diagonal

adalah matriks persegi dengan bilangan pada posisi diagonalnya tidak nol.

7. Terdapat Array : A [2][2] maka jumlah elemen Array tersebut adalah …


Terdapat Array: A[2][2] maka jumlah elemen array tersebut adalah 4 elemen. Untuk mengetahui jumlah elemen pada array dua dimensi tersebut, kita dapat menganggap array tersebut sebagai sebuah matriks, dimana dengan ukuran 2x2 maka, akan didapat empat elemen didalam matriks.

Pembahasan

Array sendiri adalah suatu struktur data yang menyimpan sekelompok elemen dimana setiap elemen akan ditandai dengan suatu index. Array dapat berupa array satu dimensi, dua dimensi, maupun multi dimensi. Dalam konteks soal ini, array yang digunakan adalah array dua dimensi. Array dua dimensi dapat diidentifikasi dengan memiliki dua index penanda seperti, alamat suatu titik pada diagram kartesius. Bentuk umum dari array dua dimensi:

TipeData Variabel [x][y]

Contoh:

int contohMatriks [4][3]

Tipe data dapat berupa int, float, char, dan sebagainyaVariabel merupakan penamaan array yang disesuaikan kebutuhanX jika dianalogikan dalam matriks adalah jumlah barisY jika dianalogikan dalam matriks adalah jumlah kolom

Sehingga dalam array: A[2][2], maka memiliki dua baris dan dua kolom, dengan index:

A[0][0]   A[0][1]

A[1][0]    A[1][1]

23. [tex]\left[\begin{array}{ccc}3&y\\5&-1\end{array}\right] + \left[\begin{array}{ccc}x&5\\-3&6\end{array}\right] -\left[\begin{array}{ccc}-3&-1\\y&9\end{array}\right] =\left[\begin{array}{ccc}8&5x\\-x&-4\end{array}\right][/tex] carilah nilai x dan y


Penjelasan dengan langkah-langkah:

Untuk mencari nilai x dan y, kita perlu menyamakan elemen-elemen matriks pada kedua sisi persamaan.

Diberikan persamaan:

[tex]\left[\begin{array}{ccc}3&y\\5&-1\end{array}\right] + \left[\begin{array}{ccc}x&5\\-3&6\end{array}\right] -\left[\begin{array}{ccc}-3&-1\\y&9\end{array}\right] =\left[\begin{array}{ccc}8&5x\\-x&-4\end{array}\right][/tex]

Kita akan menyelesaikan persamaan ini dengan membandingkan elemen-elemen matriks pada kedua sisi persamaan:

1. Elemen pada baris pertama dan kolom pertama:

3 + x + 3 = 8

6 + y + 1 = 5x

Dari persamaan pertama, kita dapat mencari nilai x:

3 + x + 3 = 8

x + 6 = 8

x = 2

Dari persamaan kedua, kita dapat mencari nilai y:

6 + y + 1 = 5x

y + 7 = 5(2)

y + 7 = 10

y = 3

Jadi, nilai x = 2 dan nilai y = 3.

Operasi hitung campuran adalah penghitungan di mana terdapat dua tanda penghitungan yang berbeda dalam satu soal.

Operasi hitung campuran dibagi menjadi 4 yaitu :

1️⃣ Perkalian

Perkalian adalah salah satu dari empat dasar operasi matematika dari aritmetika, dengan yang lainnya adalah penambahan, pengurangan dan pembagian. Hasil dari operasi perkalian disebut darab.

2️⃣ Pembagian

Pembagian adalah operasi aritmetika dasar yang merupakan kebalikan dari operasi perkalian. Operasi pembagian ini dinotasikan dengan tanda (÷) (division) atau / (slash).

3️⃣ Penjumlahan

Penambahan adalah salah satu dari empat operasi aritmetika dasar. Penjumlahan merupakan penambahan sekelompok bilangan atau lebih menjadi suatu bilangan yang disebut jumlah.

4️⃣ Pengurangan

Pengurangan adalah operasi aritmetika yang mewakili operasi menghapus objek dari koleksi. Pengurangan ditandai dengan tanda minus ( - )

Aturan-aturan operasi hitung campuran yaitu :

➡️ Operasi yang berada dalam kurung dikerjakan lebih dahulu.

➡️ Dahulukan bilangan berpangkat.

➡️ Dahulukan perkalian dan pembagian sebelum penjumlahan dan pengurangan.

➡️ Jika ada perkalian dan pembagian tanpa tanda kurung, kerjakan berurutan dari kiri ke kanan.

Langkah-langkah dalam menghitung operasi hitung campuran yaitu :

✔️Operasi penjumlahan dan pengurangan sama kuat.

✔️ Operasi perkalian dan pembagian sama kuat.

✔️ Operasi perkalian dan pembagian lebih kuat dibandingkan operasi penjumlahan atau pengurangan.
