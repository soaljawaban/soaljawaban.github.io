Berapakah jumlah dari usia mereka sekarang jika tahun depan usia Bahrun sama dengan z kali usia Nisak? Jika diketahui x, y, dan z berturut-turut adalah banyaknya bacaan ikhfa pada surat Ad-Duha, banyaknya huruf idghom bigunnah, dan urutan sifat "Baqo" pada urutan sifat wajib Allah SWT. ​


Diketahui:

x = banyaknya bacaan ikhfa pada surat Ad-Duha

y = banyaknya huruf idghom bigunnah

z = urutan sifat "Baqo" pada urutan sifat wajib Allah SWT

Ditanya:

Jumlah usia Ahmad Bahrun dan Nisak sekarang

Langkah Penyelesaian:

1. Misalkan usia Ahmad sekarang adalah A usia Bahrun sekarang adalah B dan usia Nisak sekarang adalah N.

2. Menurut informasi pertama x tahun yang lalu usia Ahmad setengah dari usia Bahrun. Jadi usia Ahmad x tahun yang lalu adalah (A - x) dan usia Bahrun x tahun yang lalu adalah (B - x). Dari informasi ini kita bisa merumuskan persamaan: A - x = (B - x)/2.

3. Menurut informasi kedua usia Nisak sekarang sama dengan usia Ahmad y tahun yang lalu. Jadi usia Nisak sekarang adalah (A - y). Dari informasi ini kita bisa merumuskan persamaan: N = A - y.

4. Menurut informasi ketiga tahun depan usia Bahrun sama dengan z kali usia Nisak. Jadi usia Bahrun tahun depan adalah (B + 1) dan usia Nisak tahun depan adalah (N + 1). Dari informasi ini kita bisa merumuskan persamaan: B + 1 = z(N + 1).

5. Menggabungkan persamaan (1 (3 dan (4 kita dapatkan:

A - x = (B - x)/2 --> Persamaan (1)

N = A - y --> Persamaan (2)

B + 1 = z(N + 1) --> Persamaan (3)

6. Selanjutnya kita perlu mencari persamaan A B dan N.

Dari Persamaan (2 kita bisa mendapatkan A = N + y.

Substitusikan A dengan N + y ke Persamaan (1):

N + y - x = (B - x)/2

2(N + y - x) = B - x

2N + 2y - 2x = B - x

B = 2N + 2y - x --> Persamaan (4)

7. Substitusikan B yang ada di Persamaan (4) ke Persamaan (3):

2N + 2y - x + 1 = z(N + 1)

2N + 2y - x + 1 = zN + z

N + 2y - x + 1 = (z - 1)N + z

N - (z - 1)N = z - 2y - 1 + x

N(1 - (z - 1)) = z - 2y - 1 + x

N(z - 1) = x - z + 2y + 1 --> Persamaan (5)

8. Substitusikan Persamaan (5) ke Persamaan (2):

x - z + 2y + 1 = A - y

x - z + 3y + 1 = A

A = x - z + 3y + 1 --> Persamaan (6)

9. Substitusikan A dari Persamaan (6) dan B dari Persamaan (4) ke Persamaan (1):

x - z + 3y + 1 = 2N + 2y - x

2x + x - z + 3y - 2y - 1 = 2N

3x + y - z - 1 = 2N

N = (3x + y - z - 1)/2 --> Persamaan (7)

10. Setelah mendapatkan A B dan N kita dapat mencari jumlah usia mereka sekarang:

Jumlah usia sekarang = A + B + N

= (x - z + 3y + 1) + (2N + 2y - x) + N

= (x - z + 3y + 1) + (2*(3x + y - z - 1)/2 + 2y - x) + (3x + y - z - 1)/2

Jadi jumlah usia mereka sekarang adalah (x - z + 3y + 1) + (2*(3x + y - z - 1)/2 + 2y - x) + (3x + y - z - 1)/2.
