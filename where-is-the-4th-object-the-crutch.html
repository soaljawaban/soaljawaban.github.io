6. An object is placed 10 cm from a concave mirror. The focal length is 5 cm. The image distance of the object is


So, the image distance of the object is 10 cm behind the concave mirror.

Explanation :

A concave mirror (in Bahasa called "cermin cekung") is a type of mirror that the mirror can make the light focus and can reflect light on the same time. A concave mirror have positive focus. So we can calculate another part with this formula :

[tex]\boxed{\bold {\frac{1}{f}=\frac{1}{s} + \frac{1}{s'}}}[/tex]

If :

f = the focus of mirror (cm)s = distance object from mirror (cm)s' = distance shadow/image from mirror (cm)Work Steps :

We know that :

s = distance object from mirror = 10 cm f = the focus of mirror = 5 cm

Question : s' = distance shadow/image from mirror = ... cm

Answer :

[tex] \frac{1}{f}=\frac{1}{s} + \frac{1}{s'} [/tex]

[tex] \frac{1}{5}=\frac{1}{10} + \frac{1}{s'} [/tex]

[tex] \frac{1}{5} - \frac{1}{10} = \frac{1}{s'} [/tex]

[tex] \frac{2}{10} - \frac{1}{10} = \frac{1}{s'} [/tex]

[tex] \frac{1}{10} = \frac{1}{s'} [/tex]

[tex] \boxed{s' = 10 \: cm} [/tex]

30. The height of the shadow is 5 cm and the real height of the object is 2 cm. The magnification of the object is


Jawaban:

M = 2,5 x

Penjelasan:

hi = 5 cm

ho = 2 cm

M = hi/ho = 5/2 = 2,5

Because the magnification of shadow is 2,5. So, the shadow is 2,5 times bigger than the object's height.

Explanation :

Magnification (M) is the ability of an optical device to be able to change its actual size at its shadow. If the magnification of the shadow more than 1 ( M > 1), the size of shadow will bigger than the actual size. If the magnification is 1 (M = 1), the shadow will show actual size of the original. If the magnification is under 1 (M < 1), the shadow will smaller than the actual size. For calculate the magnification of the shadow, we can use this formula :

[tex] \boxed {\bold{|M| = |\frac{h'}{h}|}} \:\:\: or \:\:\:\boxed {\bold{|M| = |\frac{s'}{s}|}} [/tex]

If :

|M| = magnification => always positive h' = shadow height (cm or m) h = actual height (cm or m) s = distance the actual object to optical device (cm or m) s' = distance the shadow to optical device (cm or m) Work Steps :

We know that :

h' = shadow's height = 5 cm h = object's height = 2 cm

Question : |M| = magnification = ... ?

Answer :

[tex] |M| = |\frac{h'}{h}|[/tex]

[tex] |M| = |\frac{5}{2}| = \boxed {2,5}[/tex]
