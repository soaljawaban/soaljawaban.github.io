7. hasil dari perintah : =MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum",12),10),1,7)Adalah...


hasil dari perintah : =MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum",12),10),1,7) Adalah...Pendahuluan

Microsoft Excel atau nama lengkapnya Microsoft Office Excel adalah sebuah perangkat lunak atau program yang berfungsi untuk membuat dokumen lembar kerja berupa spreadsheet yang dibuat dan didistribusikan oleh perusahaan pengembang software bernama Microsoft Corporation. Program Microsoft Office Excel ini dapat dijalankan pada sistem operasi Microsoft Windows dan Mac OS.

Pembahasan

Fungsi MID pada microsoft excel adalah mengambil karakter dari suatu teks mulai dari start index sebanyak jumlah tertentu.

Format fungsi MID adalah MID(text; start_index; jumlah_char)

misalnya MID( "abcdef" ; 1 ; 2) maka keluaranya adalah ab, karena karakter yang diambil mulai dari index pertama sebanyak 2 karakter.

Fungsi LEFT pada microsoft excel adalah mengambil karakter dengan jumlah tertentu mulai dari sebelah kiri.

Format fungsi LEFT adalah LEFT(text ; jumlah_char)

Fungsi RIGHT pada microsoft excel adalah mengambil karakter dengan jumlah tertentu mulai dari sebelah kanan.

Format fungsi RIGHT adalah RIGHT (text ; jumlah_char)

keluaran dari perintah =MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum",12),10),1,7) sebenarnya adalah ERROR. Karena tanda pemisah pada perintah tersebut bukanlah koma ( , ) melainkan titik koma ( ; ).

Penulisan yang benar adalah =MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum";12);10);1;7)

Pengerjaan perintah bersarang (nested function) dijalankan dari yang paling dalam yaitu RIGHT("Selamat Mengerjakan Ulangan Umum";12).

Keluaran perintah tersebut adalah 12 karakter dari kanan yaitu "Ulangan Umum".

Kemudian perintah yang akan dijalankan selanjutnya adalah LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum";12);10). Karena fungsi yang didalamnya sudah dikerjakan terlebih dahulu maka perintah tersebut bisa kita tuliskan:

LEFT("Ulangan Umum";10), keluaran dari perintah tersebut adalah 10 karakter dari sebelah kiri yaitu "Ulangan Um"

Kemudian pada perintah terakhir MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum";12);10);1;7) bisa kita tuliskan seperti berikut ini

MID("Ulangan Um" ; 1 ; 7)

Keluaran dari perintah tersebut adalah 7 karakter dimulai dari index 1, sehingga keluaranya adalah "Ulangan"

Kesimpulan

hasil dari perintah : =MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum",12),10),1,7) adalah ERROR karena salah menuliskan syntax pemisah antar variabel. Seharusnya ; malah ,

Hasil dari penulisan perintah yang benar = MID(LEFT(RIGHT("Selamat Mengerjakan Ulangan Umum";12);10);1;7) adalah "Ulangan"

Jika kita ingin menampilkan MEN saja, perlu mengetikkan rumus... a. =LEFT("DOKUMEN",3) b. =LEFT("DOKUMEN",7,3) c. =RIGHT("DOKUMEN",3) d. =RIGHT("DOKUMEN",7,3)


9. Jika kita ingin menampilkan KUME saja, perlu mengetikkan rumus:
=MID("DOKUMEN",3,4) (opsi d)

10. Jika kita ingin menampilkan MEN saja, perlu mengetikkan rumus:
=RIGHT("DOKUMEN",3) (opsi c)
 

Pembahasan

Untuk menampilkan string/teks di bagian tengah dalam string/teks lain, pada Excel digunakan rumus/formula:
=MID(string, posisi_awal, banyak_karakter)

Kita ingn menampilkan "KUME" yang diambil dari string "DOKUMEN".

Posisi huruf K adalah pada karakter ketiga, maka posisi awal = 3. Banyak karakter pada "KUME" adalah 4.

Jadi, rumus yang dapat menampilkan "KUME" dari string "DOKUMEN" adalah:
=MID("DOKUMEN",3,4)

Sedangkan jika kita ingin menampilkan bagian akhir (bagian paling kanan) dari string/teks lain, pada Excel digunakan rumus/formula:
=RIGHT(string, banyak_karakter)

Kita ingin menampilkan "MEN" yang diambil dari string "DOKUMEN". Hal ini berarti kita ingin menampilkan 3 karakter terakhir dari string "DOKUMEN".

Maka, rumus yang dapat menampilkan "MEN" dari string "DOKUMEN" adalah:
=RIGHT("DOKUMEN",3)

33. Himpunan penyelesaian dari pertidaksamaan [tex]\sqrt{7 x-3} \geq 2 x[/tex] adalah .... A. [tex]\left\{x \mid x \leq \frac{3}{4} \text { atau } x \geq 1\right\}[/tex] B. [tex]\left\{x \mid x \leq \frac{3}{2} \text { atau } x \geq 2\right\}[/tex] C. [tex]\{x \mid x \leq 1 \text { atau } x \geq 2\}[/tex] D. [tex]\left\{x \mid \frac{3}{4} \leq x \leq 1\right\}[/tex] E. [tex]\left\{x \mid \frac{3}{2} \leq x \leq 2\right\}[/tex]


Pertidaksamaan dari √7x - 3 ≥ 2x maka himpunana penyelesaiannya adalah {x | 3/4 ≤ x ≤ 1}.

→ Jawabannya adalah D.

Penjelasan dan langkah-langkah

Diketahui:

Pertidaksamaan = √7x - 3 ≥ 2x

Ditanyakan:

Himpunan penyelesaian dari pertidaksamaan √7x - 3 ≥ 2x = ?

Jawab:

Langkah pertama kuadratkan kedua persamaan untuk menghilangkan akar.

√7x - 3    ≥ 2x

(√7x - 3)² ≥ (2x)²

   7x - 3   ≥  4x²

           0  ≥  4x² - 7x + 3

           0  ≥ (x - 1)(4x - 3)

Dua nilai x yaitu: x = 1 atau x = 3/4

Langkah kedua mencari nilai x yang memenuhi syarat:

7x - 3 ≥ 0

 x      ≥ 3/7

Langkah ketiga menentukan nilai akhir x yang sesuai dengan garis bilangan. Perpotongan dua garis bilangan dari dua langkah di atas adalah:

x ≥ 3/4 atau x ≤ 1

Maka penyelesaian himpunannya yaitu {x | 3/4 ≤ x ≤ 1}
