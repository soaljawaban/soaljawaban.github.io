4. Hasil dari [tex] \sqrt{289} + \sqrt{196} + {24 }^{2} - {18}^{2} [/tex]adalah​


Penyelesaian soal :

[tex] \sf \sqrt{289} + \sqrt{196} + {24}^{2} - {18}^{2} [/tex]

[tex] \sf = 17 + \sqrt{196} + {24}^{2} - {18}^{2} [/tex]

[tex] \sf = 17 + 14 + {24}^{2} - {18}^{2} [/tex]

[tex] \sf = 17 + 14 + (24 - 18) - (24 + 18)[/tex]

[tex] \sf = 17 + 14 + 6(24 + 18)[/tex]

[tex] \sf = 17 + 14 + 6(42)[/tex]

[tex] \sf = 17 + 14 + (6 \times 42)[/tex]

[tex] \sf = 17 + 14 + 252[/tex]

[tex] \sf = 283[/tex]

Soal ✏

Hasil dari - 168 : 24 × ( - 28 ) adalah ...

Menjawab ⤵

Pembahasan ✒

Rumus Positif dan Negatif ✔

[tex]( + ) \times ( + ) = ( + )[/tex]

[tex]( + ) \times ( - ) = ( - )[/tex]

[tex]( - ) \times ( + ) = ( - )[/tex]

[tex]( - ) \times ( - ) = ( + )[/tex]

Note :

Ingat !!!.

SAPOBENE ☞ Sama Positif Beda Negatif.

Penyelesaian ✒

[tex] - 168 \div 24 \times ( - 28) = ...[/tex]

[tex](( - 168) \div 24) \times ( - 28) = ...[/tex]

[tex] = 196.[/tex]

Jadi, hasilnya adalah 196. ✔
