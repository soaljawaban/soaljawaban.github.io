11. pada peta pulau Lombok jarak mantaram ke Ampenan adalah 3 cm jika skala 1:3.500.000 Berapa km jarak Mataram ke Ampenan sebenarnya ​


Jawaban :

105 km

Penjelasan : → Diketahui :

Jarak Pada Peta ( JP ) = 3 cm

Skala ( S ) = 1 : 3.500.000

→ Ditanya :

Jarak Sebenarnya  ( JS ) = ????

→ Penyelesaian :

JS = JP × S          

= 3 × 3.500.000      

= 10.500.000  cm

Ubah ke satuan km

JS =  10.500.000 × 1 / 100.000      

= 10.500.000 / 100.000      

= 105 km

Jadi, jarak sebenarnya Mataram ke Ampenan  adalah 105 km
