4. Mary memiliki beberapa kue. Setelah makan 1 kue, mary memberikan 1/2 dari sisanya ke adiknya. Setelah makan 1 kue lagi, mary memberikan 1/2 dari sisanya untuk adiknya. Mary sekarang hanya memiliki 5 kue. Berapa jumlah awal kue yang dimiliki mary?


Jumlah awal kue Mary adalah 23.

Pembahasan

Diketahui:

Mary memiliki beberapa kue.

Setelah makan 1 kue, mary memberikan 1/2 dari sisanya ke adiknya.

Setelah makan 1 kue lagi, mary memberikan 1/2 dari sisanya untuk adiknya.

Mary sekarang hanya memiliki 5 kue.

Ditanya:

Berapa jumlah awal kue yang dimiliki mary?

Jawab:

Kue Mary = x

Sisa pertama kue mary = x - 1

Diberikan kepada adiknya untuk pertama kalinya = [tex]\frac{1}{2}[/tex] (x - 1)

Sisa kedua =  [tex]\frac{1}{2}[/tex] (x - 1) - 1

Diberikan kepada adiknya untuk kedua kalinya = [tex]\frac{1}{2}[/tex] { [tex]\frac{1}{2}[/tex] (x - 1) - 1}

[tex]\frac{1}{2}[/tex] { [tex]\frac{1}{2}[/tex] (x - 1) - 1} = 5

{ [tex]\frac{1}{2}[/tex] (x - 1) - 1} = 5 * 2

{ [tex]\frac{1}{2}[/tex] (x - 1) - 1} = 10

[tex]\frac{1}{2}[/tex] (x - 1) = 10 + 1

[tex]\frac{1}{2}[/tex] (x - 1) = 11

x - 1 = 11 * 2

x - 1 = 22

x = 22 + 1

x = 23

Jumlah awal kue Mary adalah 23.

6. Mary, berumur 16 tahun dan empat kali umur saudara laki-laki Mary. Berapa umur Mary ketika umurnya dua kali umur saudara laki-laki nya?


[tex]Misal\ Mary = M \\\\ dan \\\\ Saudara = S \\\\ M = 4S \\\\ 16 = 4S \\\\ S = \frac{16}{4} \\\\ \boxed { S = 4 } \\\\ M + x = 2(S + x) \\\\ 16 + x = 2S + 2x \\\\ 2x - x = 16 - 2(4) \\\\ x = 16 - 8 \\\\ x = 8 \\\\ Jadi\ umur\ Mary\ ketika\ umurnya\ 2\ kali\ umur\ saudaranya\ adalah : \\\\ 16 + 8 \\\\ \boxed { \boxed { \bold { = 24\ tahun } } }[/tex]Misalkan umur Mary = m.
Misalkan umur saudara Mary = s.

16 = 4s

16 = 4s
  4 = s

 m + a = 2(s + a)
16 + a = 2(4 + a)
16 + a = 8 + 2a
 a - 2a = 8 - 16
      -a = -8
       a = 8

m = a + 16
   = 8 + 16
   = 24

Jadi umur Mary ketika umurnya dua kali umur saudaranya adalah 24 tahun.

9. Mary, berumur 16 tahun dan empat kali umur saudara laki-laku Mary. Berapa umur Mary ketika umurnya dua kali umur saudara laki-laki nya?


umur Mary = 4 x umur adik
16 = 4 x umur adik
umur adik = 16 : 4 = 4 tahun

berapa umur Mary bila 2 kali umur adiknya?
(2 x usia adik) + usia Mary = (2 x 4) + 16  = 24 tahun

Misalkan, Mary = M, Saudara Mary = S

Mary berumur 16 tahun dan 4 kali dari umur saudaranya, persamaannya adalah :

[tex]$\begin{align} M&=4A\\16&=4A\\A&=\frac{16}{4}\\A&=4\ $tahun \end[/tex]

Umur Mary ketika umurnya 2 kali umur saudaranya, cari pertambahan umurnya :

[tex]$\begin{align} \frac{M+p}{A+p}&=\frac21\\\frac{4A+p}{A+p}&=2\\\frac{4\times4+p}{4+p}&=2\\16+p&=2p+8\\2p-p&=16-8\\p&=8\ $tahun \end[/tex]

Sehingga, umur Mary ketika umurnya 2 kali umur saudaranya adalah :

[tex]$\begin{align} M&=16+p\\&=16+8\\&=\boxed{24\ $tahun$} \end[/tex]
