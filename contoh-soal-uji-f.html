Jika diberikan f(x) = 2x + 4 dan g(x) = x + 1, kita diminta untuk mencari hasil dari (f ° g)(x) atau f(g(x)) saat x = -1.

Untuk mencari hasil tersebut, kita perlu menggantikan x pada fungsi g(x) dengan nilai -1, kemudian menggunakan hasilnya untuk menggantikan x pada fungsi f(x).

Langkah-langkahnya adalah sebagai berikut:

1. Gantikan x pada fungsi g(x) dengan -1:

  g(-1) = (-1) + 1

        = 0

2. Gantikan x pada fungsi f(x) dengan hasil g(-1) dari langkah sebelumnya:

  f(g(-1)) = f(0)

           = 2(0) + 4

           = 0 + 4

           = 4

Jadi, hasil dari (f ° g)(x) atau f(g(x)) saat x = -1 adalah 4. Oleh karena itu, jawaban yang benar adalah tidak ada dalam pilihan yang diberikan.
