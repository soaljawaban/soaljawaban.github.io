Jawaban:

1. 你叫什么名字 ?

Nǐ jiào shén me míngzì ?

=> Siapa nama kamu?

2. 我不要吃饭 !

Wǒ bù chī fàn !

=> Saya tidak mau makan nasi !

3. 你好吗?

Nǐ hǎo ma ?

=> Apa kamu baik-baik saja?

4. 你是不是不要读书了?

Nǐ shì bù shì bùyào dúshūle ?

=> Apakah kamu sudah tidak mau sekolah lagi?

Pembahasan :

你 Nǐ => Kamu

我 Wǒ => Saya

什么 Shén me => Apa, kenapa

不 Bù => Tidak, gak

吗 Ma => Kata kerja

是不是 Shì bùshì => Apakah
