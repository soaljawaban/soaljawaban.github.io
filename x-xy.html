25. x² + √xy + y² = 168 x - √xy+y=10 Jumlah semua nilai x + √xy + y yang mungkin adalah.... bantuin ya dapat poin banyak


Jawaban :

2x + 2y - 10.

Penjelasan :

Mari kita selesaikan sistem persamaan ini secara bertahap. Pertama, mari kita cari nilai dari √xy.

Dari persamaan kedua, kita dapat menyederhanakan persamaan tersebut menjadi:

x - √xy + y = 10

√xy = x + y - 10

Kemudian, kita substitusikan nilai √xy ke persamaan pertama:

x² + (x + y - 10) + y² = 168

x² + y² + x + y - 10 = 168

x² + y² + x + y = 178

Sekarang, kita ingin mencari nilai dari x + √xy + y. Kita tahu bahwa:

√xy = x + y - 10

√xy + 10 = x + y

Maka, substitusikan ke persamaan yang telah kita sederhanakan sebelumnya:

x + √xy + y = (x + y) + (√xy + 10)

x + √xy + y = (x + y) + (x + y - 10)

x + √xy + y = 2x + 2y - 10

Jadi, jumlah semua nilai x + √xy + y yang mungkin adalah 2x + 2y - 10.
