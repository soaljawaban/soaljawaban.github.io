6. bagaimana cara mengatasi printer canon ip 2700 series error 0×00000709 ?​


1. Mengubah User Account Default Dengan Yang Baru

Cara pertama yang dapat kita lakukan untuk menghilangkan pesan error 0x00000709 printer dan bisa kembali membuat set default printer adalah dengan mengubah User Account Komputer Anda. User Account yang selama ini Anda gunakan harus diubah dengan User Account yang baru. Alasannya karena User Account tersebut sudah mengidap penyakit yang menyebabkan error 0x00000709 printer.

Untuk mengubah sekaligus membuat User Account baru caranya mudah, silakan Anda pergi ke Control Panel. Mengaktifkan Control Panel bisa melalui Start kemudian Control Panel atau dengan jalan pintas Run dengan cara menekan Windows+R pada keyboard kemudian ketik Control Panel > Enter.

Jika layar kerja Control Panel sudah aktif, silakan Anda pilih Add or remove user accounts. Fitur ini selain bisa membuat User Account baru, kita juga bisa menghapus User Account.

Perlu diperhatikan sebelum Anda membuat User Account baru, silakan Anda lihat apakah User Account mode Administrator pada komputer Anda terdiri lebih dari satu? Jika terdapat dua atau lebih User Account mode Administrator maka disini Anda tidak perlu membuat User Account baru lagi. Anda cukup menggantikan posisi User Account yang aktif.

Saya asumsikan komputer Anda tidak mempunyai User Account mode Administrator lagi selain User Account yang sedang digunakan sekarang. Maka tugas Anda sekarang adalah membuat lagi User Account. Caranya silakan Anda klik pilih Create a new account.

Silakan Anda ketik nama User Account baru yang akan dibuat pada kolom New account name kemudian Anda pilih mode Administrator. Terakhir jika kedua langkah tadi sudah selesai, klik pilih Create Account.

Sekarang restart (hidupkan ulang) komputer dan ketika komputer Start Up silakan Anda pilih User Account baru yang tadi telah Anda buat. Jika komputer sudah selesai melakukan Start Up-nya silakan Anda coba untuk melakukan set default printer apakah kasus error 0x00000709 masih ada atau tidak? Seharusnya kasus ini sudah selesai karena sekarang komputer Anda sudah menggunakan User Account baru. Jika sukses, silakan Anda hapus User Account awal yang error dari Control Panel seperti yang sudah saya terangkan diawal.

28. printer canon Mp250 saya error di print nya ada tulisan E08 tapi nama error nya ngak tau cara memperbaiki nya gimana??


Jika printer MP258 yang error 5B00 atau P07 dalam keadaan hidup, matikan printer MP258 tersebut dengan menekan tombol powernya. Tunggu sampai printer MP258 mati.Tekan dan tahan tombol Stop / Reset,  kemudian tekan dan tahan tombol Power. Jadi kedua tombol masih dalam keadaan tertekan.Lepas tombol Stop / Reset, kemudian tekan tombol Stop / Reset 2 x (jangan lepas tombol powernya).Kemudian lepaskan kedua tombol secara bersamaan.Printer MP258 yang error 5B00 atau P07 akan melakukan inisialisasi beberapa saat seperti ketika printer dicoba pertama pada saat beli. Tunggu sampai lcd panel pada printer MP258 yang error 5B00 atau P07 tadi menunjukkan angka 0 (nol).Printer MP258 akan terdeteksi sebagai device baru di komputer anda, walaupun tadinya sudah terpasang driver printer tersebut. Abaikan saja kondisi ini.Kondisi MP258 menampilkan angka 0 (nol) pada LCD panelnya menunjukkan bahwa MP258 dalam keadaan Service Mode dan siap untuk direset menggunakan software.Download program resetter MP258 untuk error 5B00 atau P07Siapkan 2 kertas di MP258, karena nanti akan digunakan untuk proses reset.Buka program resetter MP258 untuk error 5B00 atau P07Klik “MAIN”, tunggu beberapa saat maka MP258 akan print dengan tulisan ” D=000.0“Klik ”EEPROM Clear“, kemudian klik ”EEPROM“, MP258 akan print 1 lembar lagi. Salah satu barisnya tulisannya  : TPAGE(TTL=00000 COPY=00000)Kemudian matikan MP258  dengan menekan tombol Power.Selesai.Jawaban terbaik ya :)

38. cara pengujian printer pada windows 8


Step1: Hidupkan komputer Windows 8 Anda . Tunggu sampai mengisi penuh. Hover mouse Anda di sekitar dan klik bar Charm > klik Search. Ketik ' kompatibilitas ' di bidang Search dan tekan Enter . Klik Settings di kolom sebelah sisi kanan > klik Run program dibuat untuk versi Windows sebelumnya .

Step2 : Ikuti sisa di layar petunjuknya dan menyelesaikan instalasi driver untuk printer Anda . Bila diminta , hubungkan printer ke komputer menggunakan kabel USB atau menggunakan koneksi nirkabel . Ini biasanya harus bekerja untuk printer sudah diinstal pada komputer. ( Lihat Juga Windows 8 fitur di sini )

Step3 : Jika belum diinstal atau ditambahkan printer, klik Desktop di ekstrim pojok kiri bawah . Ini akan memuat layar Desktop akrab . Carilah Computer , klik kanan dan pilih Manage . Ini akan membuka jendela Manajemen Komputer .

Step4 : Klik Device Manager di panel sebelah kiri . Klik pada nama dari PC di panel sebelah kanan di bagian atas dan kemudian klik Add hardware warisan . Pada kotak dialog Add Hardware , pilih tombol di sebelah Instal hardware yang secara manual memilih dari daftar (Advanced ) dan tekan tombol Next.

Step5 : Dalam daftar perangkat keras , klik Printer , lalu tekan tombol Next . Pilih Gunakan tombol pelabuhan yang ada dan tekan Next. Jika memiliki disk , klik Have Disk dan tekan tombol Next pada jendela Install driver printer . harus memasukkan disk ketika diminta . Jika tidak memiliki disk , pilih printer dari daftar dan tekan tombol Next. ( Lihat Juga Perbedaan antara Windows 8 dan Windows 7 )

Step6 : Ketika diminta untuk berbagi printer, pilih Jangan berbagi printer ini . Tekan Next atau tombol OK . Klik tombol Print halaman pengujian untuk mencetak halaman uji . Klik Finish untuk keluar dari wizard Add Printer. Dalam hal pengujian gagal , kembali ke pengaturan printer dan memperbaikinya . Setelah selesai, keluar jendela . Hapus disk jika itu ada di drive komputer dan reboot komputer. Printer sekarang siap untuk digunakan .

Step7 : Dapat mengatur printer nirkabel serta mengikuti prosedur yang disebutkan di atas sama . Jika pernah menghadapi setup printer atau masalah yang berbeda , hubungi dukungan Windows 8.
