18. rentang kuartil dari data : 4,4,5,6,7,8,8,9,10,11 adalah


Rentang kuartil dari data : 4, 4, 5, 6, 7, 8, 8, 9, 10, 11 adalah ....

Pembahasan :

Rentang Kuartil = Q₃ - Q₁

Untuk mencari kuartil data tunggal :

1) kita urutkan data tersebut dari terkecil ke terbesar,

2) lalu tentukan Q₂ terlebih dahulu (titik tengah)
4...4...5...6...7...8...8...9...10...11
-------------------|------------------------
....................... Q₂ .........................

3) Q₁ adalah titik tengah data sebelah kiri, Q₃ titik tengah sebelah kiri,

4...4...5...6...7...8...8...9...10...11
---------|---------|----------|-------------
.......... Q₁ ....... Q₂ ....... Q₃ ..................

Q₁ = 5
Q₂ = antara 7 dan 8 = (7 + 8)/2 = 7,5
Q₃ = 9

Rentang Kuartil
= Q₃ - Q₁
= 9 - 5
= 4

Diketahui:
Data : 4, 4, 5, 6, 7, 8, 8, 9, 10, 11

Ditanya :
Rentang Kuartil =?

Pembahasan :

Rentang kuartil atau jangkauan interkuartil adalah selisih antara kuartil atas dan kuartil bawah, sehingga untuk mencari rentang kuartil pertama-tama kita harus mencari nilai dari quartil atas ( [tex] Q_{3} [/tex] ) dan quartil bawah ( [tex] Q_{1} [/tex] )

Apabila deretan data belum terurut darri kecil ke besar maka terlebih dahulu harus kita urutkan dari kecil ke besar, namun pada soal deretan data sudah terurut dari kecil ke besar:

4, 4, 5, 6, 7, 8, 8, 9, 10, 11

dari data diatas dapat dilihat bahwa terdapat 10 data , sehingga unutk mencari kuartil atas ( [tex] Q_{3} [/tex] ) dapat menggunakan cara seperti berikut :

[tex] Q_{3} [/tex] = [tex] \frac{3}{4} [/tex] x 10 
[tex] Q_{3} [/tex] = 7,5

Karena hasil dari kuartil atas ( [tex] Q_{3} [/tex] ) adalah 7,5 ; maka perlu mencari rata-rata dari data ke 7 dan 8 

[tex] Q_{3} [/tex] = ( data ke-7 + data ke-8) ÷ 2
[tex] Q_{3} [/tex] = [tex] \frac{8 + 9}{2} [/tex]
[tex] Q_{3} [/tex] = 8,5

Sehingga kita dapatkan nilai dari kuartil atas ( [tex] Q_{3} [/tex] ) adalah 8,5 .
Sekarang, kita cari nilai dari kuartil bawah ( [tex] Q_{1} [/tex] )

[tex] Q_{1} [/tex] = [tex] \frac{1}{4} [/tex] x 10
[tex] Q_{1} [/tex] = 2,5

karena hasil dari kuatil bawah ( [tex] Q_{1} [/tex] ) adalah 2,5 maka kita perlu mencari nilai rata-rata dari data ke-2 dan ke-3

[tex] Q_{1} [/tex] = (nilai data ke-2 + nilai data ke-3) ÷ 2
[tex] Q_{1} [/tex] = [tex] \frac{4 + 5}{2} [/tex]
[tex] Q_{1} [/tex] = 4,5

Sehingga didapatkan nilai dari kuartil bawah ( [tex] Q_{1} [/tex] )adalah 4,5.

Sekarang kita bisa mencari jangkauan kuartil dengan cara sebagai berikut ini :

Jangkauan interkuartil = quartil atas ( [tex] Q_{3} [/tex] ) - quartil bawah ( [tex] Q_{1} [/tex] )
Jangkauan interkuartil = 8,5 - 4,5 
Jangkauan interkuartil = 4 

Sehingga didapatkan nilai rentang kuartil dari data diatas adalah 4

22. perhatikan data berikut 8,7,7,5,8,10,9,8,7,6,5,4,2,6,7 dari data di atas tentukan median kuartil atas, kuartil bawah, rentang antar kuartil dan simpangan kuartil ! mohon di bantu yahh ​


Penjelasan dengan langkah-langkah:

Pertama-tama, kita akan mengurutkan data secara berurutan dari yang terkecil hingga yang terbesar:

2, 4, 5, 5, 6, 6, 7, 7, 7, 7, 8, 8, 8, 9, 10

Maka nilai median adalah rata-rata dari dua bilangan di tengah, yaitu (6+7)/2 = 6.5

Untuk mencari kuartil bawah dan kuartil atas, kita perlu membagi data menjadi dua bagian yang sama besar dari median. Karena data terdiri dari 15 bilangan, kita akan membagi data menjadi dua bagian yang sama besar dari 7 bilangan.

Kuartil bawah:

Bagian pertama data adalah 2, 4, 5, 5, 6, 6, dan 7.

Median dari bagian pertama adalah (5+6)/2 = 5.5.

Oleh karena itu, kuartil bawah adalah 5.5.

Kuartil atas:

Bagian kedua data adalah 7, 7, 7, 8, 8, 8, 9, dan 10.

Median dari bagian kedua adalah (8+8)/2 = 8.

Oleh karena itu, kuartil atas adalah 8.

Rentang antar kuartil dapat ditemukan dengan mengurangi kuartil bawah dari kuartil atas, yaitu:

Rentang antar kuartil = kuartil atas - kuartil bawah = 8 - 5.5 = 2.5

Simpangan kuartil dapat ditemukan dengan mengalikan rentang antar kuartil dengan 0.5, yaitu:

Simpangan kuartil = (Rentang antar kuartil) x 0.5 = 2.5 x 0.5 = 1.25

Jadi, nilai median adalah 6.5, kuartil bawah adalah 5.5, kuartil atas adalah 8, rentang antar kuartil adalah 2.5, dan simpangan kuartil adalah 1.25

36. iketahui suatu data ulangan matematika sebagai berikut 57, 84, 91, 72, 68, 87, 90, dan 83. Hitunglah: a. rata-rata b. modus dan median c. kuartil pertama dan kuartil ketiga d. rentang antar kuartil dan rentang semi kuartil e. simpangan rata-rata f. simpangan baku​


Jawaban:

Untuk menghitung statistik yang diminta, mari kita analisis satu per satu:

a. Rata-rata:

Rata-rata dapat dihitung dengan menjumlahkan semua data dan membaginya dengan jumlah data.

Rata-rata = (57 + 84 + 91 + 72 + 68 + 87 + 90 + 83) / 8 = 712 / 8 = 89

b. Modus:

Modus adalah nilai yang muncul paling sering dalam himpunan data. Jika tidak ada nilai yang muncul lebih dari satu kali, maka tidak ada modus.

Modus = Tidak ada (semua nilai muncul hanya sekali)

c. Median:

Median adalah nilai tengah ketika data diurutkan secara terurut. Jika jumlah data ganjil, median adalah nilai di tengah. Jika jumlah data genap, median adalah rata-rata dari dua nilai di tengah.

Data terurut: 57, 68, 72, 83, 84, 87, 90, 91

Median = (83 + 84) / 2 = 167 / 2 = 83.5

d. Kuartil Pertama dan Kuartil Ketiga:

Kuartil Pertama adalah nilai yang membagi 25% data terkecil, sedangkan Kuartil Ketiga adalah nilai yang membagi 75% data terkecil.

Data terurut: 57, 68, 72, 83, 84, 87, 90, 91

Kuartil Pertama = 68

Kuartil Ketiga = 87

e. Rentang Antar Kuartil dan Rentang Semi Kuartil:

Rentang Antar Kuartil adalah selisih antara Kuartil Ketiga dan Kuartil Pertama, sedangkan Rentang Semi Kuartil adalah setengah dari Rentang Antar Kuartil.

Rentang Antar Kuartil = Kuartil Ketiga - Kuartil Pertama = 87 - 68 = 19

Rentang Semi Kuartil = Rentang Antar Kuartil / 2 = 19 / 2 = 9.5

f. Simpangan Rata-rata dan Simpangan Baku:

Simpangan Rata-rata adalah selisih antara setiap data dengan rata-rata, sedangkan Simpangan Baku adalah akar kuadrat dari rata-rata simpangan kuadrat.

Simpangan Rata-rata = |57 - 89| + |84 - 89| + |91 - 89| + |72 - 89| + |68 - 89| + |87 - 89| + |90 - 89| + |83 - 89| / 8 = 216 / 8 = 27

Simpangan Baku = √((|57 - 89|^2 + |84 - 89|^2 + |91 - 89|^2 + |72 - 89|^2 + |68 - 89|^2 + |87 - 89|^2 + |90 - 89|^2 + |83 - 89|^2) / 8) ≈ 9.5

Jadi, hasil perhitungan statistik untuk data ulangan matematika tersebut adalah:

a. Rata-rata = 89

b. Modus = Tidak ada

c. Median = 83.5

d. Kuartil Pertama = 68, Kuartil Ketiga = 87

e. Rentang Antar Kuartil = 19, Rentang Semi Kuartil = 9.5

f. Simpangan Rata-rata = 27, Simpangan Baku ≈ 9.5
