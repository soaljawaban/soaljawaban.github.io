Detail :

Diketahui x + y = 1

Misalnya kita anggap

36. tolong bantuin yukk​


persamaan kuadrat

x² + 2x - 2 = 0

x1 + x2 = -b/a = -2/1 = -2

x1 . x2 = c/a = -2/1 = -2

akar baru : 1/x1 dan 1/x2

m = 1/x1 + 1/x2

m = (x1 + x2)/(x1x2)

m = 1

n = 1/x1 . 1/x2

n = -1/2

persamaan kuadrat dg akar baru :

x² - mx + n = 0

x² - x - 1/2 = 0

2x² - 2x - 1 = 0
