Oli Mesin Mobil = X

Oli Mesin Motor = Y

Diketahui : 2x + y = 330.000

                  3x + 4y = 570.000

Ditanya : x & y ...?

Dijawab :

kita akan menggunakan metode eliminasi

3x + 4y = 570.000 |x2| 6x + 8y = 1.140.000

2x +  y  = 330.000 |x3| 6x + 3y = 990.000

                                     --------------------------  -

                                              5y = 150.000

                                                y = 30.000

setelah mendapatakan salah nominal dari salah satu variabel maka tinggal gunakan metode substitusi

2x + y = 330.000

2x + 30.000 = 330.000

2x = 300.000

x = 150.000

Jadi...

Nilai X = 150.000

Nilai Y = 30.000
