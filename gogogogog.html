aCuS + bHNO3 => cCu(NO3)2 + dS + eNO + fH2O, disini d e f saya tambahkan untuk mempermudah pengerjaan

Cu : a = c

S : a = d

H : b = 2f

N : b = 2c + e

O : 3b = 6c + e + f

1 = 2f, maka f = 1/2

1 = 2c + e ............ (1)
