4. Apakah yang dimaksud dengan metode NaÃ¯ve?

7. Tuliskan jenis-jenis file proxmox ve?


Jawaban: Jenis Jenis file proxmox ve

Kita akan membahas beberapa hal menyangkut konfigurasi cluster proxmox dan juga struktur directory

authkey.pub

Dari namanya kita dapat memahami bahwa authkey.pub ini merupakan public key.

 

.clusterlog

Dari namanya kita dapat memahami juga bahwa .clusterlog ini berisi 50 log terakhir yang ada dalam cluster ini

datacenter.cfg

File datacenter.cfg ini merupakan konfigurasi untuk datacenter ini. Pada contoh kali hanya berisi

keyboard: en-us

local

Pada gambar di atas terlihat bahwa local merupakan alias dari folder nodes/cormicpve. Tampilan di atas merupakan tampilan pada host cormicpve, sehingga kita dapat memahami bahwa local ini merupakan alias bagian cluster yang berada pada host ini. Jadi local ini merupakan bagian cluster yang berada di local.

 

lxc

Folder lxc ini merupakan alias dari nodes/cormicpve/lxc. Jadi lxc ini merupakan bagian cluster lxc (Linux Container) yang berada dalam local server ini.

.members

File .members ini berisi anggota dari cluster, pada contoh kali ini isinya:

{

“nodename”: “cormicpve”,

“version”: 0

}

nodes

Folder nodes ini merupakan folder untuk masuk ke node-node anggota cluster.

 

priv

Directory priv ini berkaitan dengan privilleges.

 

qemu-server

Folder qemu-server ini merupakan alias dari nodes/cormicpve/qemu-server. Jadi qemu-server ini merupakan bagian cluster gemu-server (KVM) yang berada dalam local server ini.

storage.cfg

File storage.cfg ini merupakan file untuk konfigurasi storage. Pada contoh kali ini isinya:

dir: local

       path /var/lib/vz

       content iso,vztmpl,backup

lvmthin: local-lvm

       thinpool data

       vgname pve

       content rootdir,images

Pada contoh kali ini kita akan melihat bahwa storage berada pada folder /var/lib/vz.

user.cfg

File user.cfg ini berisi user.

.version

Pada file .version ini akan berisi versi-versi resources untuk mendeteksi perubahan.

 

.vmlist

Pada file .vmlist ini berisi daftar virtual machine yang berada pada cluster. Pada contoh kali ini isinya:

{

“version”: 1,

“ids”: {

“100”: { “node”: “cormicpve”, “type”: “qemu”, “version”: 2 },

“101”: { “node”: “cormicpve”, “type”: “qemu”, “version”: 3 },

“102”: { “node”: “cormicpve”, “type”: “qemu”, “version”: 1 }}

}

36. ini ( 5+5 ) ve y ( 5-5 ) u =​


Jawab:Asumsi:

"ve" mungkin bermaksud "x" (tanda kali) atau "dan" (operator logika "AND")

"u" mungkin bermaksud "=" (operator "sama dengan")

Dengan asumsi tersebut, maka pernyataan "(5+5) ve y(5-5) u" dapat diinterpretasikan sebagai berikut:

Lakukan operasi matematika pada angka 5+5 = 10 dan 5-5 = 0.

Kemudian, lakukan operasi "x" atau "dan" (tergantung pada maksud dari "ve") pada hasil 10 dan 0.

Terakhir, hasil tersebut dibandingkan dengan suatu nilai atau variabel (tergantung pada maksud dari "u").

Sebagai contoh, jika maksud dari "ve" adalah operator "x", dan maksud dari "u" adalah operator "sama dengan", maka pernyataan tersebut dapat ditulis ulang menjadi:

(5+5) x (5-5) = u

10 x 0 = u

0 = u

Jadi, jika asumsi tersebut benar, maka hasil dari pernyataan tersebut adalah "u = 0".

39. apakah kata kerja dari 've
