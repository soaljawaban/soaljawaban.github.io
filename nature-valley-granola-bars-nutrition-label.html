2. Angie has $30. If she buys six granola bars and two boxes of cereal, she will receive $0.40 in change. If she buys four granola bars and three boxes of cereal, she will be short of $0.40. Find the price of each item.


Jawab:

Ah, finally an English Math.

Suppose granola bars is x, and a box of cereal is y. Then, write the problem in the following equation

6x+2y = $30-$0.40 -> she receive change, so the total price is Angie's money minus the change.

4x+3y=$30+$0.40 -> she will be short, so the total price is Angie's money plus the short.

Then, we can substitute, eliminate, or combine both. In this problem, Sa-chan uses elimination.

6x+2y=$29.60      |×3| = 18x+6y=$88.80

4x+3y=$30.40       |×2| = 8x+6y=$60.80

------------------------------------------------------------ -

                                              10x=$28.00

                                                  x=$2.80

Then, substitute x to either the first or second equation. Sa-chan uses the first.

6x+2y=$29.60

6($2.80)+2y=$29.60

               2y=$12.80

                 y=$6.40

∴ Hence, the price for one granola bars is $2.80 and the price for a box of cereal is $6.40

Jawaban:

A. 47

Penjelasan dengan langkah-langkah:

Let's solve the problem step by step:

Let's assume Janet bought "x" bars of chocolate.

1. If she shares the bars equally with 7 friends, she will have 7 bars left over.

This can be expressed as the following equation:

x mod 7 = 7

2. If she shares the bars equally with 6 friends, she will have 5 bars left over.

This can be expressed as the following equation:

x mod 6 = 5

Now, let's find the value of "x" that satisfies both equations.

First, let's try to find the possible range of values for "x":

We know that she bought between 35 and 50 bars, so 35 ≤ x ≤ 50.

Now, we will check each value within this range to see which value satisfies both equations:

For x = 35:

35 mod 7 = 0 (not 7, so it doesn't satisfy the first equation)

35 mod 6 = 5 (it satisfies the second equation)

For x = 36:

36 mod 7 = 1 (not 7, so it doesn't satisfy the first equation)

36 mod 6 = 0 (not 5, so it doesn't satisfy the second equation)

For x = 37:

37 mod 7 = 2 (not 7, so it doesn't satisfy the first equation)

37 mod 6 = 1 (not 5, so it doesn't satisfy the second equation)

... and so on.

Let's continue checking until we find a value that satisfies both equations:

For x = 41:

41 mod 7 = 6 (not 7, so it doesn't satisfy the first equation)

41 mod 6 = 5 (it satisfies the second equation)

For x = 42:

42 mod 7 = 0 (not 7, so it doesn't satisfy the first equation)

For x = 45:

45 mod 7 = 3 (not 7, so it doesn't satisfy the first equation)

45 mod 6 = 3 (not 5, so it doesn't satisfy the second equation)

For x = 46:

25. kata "nutrition" di "nutrition facts" lebih mirip ke "nourishment" atau "substance" utamakan jawaban dengan alasan yaa ^^​


Jawaban  :Kata "nutrition" dalam "nutrition facts" lebih mirip dengan "nourishment" daripada "substance."

Penjelasan:  Alasan utamanya adalah bahwa "nutrition" dalam konteks ini merujuk kepada nilai gizi atau zat-zat yang ditemukan dalam makanan yang memberikan nutrisi atau makanan bagi tubuh manusia. "Nourishment" berarti memberi makan atau memberikan nutrisi, yang sesuai dengan konteks "nutrition facts" yang menyediakan informasi tentang nilai gizi makanan. Sementara "substance" lebih umum merujuk pada materi atau bahan apapun tanpa spesifik merujuk kepada nilai gizi atau nutrisi.
