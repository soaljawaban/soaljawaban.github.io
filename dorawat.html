Jawaban:

c. 167 derajat F

Penjelasan:

Diketahui :

tK = 348K

Ditanyakan : tF ?

Dijawab :

tF = 9/5 * (k - 273) + 32

= 9/5 (348 - 273) + 32

= 9/5 (75) + 32

= 135 + 32 = 167 F

Penjelasan:

[tex]R = \frac{4}{5} C \\ C = \frac{5}{4} R \\ C = \frac{5}{4} 50 \\ C = \frac{250}{4} \\ C = 62.5[/tex]

[tex]F = \frac{9}{5} C + 32 \\ F = \frac{9}{5} 62.5 + 32 \\ F = 9 \times 12.5 + 32 \\ F = 112.5 + 32 \\ F = 144.5[/tex]

[tex]K = C + 273 \\ K = 62.5 + 273 \\ K = 335.5[/tex]
