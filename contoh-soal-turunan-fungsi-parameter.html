function rangeWithStep(startNum, stopNum, step) {

   if (typeof stopNum == 'undefined') {

       // one param defined

       stopNum = startNum;

       startNum = 0;

   }

   if (typeof step == 'undefined') {

       step = 1;

   }

   if ((step > 0 && startNum >= stopNum) || (step < 0 && startNum <= stopNum)) {

       return [];

   }

   var result = [];

   for (var i = startNum; step > 0 ? i < stopNum : i > stopNum; i += step) {

       result.push(i);

   }

   return result;

};

17. • Apakah fungsi dari parameter ALT pada tag Image? • Apakah fungsi dari parameter WIDTH dan HEIGHT pada tag Image? • Kapan parameter tersebut digunakan? Apakah perlu digunakan keduanya? • Bagaimana cara membuat hyperlink dari suatu gambar yang rujukannya merupakan URL (misalnya http://www.google.co.id atau ftp://sun.uio.org)?


Sederhananya fungsi dari atribut "alt" pada elemen atau tag image <img> adalah sebagai teks alternatif; artibut "width" dah "height" berfungsi untuk mendeklarasikan ukuran dari elemen image <img> (gambar); kedua atribut (width dan height) tersebut harus selalu digunakan secara bersamaan pada sebuah gambar; Cara membuat hyperlink atau tautan dari sebuah gambar bisa dilakukan dengan mengapit tag <img> dengan tag <a> dimana gambar menjadi jangkarnya (anchor)

Pembahasan:

ALT

Atribut alt pada image adalah singkatan dari alternative text yang pada dasarnya digunakan untuk membuat alternatif teks dari sebuah gambar yang berasal dari elemen <img>. Bagi pengguna pada umumnya fungsi alt ini tidak terlalu signifikan karena pada dasarnya ditujukan untuk:

peramban atau browser agar menampilkan teks ketika halaman dimana gambar tersebut berada gagal loading dimana gambar tidak bisa ditampilkan, misalnya ketika ada galat (error) atau koneksi internet yang tidak mencukupi-keperluan komputasi agar peramban dan browser bisa membaca teks pada alt untuk dirubah menjadi suara. Biasanya fasilitas ini digunakan oleh pengguna disabilitas yang memiliki gangguan pengelihatan (tuna netra)  dibaca oleh bot mesin penelusur agar mesin pencari bisa menyimpan gambar tersebut sesuai dengan konteksnya dimana nantinya bisa ditampilkan pada halaman pencarian yang tepat sesuai kata kunci para penggunanya

Width dan Height

Dalam generasi HTML yang ada saat ini width dan height hanya bisa digunakan pada elemen <img>. Fungsinya sendiri adalag agar mempercepat peramban untuk menampilkan gambar dalam ukuran aslinya. Agar tidak ada ketimpangan ukuran pada saat proses loading width dan height harus diaplikasikan secara bersamaan.

Tautan Image

Image atau gambar bisa dijadikan tautan, caranya hampir sama ketika kita membuat tautan teks hanya saja pada gambar yang dijadikan jangkarnya adalah image tersebut. Berikut contohnya:

<a href="//alamatwebtujuan.com">

<img alt="isi dengan alternatif teks gambar" src="contohsumbergambar.com/google.jpg">

</a>

19. Buat lah sebuah 3 fungsi overloading yang memiliki 1 parameter - Jika parameternya int maka outputnya “Tipe data int, hasilnya [parameter * 100]” - Jika parameternya double maka outputnya “Tipe data double, hasilnya [parameter * 1000]” - Jika parameternya string maka outputnya “Tipe data string, nilainya Hello [parameter]” contoh pada fungsi main : int main() { overloadingFunc(10; overloadingFunc(10.0); overloadingFunc("World); return 0; } outputnya : Tipe data int, hasilnya 10000 Tipe data float, hasilnya 1000 Tipe data string, nilainya Hello World


Jawaban dan Penjelasan:

Overloaded Functions

Saya buat 2 versi. Yang pertama, argumen bertipe string menggunakan tipe "string" dari namespace std. Karena ada baris "use namespace std;", maka nama tipenya "string" saja, tidak perlu "std::string".

Versi kedua, argumen bertipe string menggunakan "char*". Pada saat passing argumen ke fungsi, harus menggunakan casting berupa "(char*)", agar tidak ada warning pada saat kompilasi.

Versi 1: tipe data string menggunakan string dari namespace std (std::string)

#include <iostream>

using namespace std;

void overloadingFunc(int i) {

   cout << "Tipe data int, hasilnya " << (100*i) << endl;

}

void overloadingFunc(double d) {

   cout << "Tipe data double, hasilnya " << (1000*d) << endl;

}

void overloadingFunc(string s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc("World");

   return 0;

}

Versi 2: tipe data string menggunakan char*

Bagian di atas ini persis sama seperti di atas.

void overloadingFunc(char* s) {

   cout << "Tipe data string, nilainya Hello " << s << endl;

}

int main() {

   overloadingFunc(10);

   overloadingFunc(10.0);

   overloadingFunc((char*)"World");

   return 0;

}

1. Fungsi/function adalah grup/blok program yang dibuat untuk melakukan tugas tertentu secara berulang kali sehingga kita tidak perlu menuliskan ulang kembali kode yang sama untuk baris kode selanjutnya.

3. fungsi tanpa kembalian(fungsi yang tidak mengembalikan value/nilai apa-apa), dan fungsi kembalian(fungsi yang mengemabilkan value/nilai setelah fungsi selesai dijalankan).

4.  - Pendeklarasian Fungsi:

void sapa(nama){

   cout << "Hello, " << nama << endl;

}

void = sebuah tipe dari fungsi, lebih tepatnya fungsi tanpa kembalian.

sapa = sebagai nama fungsi yang telah kita buat dan akan digunakan untuk memanggil fungsi itu nantinya di main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik fungsi sapa.

nama = sebagai paramater.

cout = berarti console out, atau sederhannya menampilkan hasil dari program ke console/terminal.

<< = tanda yang berati akan mengeluarkan/menampilkan ke terminal

"Hello" = teks/string biasa.

nama = akan menampilkan nama(Parameter) ke console/terminal.

endl = bisa diartikan sebagia "\n" atau disebut juga dengan new line/baris baru.

; = titik dua/semicolom berarti sebagai penanda jika kode pada baris ini akan berhenti.

- Pendefinisian(pemanggilan?) :

#include <iostream>

using namespace std;

int main(){

string nama;

cout << "masukan nama kamu : " ; cin >> nama;

sapa(nama);

}

#include <iostream> = prosesor pengarah yang mengatakan kepada kompiler untuk meletakan kode dari header file iostream.

using namespace std; =  membuat kita tidak perlu menggunakan std:: lagi di dalam program.

int main() = sebagi main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik main program.

str nama; = sebagai variabel yang menyimpan data/value yang dimasukan oleh user.

cout << =  berarti console out, atua sederhannya menampilkan hasil dari programnya ke console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

cin >> = console input, yang berarti user akan memasukan/input data ke variabel nama;.

sapa(nama); = memnaggil fungsi. kita akan mengoper data yang telah di-input-kan oleh user di variabel nama; ke dalam fungsi untuk natinya ditampilkan ke dalam console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

Pada MYSQLI_ASSOC = Item array akan menggunakan nama kolom sebagai index key

Pada MYSQLI_BOTH = Item array akan diduplikasikan, yang satu menggunakan indeks key bertipe numerik, dan yang lainnya menggunakan nama kolom sebagai index key


Parameter yang digunakan dalam fungsi mysqli_fetch_rows adalah sebagai berikut

mysqli_fetch_row(result);


Result wajib diisi. Menandakan kumpulan hasil identifier yang dikembalikan oleh mysqli_query(), mysqli_store_result() atau mysqli_use_result().
