40. 3/4 of Jevan’s height is equal to 2/5 of Nicslie’s height. The difference in their height is 77 cm . What is Nicslie’s height in cm?


Jawaban:

jevan height = x

Nicslie height = y

3/4x = 2/5y

x + 77 = y

use substitution method

¾x = ⅖ (x + 77)

¾x = ⅖x + ¹⁵⁴/5

¾x - ⅖x = ¹⁵⁴/5

¹⁵/20x - ⁸/20x = ¹⁵⁴/5

⁷/20x = ¹⁵⁴/5

x = ¹⁵⁴/5 ÷ ⁷/20

x = ¹⁵⁴/5 x 20/7

x = ⁴⁴⁰/5

x = 88cm

Jevan's height is 88cm

¾x = ⅖y

¾ • 88 = ⅖y

66 = ⅖y

y = 66 ÷ ⅖
