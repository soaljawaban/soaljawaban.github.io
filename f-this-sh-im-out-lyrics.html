23. Tentukan RE(F(z)) dan Im (f(z)) dari fungsi f(z)=z²-5z​


Jawaban bsrta penjelasan:

Untuk mencari bagian RE, kita dapat menggunakan rumus sebagai berikut:

RE(f(z)) = (f(z) + f*(z)) / 2

Dalam rumus di atas, f*(z) merupakan konjugat dari f(z). Oleh karena itu, untuk fungsi f(z) = z² - 5z, maka konjugatnya adalah f*(z) = z² - 5z.

Maka:

RE(f(z)) = (z² - 5z + z² - 5z) / 2

= (2z² - 10z) / 2

= z² - 5z

Sehingga, RE(f(z)) = z² - 5z.

Sedangkan untuk bagian Im, kita dapat menggunakan rumus berikut:

Im(f(z)) = (f(z) - f*(z)) / (2i)

Maka:

Im(f(z)) = (z² - 5z - z² + 5z) / (2i)

= 0 / (2i)

= 0

Sehingga, Im(f(z)) = 0.
