8. Ubahlah menjadi negative My mother cooks meat ball in the kitchen


My mother doesn't cook meat ball in the kitchen.

Rumusnya :

S + do/does not + bare infinitive

S + am/is/are not + subject complement (noun/adjective/…)

PART ; SENTENCE STRUCTURE - HOW TO FORM POSITIVE, NEGATIVE , INTERROGATIVE SENTENCE IN THE SIMPLE PRESENT TENSE

---------------------------

Jawaban:

(+) My mother cooks meat ball in the kitchen.

(-) My mother doesn't cook meat ball in the kitchen

(? ) Does my mother cook meat ball in the kitchen?

Penjelasan:

SIMPLE PRESENT TENSEmerupakan tenses yang digunakan dalam menjelaskan kejadian nyata di masa lampau ,  saat ini dan akan tetap selalu nyata mendatang.Tenses ini juga menjabarkan fakta kebenaran umum, kebenaran yang tak terbatas dan aktivitas yang terus- menerus berulang  ( kebiasaan/habit )

Penggunaan keterangan frekuensi ( Adverb of frequency) untuk menyatakan habit/kebiasaan :

Always, usually, often, everyday, seldom, rarely, daily, never, frequently, occasionally , ever, sometimes, regularly.

Berdasarkan sruktur kalimatnya, pola simple present tense dibagi dalam 2 jenis :

1. Pola Simple present tense ( verbal sentence )

Verbal sentence merupakan kalimat yang predikatnya berupa kata kerja aksi.

PATTERN :

(+ ) SUBJECT + VERB ( s/es ) + O/COMPLEMENT

(- )  SUBJECT + DO/DOES + NOT + VERB 1 + O/COMPLEMENT  

(? ) DO/DOES+SUBJECT + VERB 1 + O/COMPLEMENT?

Example :

(+ ) George eats fried chicken everyday.   ( habit/ kebiasaan )

( - ) George doesn't eat fried chicken everyday

( ? ) Does George eat fried chicken everyday?

2. Pola simple present tense ( nominal sentence )

Nominal sentence merupakan kalimat yang predikatnya bukan berupa kata kerja aksi. Pada umumnya berupa kata benda ( noun), kata sifat ( adjective ), dan keterangan ( adverb )

Pattern

( + ) SUBJECT + TOBE  (iS/AM/ARE + NOUN/ADJECTIVE/ADVERB

( - ) SUBJECT +TOBE (IS/AM/ARE )+NOT + NOUN/ADJECTIVE/ADVERB

( ? ) TOBE(IS/AM/ARE)+SUBJECT+NOUN/ADJECTIVE/ADVERB?

Example :

( + ) He is  a doctor       ( fakta/menjabarkan profesi saat ini )

(  - ) He is not a doctor

( ? )  Is he a doctor

PEMBAHASAN Active Voice kalimat aktif,Subjek melakukan aksiPassive Voice Kalimat Pasif,Object dikenai aksi oleh subject Pola PassiveBe + V3

Maka Berdasarkan Rumus:

Present Tense Active: (+) S + V1 (-s/es) + OPassive: (+) O + Be (is/am/are) + V3 + By S

1. My mother takes my ball from the bag

Ibuku mengambil bolaku dari tas

➡ My ball is taken by mother from the bag

Bolaku di ambil oleh ibuku dari tas

2. I call them in the class

Saya memanggil mereka dalam kelas

➡ They are called by me in the class

Mereka dipanggil olehku dalam kelas

Penjelasan:

Pada Simple Present Tense, kata kerja yang digunakan ialah kata kerja bentuk pertama atau dasar (V1).

Ketika subjek yang melakukan adalah orang ketiga tunggal, maka biasanya kata kerja diimbuhi +s/es.

Rumus penulisan Simple Present Tense adalah sebagai berikut:

Kalimat positif:

S + infinitive (V1) (+s/es) / to be (is, am, are) + O.

Kalimat negatif:

S + do/does not + infinitive (V1) + O.

S + to be (is, am, are) + not + O.

Kalimat interogatif:

Do/does + S + infinitive (V1) + O?

to be (is, am, are) + S + O?
