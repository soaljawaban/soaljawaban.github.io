13. if twice the age of son is added to age of father, the sum is 56.but if wice the age of the father is added to the age of son, the sum is 82.fund the ages of father and son.....? tolong di jawab ya​


Jawab:

son age (x) =10

father age (y) = 36

Penjelasan dengan langkah-langkah:

Solve the following system:

y + 2 x = 56

2 y + x = 82

In the first equation, look to solve for y:

y + 2 x = 56

2 y + x = 82

Subtract 2 x from both sides:

y = -2 x + 56

2 y + x = 82

Substitute y = -2 x + 56 into the second equation:

y = -2 x + 56

2 (-2 x + 56) + x = 82

2 (-2 x + 56) + x = x + (-4 x + 112) = -3 x + 112:

y = -2 x + 56

(-3 x + 112) = 82

In the second equation, look to solve for x:

y = -2 x + 56

-3 x + 112 = 82

Subtract 112 from both sides:

y = -2 x + 56

-3 x = -30

Divide both sides by -3:

y = -2 x + 56

x = 10

Substitute x = 10 into the first equation:

y = 36

x = 10
