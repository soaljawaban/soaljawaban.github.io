11. apa fungsi windows live messenger​


Jawaban:

Windows Live Messenger adalah program perpesanan instan yang dikembangkan oleh Microsoft. Fungsinya adalah untuk memungkinkan pengguna berkomunikasi secara langsung melalui pesan teks, suara, dan video dengan orang lain yang menggunakan Windows Live Messenger atau layanan yang kompatibel.

Penjelasan:

Pengguna dapat mengirim pesan teks, berbagi file, membuat panggilan suara atau video, mengatur status online, dan berpartisipasi dalam obrolan grup. Meskipun Windows Live Messenger telah dihentikan oleh Microsoft pada tahun 2013 dan digantikan oleh Skype, tetapi pada saat itu, Windows Live Messenger adalah salah satu aplikasi perpesanan instan yang populer di dunia.

Penjelasan:

Windows Live Messenger, juga dikenal sebagai MSN Messenger, adalah program komunikasi instan yang dikembangkan oleh Microsoft. Fungsinya mencakup:

1. Pesan Teks dan Emotikon: Pengguna dapat mengirim pesan teks dan menggunakan emotikon untuk mengekspresikan emosi mereka dalampercakapan.

2. Panggilan Suara dan Video: Live Messenger memungkinkan pengguna untuk membuat panggilan suara dan video gratis ke pengguna lainnya melalui internet.

3. Transfer Berkas: Pengguna dapat mentransfer file, gambar, dan dokumen kepada teman-teman mereka dengan mudah.

4. Status dan Kehadiran: Pengguna dapat menetapkan status kehadiran mereka (online, offline, sibuk, dll.) dan melihat status kehadiran kontak mereka.

5. Obrolan Grup: Pengguna dapat membuat obrolan grup dengan beberapa teman untuk berbicara tentang topik tertentu.

6. Integrasi dengan Layanan Lain: Windows Live Messenger terintegrasi dengan berbagai layanan Microsoft, seperti Hotmail dan Xbox Live.

7. Peringatan Email: Pengguna dapat menerima peringatan ketika mereka menerima email baru di akun Hotmail mereka.
