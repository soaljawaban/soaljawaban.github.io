3e + p = 50  |×2| 6e + 2p = 100  ... (1)
1e + 2p = 25  ... (2)

eliminasi persamaan (1) dan (2)

maka
6e + 2p = 100
1e + 2p = 25
____________ -
5e = 75
e = 15 ⇒ p = 5

20. Ibu membeli 3ember dan 1panci dengan harga 50.000,ditoko yg sama Ani membeli 1ember dan 2panci dengan harga 65.000,berapakah harga untuk 1ember dan 1panci.. A.15.000 B.14.500 C.14000 D.13.500 E.13000


Sistem Persamaan Linier Dua Variabel (SPLDV)

Misal;

Ember = x

Panci = y

Diketahui

3x + y = 50.000...(1)

x + 2y = 65.000...(2)

Penyelesaian

Samakan nilai y, lalu eliminasikan

3x + y = 50.000 |x2|

x + 2y = 65.000 |x1 |

             ↓

6x + 2y = 100.000

x   + 2y = 65.000

-------------------------- -

5x = 35.000

x = 35.000/5

  = 7.000

Substitusikan ke persamaan 1

3x + y = 50.000

→ 3(7.000) + y = 50.000

→ 21.000 + y = 50.000

→ y = 50.000 - 21.000

     = 29.000

Harga 1 ember dan 1 panci = x + y

                                            = 7.000 + 29.000

                                           = 36.000

Panci apa ini, mahal banget?
