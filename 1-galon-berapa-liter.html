37. sebuah galon terisi 1/3 nya, jika ditambah 3 liter isi galon menjadi setengah nya. berapa liter kah jika galon terisi penuh?

Bab : Sistem Persamaan Linear Satu Variabel

- Volume Galon = x
- ⅓x + 3 = ½x

Ditanya :
- Nilai x = .....?

Penyelesaian :
- Nilai x :
⅓x + 3 = ½x
6(⅓x + 3 = ½x)
2x + 18 = 3x
2x - 3x + 18 = 0
- x + 18 = 0
- x = - 18
x = 18

Jadi, Volume Galon saat penuh adalah 18 liter
