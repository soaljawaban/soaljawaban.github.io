32. Hitunglah jumlah dari 12 + 17 + 23 + ... + 107 !Makasih banyak tolong pake cara deret aritmatika..


Diketahui,
a = 12
b = 17 - 12 = 5
Sn = 107
n = ?
Ditanya, jumlah n suku tersebut

Rumus Aritmatika
Sn   = a + (n-1)b
107 = 12 + (n-1)5
107 = 12 + 5n -5
107 = 7 + 5n
5n   = 107 - 7
n     = 100/5
n     = 20

Gunakan Rumus Deret
Dn = 1/2 x n (a + Sn)
Dn = 1/2 x 20 (12 + 107)
Dn = 10 (119)
Dn = 1.190

Jadi, jumlah keseluruhan dari 12 + 17 + 23 + ... + 107 adalah 1.190

Dengan demikian, modus dari data tersebut adalah 106, karena angka tersebut muncul paling sering, yaitu sebanyak 7 kali
