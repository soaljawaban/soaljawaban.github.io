26. Tentukqn nilai x pyang memenuhi |3x+1/2|=10


[tex]\textbf{Jawab}\\[/tex]
Nilai x yang memenuhi persamaan nilai mutlak tersebut adalah x = -7/2 atau x = 19/6.
Nilai absolut atau nilai mutlak atau modulus adalah nilai suatu bilangan riil tanpa tanda plus atau minus. Baik |a| ataupun |-a| sama-sama bernilai a. Sebagai contoh, nilai absolut dari 3 adalah 3, dan nilai absolut dari –3 juga 3.

[tex]\textbf{Pembahasan}\\[/tex]

| 3x + ½ | = 10
(| 3x + ½ |)² = 10²
9x² + 3x + ¼ = 100
9x² + 3x + - 399/4 = 0
36x² + 12x - 399 = 0
12x² + 4x - 133 = 0
(2x + 7) (6x - 19) = 0
x = -7/2 atau x = 19/6
