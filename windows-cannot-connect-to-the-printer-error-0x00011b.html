Windows Error Recovery dapat dihilangkan jika kamu mengetahui permasalahan yang terjadi pada komputermu baik hardware maupun software. ketika terjadi hal ini, mungkin komputermu mengingatkanmu untuk mengganti file atau memperbaiki hardware yang bersangkutan

Windows Error Recovery ada 2
a. dikarenakan masalah kecil
b. dikarenakan kerusakan file Windows/hardware

coba ganti PSU atau masalah pada kestabilan listrik

kalau kamu sudah pernah kena Blue Screen dan muncul Windows Error Recovery, coba startup repair (perbaiki hal yang salah dengan komputer)

tetapi kalau kamu belum pernah kena Blue Screen tetapi muncul Windows Error Recovery, kerusakan ada di hardware (layaknya hardisk). lakukan startup repair, cari kode erroronya di internet dan lakukan tindakan

Step 1. Open control panel

Step 2. Click on uninstall programs and features

24. cara pengujian printer pada windows 8


Step1: Hidupkan komputer Windows 8 Anda . Tunggu sampai mengisi penuh. Hover mouse Anda di sekitar dan klik bar Charm > klik Search. Ketik ' kompatibilitas ' di bidang Search dan tekan Enter . Klik Settings di kolom sebelah sisi kanan > klik Run program dibuat untuk versi Windows sebelumnya .

Step2 : Ikuti sisa di layar petunjuknya dan menyelesaikan instalasi driver untuk printer Anda . Bila diminta , hubungkan printer ke komputer menggunakan kabel USB atau menggunakan koneksi nirkabel . Ini biasanya harus bekerja untuk printer sudah diinstal pada komputer. ( Lihat Juga Windows 8 fitur di sini )

Step3 : Jika belum diinstal atau ditambahkan printer, klik Desktop di ekstrim pojok kiri bawah . Ini akan memuat layar Desktop akrab . Carilah Computer , klik kanan dan pilih Manage . Ini akan membuka jendela Manajemen Komputer .

Step4 : Klik Device Manager di panel sebelah kiri . Klik pada nama dari PC di panel sebelah kanan di bagian atas dan kemudian klik Add hardware warisan . Pada kotak dialog Add Hardware , pilih tombol di sebelah Instal hardware yang secara manual memilih dari daftar (Advanced ) dan tekan tombol Next.

Step5 : Dalam daftar perangkat keras , klik Printer , lalu tekan tombol Next . Pilih Gunakan tombol pelabuhan yang ada dan tekan Next. Jika memiliki disk , klik Have Disk dan tekan tombol Next pada jendela Install driver printer . harus memasukkan disk ketika diminta . Jika tidak memiliki disk , pilih printer dari daftar dan tekan tombol Next. ( Lihat Juga Perbedaan antara Windows 8 dan Windows 7 )

Step6 : Ketika diminta untuk berbagi printer, pilih Jangan berbagi printer ini . Tekan Next atau tombol OK . Klik tombol Print halaman pengujian untuk mencetak halaman uji . Klik Finish untuk keluar dari wizard Add Printer. Dalam hal pengujian gagal , kembali ke pengaturan printer dan memperbaikinya . Setelah selesai, keluar jendela . Hapus disk jika itu ada di drive komputer dan reboot komputer. Printer sekarang siap untuk digunakan .

Step7 : Dapat mengatur printer nirkabel serta mengikuti prosedur yang disebutkan di atas sama . Jika pernah menghadapi setup printer atau masalah yang berbeda , hubungi dukungan Windows 8.
