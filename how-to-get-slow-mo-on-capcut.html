6. How much= what price How large/big=what... How fast/slow=what... How high/tall=what How strong/weak=what How long/short=what How deep/shalow=what How hot/cool=what


Subject : English

Category : Question Words

Level : JHS  

Code : 5

Keywords : how

--------

"How" adalah sebuah kata tanya dalam bahasa Inggris.

"How", jika berdiri sendiri, memiliki arti "bagaimana".

"How" digunakan untuk menanyakan keadaan (e.g. How are you?) atau cara (e.g. How do you cook spaghetti?)

"How" jika disandingkan dengan adjective/ adverb artinya berubah menjadi "berapa/ seberapa/ sebagaimana".

Misalnya How often do you come here? (seberapa sering kamu datang ke mari?)

-------

Answers :

How much = what price ⇒ menanyakan harga How large/big = what size ⇒ menanyakan ukuran How fast/slow = what speed ⇒ menanyakan kecepatan How high/tall = what height ⇒ menanyakan ketinggian  How strong/weak = what strength ⇒ menanyakan kekuatan How long/short = what length ⇒ menanyakan panjang  How deep/shallow = what depth ⇒ menanyakan kedalaman  How hot/cool = what temperature ⇒ menanyakan suhu

How large/big = what size?
(large/big = menyatakan ukuran)

How fast/slow = what race/rate?
(fast/slow = menyatakan kecepatan)

How high/tall = what height
(high/tall = menyatakan ketingggian)

How strong/weak = what strength
(strong/weak = menyatakan kekuatan)

How long/short = what lenght
(strong/short = menyatakan panjang)

How deep/shalow = what.....?
(deep/shalow = menyatakan kedalaman, tpi aku ngga tahu bahasa inggrianya, maaf)

How hot/cool = what temperature?

(hot/cool = menyatakan suhu/cuaca/udara. Mungkin ada kata lain yang lebih tepat)
