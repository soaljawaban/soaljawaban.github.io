24. Bayangan Dari titik P(2,1) jika di rotasikan terhadap R[O, -30°] adalah..


Bayangan dari titik P(2,1) jika dirotasikan terhadap R(O, -30°) adalah...Jawaban Pendahuluan

Soal matematika di atas merupakan bagian materi dari transformasi yaitu rotasi atau perputaran suatu objek.

Pembahasan

Rotasi (perputaran) :  Perputaran suatu objek dengan bertumpu pada suatu titik pusat sejauh nilai derajatnya yaitu 0 sampai 360 derajat. Dalam geometri transformasi, rotasi dapat dirumuskan dengan perkalian dan penjumlahan matriks.

Sebagai contoh, A adalah sebuah titik (Ax,Ay), P adalah sebuah titik (Px,Py) sebagai pusat rotasi dengan menggunakan rumus trigonometri (perbandingan sudut segitiga) yaitu sin dan cos, dan α adalah besarnya sudut suatu rotasi yang dinyatakan dengan derajat (°).

A' = ( cos α  sin α )  ( Ax-Px )

      ( -sin α  cos α ) ( Ay-Py )

Nilai (Ax-Px) adalah besar jarak antara titik A dan P secara datar atau horizontal, sedangkan nilai (Ay-Py) adalah besar jarak antara titik A dan P secara tegak atau vertikal.

Jika rotasinya searah jarum jam atau arah kanan, maka besar derajat sudutnya bernilai positif, sedangkan jika rotasinya berlawanan arah jarum jam atau arah kiri, maka besar derajat sudutnya bernilai negatif.

Jika diketahui terdapat titik pusat rotasi (O, α), maka objek dirotasikan terhadapa titik asal atau titik pertemuan di sumbu x dan sumbu y yang berada di koordinat (0,0), sedangkan α adalah besarnya derajat sudut suatu rotasi.


Untuk menyelesaikan soal ini, perlu diketahui bahwa titik R (0,30°) adalah titik asal yaitu (0,0), sehingga jarak titik P (2,1) ke titik R adalah (2,1).

Derajat sudutnya bernilai negatif atau α=-30°, jadi putarannya sejauh 30° berlawanan arah jarum jam.

P' = ( cos 30  sin 30 )  ( Px )

      ( -sin 30  cos 30 ) ( Py )

P' = ( √3/2  1/2 )  ( 2 )

      ( -1/2  √3/2 ) ( 1 )

P' = ( √3/2*2 + 1/2*1 )

      ( -1/2*2 + √3/2*1 )

P' = ( √3 + 1/2 )

      ( -1 + √3/2 )

Jadi hasil rotasi titik P adalah P' (√3+1/2,-1+√3/2).

Kesimpulan

Hasil rotasi titik P (2,1) pada titik R (0,30°) yaitu titik asal (0,0) dengan rotasi sebesar 30° adalah P' (√3+1/2,-1+√3/2).
