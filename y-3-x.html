27. jikax+y/2 - x-y/3 = 8 x+y/3 + x-y/4 = 11maka nilai dari y adalah..​


Jawab:

Mari kita selesaikan sistem persamaan tersebut untuk menemukan nilai [tex]\(y\)[/tex]. Sistem persamaannya adalah:

1. [tex]\(\frac{x+y}{2} - \frac{x-y}{3} = 8\)[/tex]

2. [tex]\(\frac{x+y}{3} + \frac{x-y}{4} = 11\)[/tex]

Kita akan menyelesaikan langkah demi langkah:

**Langkah 1**: Sederhanakan kedua persamaan dengan menghilangkan pecahan.

1. [tex]\(2(x+y) - 3(x-y) = 16\)[/tex] (Kali kedua persamaan pertama dengan 2 dan persamaan kedua dengan 3)

2. [tex]\(4(x+y) + 3(x-y) = 44\)[/tex] (Kali kedua persamaan kedua dengan 4 dan persamaan pertama dengan 3)

**Langkah 2**: Ekspansi.

1. [tex]\(2x + 2y - 3x + 3y = 16\)[/tex]

2. [tex]\(4x + 4y + 3x - 3y = 44\)[/tex]

**Langkah 3**: Gabungkan suku-suku yang sejenis.

1. [tex]\(-x + 5y = 16\)[/tex]

2. [tex]\(7x + y = 44\)[/tex]

**Langkah 4**: Solusi untuk [tex]\(y\)[/tex].

Dari persamaan pertama, kita bisa mengisolasi [tex]\(y\)[/tex]:

[tex]\(5y = x - 16\)[/tex]

[tex]\(y = \frac{x - 16}{5}\)[/tex]

Substitusi hasil ini ke dalam persamaan kedua:

[tex]\(7x + \frac{x - 16}{5} = 44\)[/tex]  

[tex]\(35x + x - 16 = 220\)[/tex]

[tex]\(36x = 236\)[/tex]

[tex]\(x = \frac{236}{36}\)[/tex]

[tex]\(x = \frac{59}{9}\)[/tex]

Substitusi nilai [tex]\(x\)[/tex] ke dalam persamaan untuk [tex]\(y\)[/tex]:

[tex]\(y = \frac{x - 16}{5} = \frac{59/9 - 16}{5} = \frac{-25}{45} = -\frac{5}{9}\)[/tex]

Jadi, nilai dari [tex]\(y\)[/tex] adalah [tex]\(-\frac{5}{9}\)[/tex].
