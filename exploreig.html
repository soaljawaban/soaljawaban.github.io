16. apakah kegunaan windows explorer


Windows Explorer

Explorer dalam perangkat lunak windows merupakan penjelajah file yang menyajikan seluruh file yang tersimpan di dalam komputer pribadi atau PC.

Pembahasan

Windows explorer dapat diartikan sebagai wadah yang menyimpan berbagai berkas di dalamnya, windows explorer terbatas dengan kapasitas tertentu namun ia dapat diatur dengan kehendak yang kita mau.

Komputer atau PC yang menjadi alat utama yang digunakan untuk beragam kebutuhan akan menyimpan segala proyek yang kita kerjakan di dalamnya, dalam hal itulah komputer - komputer dilengkapi dengan software atau perangkat lunak berupa penjelajah file atau yang disebut sebagai windows explorer, tanpa adanya windows explorer berkas atau file yang kita simpan di dalam komputer akan berserakan tidak teratur, semuanya bercampur menjadi satu tanpa ada pemisah yang menggolongkannya.

Cara kerja windows explorer yaitu menampilkan isi atau konten yang ada di dalam komputer dengan cara menggolongkan ke dalam folder - folder. Isi komputer itu baik berupa audio, gambar, video, dokumen, atau bentuk - bentuk file lainnya.

Dari windows explorer itu pula kita dapat melakukan manajemen berkas sehingga teratur, dengan menata letak berkas, memindahkan berkas dari satu folder ke folder lain, melakukan back up dengan membuat salinan berkas, atau bahkan menghapus berkas yang dirasa tidak diperlukan. Windows explorer juga membantu kita untuk melakukan pencarian berkas letak penyimpanannya terlupakan.

36. ikon yang di internet explorer


1.Ikon Back berfungsi untuk kembali ke halaman sebelumnya yang sudah diakses.
2.Ikon Forward berfungsi untuk kembali ke halaman sebelum di Back dan untuk ke halaman berikutnya lagi.
3.Ikon Stop berfungsi untuk menghentikan browser yang sedang mengakses penampilan suatu situs web.
4.Icon Refresh berfungsi untuk mengulangi kembali proses penampilan situs web.
5.Icon Home berfungsi untuk kembali ke halaman awal dari Internet Explorer.
6.Icon Search berfungsi untuk membuka fasilitas area pencarian data atau informasi di Internet.
7.Icon Favorites berfungsi untuk menampilkan daftar alamat internet yang pernah kita simpan sehingga memudahkan kita untuk mengakses kembali.
8.Icon media berfungsi untuk menampilkan jendela WindowsMedia.com yang berisi program music, video, dan film
9.Icon History berfungsi untuk menampilkan daftar alamat yang pernah dikunjungi pada hari ini maupun pada hari sebelumnya.
10.Icon Mail berfungsi untuk mengaktifkan fasilitas Outlook Express dan Internet Explorer.
11.Icon Print berfungsi untuk mencetak halaman situs yang terbuka.
12.Icon Edit with Microsoft Programme (Ms. FontPage, Ms. Excel, Ms. Word, Ms. Wordpad) berfungsi untuk mengedit atau membuat data melalui program Microsoft.
13.Icon Discuss berfungsi untuk menampilkan Discussion Bar pada Internet Explorer (pada Windows XP).

38. Windows Explorer adalah......​


Jawaban:

Sebuah aplikasi perangkat lunak yang menjadi bagian yang tak terpisahkan dari beberapa versi sistem operasi Microsoft Windows yang menyediakan antarmuka grafis kepada pengguna untuk mengakses sistem berkas

Apa itu windows explorer?

Windows Explorer adalah sebuah aplikasi perangkat lunak yang menjadi bagian yang tidak terpisahkan dari beberapa versi sistem operas Microsoft Window.

Aplikasi ini menyediakan antarmuka grafi kepada pengguna untuk mengakses sistem berka (baik itu sistem berkas cakra ataupun sistem berkas jaringan dan mengizinkannya untuk mengontrol komputer.

Windows Explorer merupakan istilah umum yang digunakan untuk menjelaskan aspek manajemen berkas komputer, dijalankan oleh proses dengan nama EXPLORER.EXE. Proses tersebut juga menangani beberapa fungsionalitas lainnya, seperti pencarian (searching), asosiasi jenis ekstensi berkas, dan bertanggung jawab dalam menampilkan desktop, iko, wallpaper, Start Men, taskbar, dan Control Panel Windows itu sendiri sebagai pusat kontrol dan manajemen sistem operasi Windows.

Maka, tidak salah Windows Explorer juga disebut sebagai Windows Shell.

Dengan Windows Explorer, pengguna dapat melakukan hal-hal berikut:

Membuat Folder, memindahkan Folder, menyalin folder, membuka folder, mengedit, atau menghapus berkas dan folder di dalam komputer.

Mencari berkas-berkas dengan menggunakan query yang kompleks

Memetakan sebuah direktori yang terdapat di dalam sebuah host di jaringan ke sebuah drive lokal dan memutuskan pemetaan tersebut.

Melihat dan memanipulasi "properties" atau metadata untuk beberapa berkas dan beberapa objek sistem berkas lainnya.

Mampu menjadikan sebuah direktori dan alat penceta dapat digunakan secara bersama-sama melalui jaringan.
