17. Tentukan 4 suku pertama dari barisan barisan eengan rumus suku ke n 1. Un=n+5 2. Un= (n+1) ² 3. Un=n²+n


4 suku pertama dari barisan dengan rumus suku ke-n

1. 6, 7, 8, dan 9.

2. 4, 9, 16, dan 25.

3. 2, 6, 12, dan 20.

Pendahuluan

Barisan Aritmatika

Barisan aritmatika adalah barisan yang dimana jika selisih dua suku yang berurutan adalah tetap.

Menentukan Suku Ke - n

[tex]Un = a + (n - 1)b[/tex]

Menentukan Jumlah Suku Ke - n

[tex]Sn = \frac{n}{2} × (2a + (n - 1)b[/tex]

atau

[tex]Sn = \frac{n}{2} × (a + Un)[/tex]

Keterangan:

a = Suku pertama

B = Beda barisan

n = Banyak suku

Un = Suku ke - n

Sn = Jumlah suku ke - n

Dari penjelasan diatas, mari kita selesaikan permasalahan tersebut!

Pembahasan

Jawab:

Mencari 4 suku pertama dari barisan dengan menggunakan rumus suku ke-n berikut.

1. Un = n + 5

[tex]U1 = 1 + 5 = 6[/tex]

[tex]U2 = 2+ 5 = 7[/tex]

[tex]U3 = 3 + 5 = 8[/tex]

[tex]U4 = 4+ 5 = 9[/tex]

4 suku pertama : 6, 7, 8, dan 9.

2. Un = (n + 1)²

[tex]U1 = (1 + 1)^{2} = 4[/tex]

[tex]U2 = (2 + 1)^{2} = 9[/tex]

[tex]U3 = (3 + 1)^{2} = 16[/tex]

[tex]U4 = (4 + 1)^{2} = 25[/tex]

4 suku pertama : 4, 9, 16, dan 25.

3. Un = n² + n

[tex]U1 = {1}^{2} + 1 = 2[/tex]

[tex]U2 = {2}^{2} + 2 = 6[/tex]

[tex]U3 = {3}^{2} + 3 = 12[/tex]

[tex]U4 = {4}^{2} + 4 = 20[/tex]

4 suku pertama : 2, 6, 12, dan 20.

Kesimpulan

Jadi, 4 suku pertama dari barisan dengan rumus suku ke-n

1. 6, 7, 8, dan 9.

2. 4, 9, 16, dan 25.

3. 2, 6, 12, dan 20.

20. Cahaya monokromatik eengan panjang gelombang 500 nm datang tegak lurus pada kisi dengan lebar celah 0,0025 mm.tentukan orde maksimum yang mungkin terlihat pada layar


JAWABAN :
 Dari soal dapat diketahui bahwa:
panjang gelombang digunakan --> λ = 500 nm = 500. 10^-9 m
Lebar celah > d =  0,0025 mm = 2,5. 10^-6 m

Ditanyakan orde maksimum yang mungkin terlihat pada layar > n

Jawab:
Jumlah garis difraksi/ orde maksimum yang dapat diamati maksimum mungkin berarti θ = 90°, sehingga:
nλ = d sin θ
n. 500. 10^-9 = 2,5. 10^-6. sin 90
n = 2500/500 = 5

Berdasarkan uraian di atas, didapat jumlah garis difraksi yang dapat diamati maksimum 5 orde.

PEMBAHASAN LEBIH LANJUT :
 
Persamaan umum yang digunakan pada interferensi dan difraksi cahaya yaitu:
nλ = d.p/l
 
atau
nλ = d sin θ
 
keterangan:
n = orde terang/ gelap
λ = panjang gelombang
d = jarak celah
p = jarak antar pola
l = jarak layar ke celah
θ = sudut deviasi
 
A.      Interferensi celah ganda/ Young
Pola terang: n = 1, 2, 3, 4, dst..
Pola gelap: n = ½, 3/2, 5/2, 7/2, dst.
 
B.       Difraksi celah tunggal
Pola terang: n = 3/2, 5/2, 7/2, 9/2, dst.
Pola gelap: n = 1, 2, 3, 4, dst..
 
C.       Difraksi kisi
Pada kisi difraksi terdapat jumlah celah/ goresan tiap meter yang disebut N
N = 1/d

21. Randi ingin membeli sepeda eengan harga 1.500.000.toko memberikan diskon 25% berapa total harga yang dibayarkan randi setalah dilakukan potongan diskon​


Penjelasan dengan langkah-langkah:

[tex] = 1.500.000 - (1.500.000 - 25\%)[/tex]

[tex] = 1.500.000 - 375.000[/tex]

[tex] = 1.125.000[/tex]

Jawaban:

Harga sepeda = 1.500.000

diskon = 25%

Diskon

= 25/100 × 1.500.000

= 375.000

harga akhir = harga awal - diskon

= 1.500.000 - 375.000

= 1.125.000

Maka total harga yang dibayarkan randi setalah dilakukan potongan diskon adalah Rp. 1.125.000
