18. jika luas permukaan bola 90 cm², luas seluruh permukaan tabung adalah??


Luas permukaan tabung adalah 135 cm².

Jawaban C.

Pembahasan

Tabung adalah bangun ruang dengan sisi lengkung, alas dan tutup tabung berbentuk bangun datar lingkaran, adapun selimut tabung berbentuk persegi panjang dengan keliling lingkaran sebagai sisi panjangnya dan tinggi tabung sebagai lebarnya.

Rumus-rumus yang ada pada bangun tabung

1) Luas alas tabung = π x r²

2) Luas selimut tabung = Keliling alas x tinggi

= 2 x π x r x t atau = π x d x t

Keliling alas = panjang selimut

3) Luas permukaan tabung tanpa tutup = luas alas + luas selimut

= π x r² + 2 x π x r x t

= π x r x (r + 2t)

4) Luas permukaan tabung dengan tutup =  luas alas + luas tutup + luas selimut

= π x r² + π x r² + 2 x π x r x t

= 2 x π x r² + 2 x π x r x t

= 2 x π x r x (r + t)

5) Volume tabung = π x r² x t atau luas alas x t

r = jari-jari

d = diameter

t = tinggi

   Materi tentang tabung → Luas selimut tabung dan luas permukaan tabung brainly.co.id/tugas/3731207

Bola adalah bangun ruang 3 dimensi yang terdiri dari beberapa lingkaran yang tak hingga jumlahnya dengan jari-jari yang sama.

Rumus-rumus tentang bangun ruang bola

1) Volume bola = ⁴/₃ x π x r³

2) Luas permukaan bola = 4 x π x r²

3) Luas belahan bola pepat (padat) = 3 x π x r²

Untuk belajar, Simak lebih lanjut di https://brainly.co.id/tugas/8385654

Penyelesaian Soal

Diketahui:

Luas permukaan bola = 90 cm²

Ditanya:

Luas seluruh permukaan tabung

Jawab:

Langkah pertama kita cari panjang jari-jari bola terlebih dahulu.

4 x π x r² = Luas permukaan bola

4 x 3,14 x r² = 90

12,56 x r² = 90

r² = 90 : 12,56

r² = 7,16

r = √7,16

 = 2,68 cm

Langkah kedua, perhatikan gambar, dari gambar dapat disimpulkan bahwa:            

Jari-jari tabung = jari-jari bola

                         = 2,68 cm

Tinggi tabung = 2 x jari-jari bola

                       = 2 x 2,68 cm

                       = 5,36 cm

Langkah terahir, kita cari luas permukaan tabung

Luas permukaan tabung = 2 x π  x r x (r + t )

                                        = 2 x 3,14 x 2,68 x (2,68 + 5,36)

                                        = 2 x 3,14 x 2,68 x 8,04

                                        = 135,31 cm² kita bulatkan menjadi 135 cm²

Luas permukaan tabung adalah 135 cm².

20. Jika Luas permukaan bola = 90,Luas seluruh permukaan tabung adalah ?


[tex]t=d[/tex]

[tex]\text{Rumus luas lingkaran :}\\\pi r^2\\\\\text{Rumus luas permukaan bola :}\\4\pi r^2\\\\\text{Rumus luas permukaan tabung :}\\2\pi r^2+\pi dt[/tex]

[tex]L_{pB}=4\pi r^2\\90=4\pi r^2\\45=2\pi r^2\\\\\pi dt\\\pi d\times d\\\pi d^2\\\\d=2r\\\frac{1}{2}d=r\\\\4\pi r^2=90\\4\pi(\frac{1}{2}d)^2=90\\4\pi(\frac{1}{2}d)(\frac{1}{2}d)=90\\4\pi\frac{1}{4}d^2=90\\\pi d^2=90[/tex]

[tex]L_{pT}=2\pi r^2+\pi dt\\L_{pT}=2\pi r^2+\pi d\times d\\L_{pT}=2\pi r^2+\pi d^2\\L_{pT}=45+90\\L_{pT}=135cm^2[/tex]
