18. Bentuk sederhana dari sin ( 90° + a) + ( 90°-x) =


Dalam trigonometri, besar sudut α dibagi menjadi 4 kelompok, yaitu:

Kuadran I: 0° < α < 90° (0 < α < [tex] \frac{ \pi }{2} [/tex])
sin (90° - α) = cos α
cos (90° - α) = sin α
tan (90° - α) = cotan α
cotan (90° - α) = tan α
cosec (90° - α) = sec α
sec (90° - α) = cosec α

Kuadran II: 90° < α < 180° ([tex] \frac{ \pi }{2} [/tex] < α < π)
sin (180° - α) = sin α
cos (180° - α) = -cos α
tan (180° - α) = -tan α
cotan (180° - α) = -cotan α
cosec (180° - α) = cosec α
sec (180° - α) = -sec α

Kuadran III: 180° < α < 270° (π < α < [tex] \frac{3\pi }{2} [/tex])
sin (270° - α) = -cos α
cos (270° - α) = -sin α
tan (270° - α) = cotan α
cotan (270° - α) = tan α
cosec (270° - α) = -sec α
sec (270° - α) = -cosec α

Kuadran IV: 270° < α < 360° ([tex] \frac{3\pi }{2} [/tex] < α < 2π)
sin (360° - α) = sin (-α) = -sin α
cos (360° - α) = cos (-α) = cos α
tan (360° - α) = tan (-α) = -tan α
cotan (360° - α) = cotan (-α) = -cotan α
cosec (360° - α) = secan (-α) = -cosec α
sec (360° - α) = sec (-α) = sec α

30. Jika sin²3°+sin²6°+sin²9°+....+sin²87°+sin²90°=x Berapakah nilai x???


sin² 3° + sin² 6° + sin² 9° +.... + sin² 87° + sin² 90° = x

Sekarang pasangkan

(sin² 3° + sin² 87°) + (sin² 6° + sin² 84°) + (sin² 9° + sin² 81°) + .... (sin² 38° + sin² 52°) + (sin² 41° + sin² 49°) + (sin² 44° + sin² 46°) + sin² 45° +  sin² 90° = x

Ingat sifat ini :
sin a = cos (90 - a), maka :

(sin² 3° + cos² 3°) + (sin² 6° + cos² 6°) + (sin² 9° + cos² 9°) + .... (sin² 38° + cos² 38°) + (sin² 41° + cos² 41°) + (sin² 44° + cos² 44°) + sin² 45° +  sin² 90° = x

Ingat identitas ini :
sin² x° + cos² x° = 1, maka :

1 + 1 + 1 + ...... (Sebanyak 15 kali) + (½√2)² + 1² = x
15 + ½ + 1 = x
16,5 = x

Jadi, nilai x adalah 16.5
