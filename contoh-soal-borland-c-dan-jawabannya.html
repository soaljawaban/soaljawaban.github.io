10. Mengapa dalam membuat program harus secara manual menggunakan borland c++​


Penjelasan:

ngga semua program harus dibuat secara manual menggunakan Borland C++. Ada banyak pilihan lain seperti menggunakan bahasa pemrograman lain, seperti Python, Java, C#, atau bahkan menggunakan platform pembangunan aplikasi seperti Visual Studio atau Eclipse.

Mungkin aku sebutkan beberapa alasan mengapa beberapa orang memilih untuk menggunakan Borland C++ atau bahasa pemrograman C++ secara umum:

1. Kinerja yang cepat: C++ adalah bahasa pemrograman yang dikenal memiliki kinerja yang sangat cepat, karena ia dikompilasi menjadi kode mesin yang dapat dieksekusi secara langsung oleh komputer. Hal ini membuatnya cocok untuk pengembangan aplikasi yang membutuhkan performa tinggi.

2. Kontrol yang tinggi: C++ memberikan kontrol yang lebih tinggi pada pengembang dalam hal pengaturan memori, manajemen pointer, dan pengoperasian low-level yang memungkinkan pengembang untuk mengoptimalkan kode mereka sesuai kebutuhan.

3. Keterampilan yang berguna: Pengembangan aplikasi dengan C++ dapat membantu pengembang memahami konsep dasar pemrograman, dan meningkatkan kemampuan mereka dalam pemrograman pada umumnya.

Semoga Membantu

Saat ini, ada banyak sekali pilihan bahasa pemrograman dan lingkungan pengembangan perangkat lunak yang tersedia. Sehingga, penggunaan Borland C++ atau bahasa pemrograman tertentu lainnya tergantung pada kebutuhan dan preferensi pengembang perangkat lunak.

Namun, beberapa alasan mengapa beberapa pengembang memilih Borland C++ untuk membuat program secara manual adalah:

Performa tinggi: Borland C++ terkenal karena kecepatan dan performanya yang tinggi, karena compiler dan toolchain yang dikembangkan dengan sangat baik.
Kontrol penuh: Dalam pengembangan program menggunakan bahasa pemrograman tingkat rendah seperti C++, pengembang memiliki kontrol penuh atas implementasi dan kinerja program. Dalam beberapa kasus, ini dapat menghasilkan program yang lebih efisien dan cepat dibandingkan dengan bahasa pemrograman yang lebih tingkat tinggi.

Kemampuan platform cross: Borland C++ dapat digunakan untuk mengembangkan program pada berbagai sistem operasi dan arsitektur hardware, sehingga membuatnya fleksibel dan dapat diandalkan dalam banyak kasus.

Penggunaan legacy code: Beberapa perusahaan masih menggunakan perangkat lunak lama atau legacy code yang ditulis dalam bahasa C++. Maka penggunaan Borland C++ dapat memungkinkan pengembang untuk mempertahankan dan memperbarui kode lama tersebut.

Namun, penting untuk dicatat bahwa Borland C++ juga memiliki beberapa kelemahan, seperti kesulitan dalam pembuatan antarmuka pengguna yang modern dan kurangnya dukungan untuk beberapa fitur yang tersedia di bahasa pemrograman modern lainnya. Oleh karena itu, sebelum memilih Bahasa Pemrograman atau alat pengembangan perangkat lunak tertentu, penting untuk mempertimbangkan kebutuhan proyek dan sumber daya yang tersedia.

34. Cara membuat output angka 5 berbentuk segitiga dengan bahasa C menggunakan borland C++​


Jawaban:

#include <stdio.h>

#include <conio.h>

main(){

int j,i,n;

printf("masukan angka : ");

scanf("%d",&n);

for(i=n;i>=1;i--){

for(j=1;j<=i;j++)

{

printf("%d",j);

}

printf("\n");

}

getch();

}

Penjelasan:

Penjelasan kode diatas :

– baris 1 dan 2 itu untuk memanggil library stdio.h dan conio.h, fungsinya agar dapat menjalankan perintah printf(), scanf(), getch() dll

– baris 5 membuat variabel j, i dan n dengan tipe data int

– baris 6 mencetak kata “masukan angka : ”

– baris 7 meminta inputan berupa angka yang nantinya akan dimasukan kedalam variabel n. karena n bertipe data int maka menggunakan %d, kalau char %c, kalau string %s, kalau float %f dll. untuk variabel didalam scanf() harus ditambah simbol “&” agar tidak error. contoh : &n , seperti kode diatas

– baris 8 itu perintah perulangan pertama yang nantinya kalau variabel i dicetak akan menhasilkan output : n n-1 n-2 sampai ketemu 1

– baris 9 itu perintah perulangan kedua yang nantinya kalau variabel j dicetak akan menhasilkan output : 1 2 3 sampai dengan i, misal n yang diinputkan 5, sudahterjadi 2 perulangan pada perulangan pertama maka nilai i adalah 3 dan nilai j adalah 123

– baris 11 untuk mencetak nilai j

– baris 13 mencetak enter. setelah perulangan kedua dijalankan akan mencetak enter

– baris 15 meminta inputan berupa charakter, tapi dalam program ini, perintah ini dipakai hanya untuk menahan output atau agar outputnya tetap muncul dilayar

Dan ini ss hasil / outputnya

Berikut ini adalah contoh kode untuk membuat sebuah output angka 5 berbentuk segitiga menggunakan bahasa C dengan Borland C++:

#include <iostream>

using namespace std;

int main()

{

for (int i=0; i<5; i++)

{

for (int j=0; j<=i; j++)

{

cout << "5";

}

cout << endl;

}

return 0;

}

Output dari kode tersebut akan seperti berikut:

5

55

555

5555
