2. Jelaskan apa kegunaan dari touch type tool secara jelas dan details


Penjelasan:

1. Selection Tool (V)

Berfungsi untuk memilih atau menyeleksi suatu objek yang dipilih.

2. Direct-Select Tool (A)

Berfungsi untuk menyeleksi sebuah objek lebih dasar lagi dan memilih objek tertentu yang sudah tergabung dalam sebuah grup atau juga bisa memilih titik kurva.

3. Magic Wand Tool (Y)

Berfungsi untuk menyeleksi atau memilih suatu objek lebih dari satu dalam berbentuk vector yang mempunyai atribut yang sama seperti warna , garis , gradient.

5. Lasso Tool (Q)

Berfungsi untuk memilih suatu objek dengan cara mengelilingi objek yang secara bebas dengan sesuai gerakan kursor garis lasso tool.

6.Pen Tool (P)

Berfungsi dalam membuat sebuah garis dengan titik ke titik yang dihasilkan oleh pen tool untuk membentuk garis terbuka dan tertutup yang titik awal dan titik akhirnya menjadi terhubung.


3. Jelaskan apa kegunaan dari Touch Type Tool secara jelas dan detail dengan bahasamu sendiri!​


Jawaban:

1. Selection Tool (V)

Berfungsi untuk memilih atau menyeleksi suatu objek yang dipilih.

2. Direct-Select Tool (A)

Berfungsi untuk menyeleksi sebuah objek lebih dasar lagi dan memilih objek tertentu yang sudah tergabung dalam sebuah grup atau juga bisa memilih titik kurva.

3. Magic Wand Tool (Y)

Berfungsi untuk menyeleksi atau memilih suatu objek lebih dari satu dalam berbentuk vector yang mempunyai atribut yang sama seperti warna , garis , gradient.

5. Lasso Tool (Q)

Berfungsi untuk memilih suatu objek dengan cara mengelilingi objek yang secara bebas dengan sesuai gerakan kursor garis lasso tool.

6.Pen Tool (P)

Berfungsi dalam membuat sebuah garis dengan titik ke titik yang dihasilkan oleh pen tool untuk membentuk garis terbuka dan tertutup yang titik awal dan titik akhirnya menjadi terhubung.

7. Type Tool (T)

Berfungsi untuk membuat text atau mengedit teks yang berbentuk sebuah tulisan.

8. Line segment Tool (\)

Berfungsi dalam membuat sebuah bentuk garis yang lurus.

9. Rectangular Tool (M)

Adalah sebuah tool yang berfungsi untuk membuat sebuah kotak, panjang dan lebar.

10. Paintbrush tool (B)

Merupakan sebuah tool yang berfungsi untuk menggambar dengan menggunakan kuas, untuk gambar yang sudah digambar menghasilkan sebuah gambar yang berbentuk path.

11.Pencil Tool (N)

Adalah tool yang berfungsi dalam menggambar atau membuat garis.

12. Eraser Tool (Shift+E)

Berfungsi untuk menghapus suatu objek yang tidak diinginkan.

13. Rotate Tool

Berfungsi untuk memutar sebuah objek yang dipilih, dngan sembil menekan tombol shift dengan mendapatkan putaran secara 45 derajat.

14. Scale Tool

Berfungsi untuk merubah sebuah ukuran objek dengan cara klik dan menahan salah satu titik kurva lau drag dan setalah itu sesuaikan ukuran sambil menahan shift untuk mendapatkan bentuk gambar yang sama dengan saat proses scale tool.

Beberapa Fungsi-fungsi Tools di Photoshop:

1. Move Tool, Fungsinya untuk memindah objek baik itu berupa teks maupun gambar.

2. Marque Tool

- Rectangular Marquee Tool : untuk membuat area seleksi berupa segi empat dengan sudut 90 derajat.

- Elliptical Marquee Tool : untuk membuat area seleksi berbentuk lingkaran.

- Single Row Marquee Tool : untuk membuat area seleksi berbentuk garis horizontal.

- Single Coloumn Marquee Tool : membuat area seleksi berbentuk garis vertikal.

3. Lasso Tool

Lasso Tool : untuk membuat area seleksi secara bebas sesuai dengan gerakan mouse.

Polygonal Lasso Tool : digunakan untuk membuat area seleksi berbentuk persegi/garis lurus.

Magnetic Lasso Tool : digunakan untuk membuat area seleksi namun akan secara otomatis dilelengketkan pada batas objek berdasarkan perbedaan warnanya.

4. Quick Selection & Magic Wand Tool

Quick Selection Tool : untuk membuat seleksi dengan cepat..

Magic Wand Tool : untuk membuat seleksi pada daerah yang memiliki kesamaan warna namun dapat diatur toleransinya pada options bar.

5. Croping & Slice Tool

Crop Tool : untuk memotong secara persegi dan sejajar

Perspektif Crop Tool : bisa disesuaikan sudutnya, tidak mesti 90 derajat.

Slice Tool : untuk membuat potongan-potongan gambar.

Slice Select Tool : untuk memilih potongan gambar jika kita masih ingin menggesernya.

6. Screen Mode : Untuk merubah tampilan jendela dari tampilan standar, full screen dengan menu, atau full screen.

7. Quick Mask Mode : Untuk mengolah file gambar atau foto ke dalam mode normal dan Quick Mask.

8. Set Foreground & Background : Untuk mengganti warna foreground dan background.

9. Zoom Tool : Untuk meperbesar dan Memperkecil tampilan gambar dalam dokumen kerja.

10. Hand and Rotate View Tool

Hand Tool : Untuk menggeser tampilan gambar dalam dokumen saja.

Rotate View Tool : Untuk memutar tampilan gambar dalam dokumen kerja.
