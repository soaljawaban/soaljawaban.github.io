4. Artinya hunaka maktab


> Isim Isyarah (Kata Tunjuk) <

Hunaka maktab (Disana ada meja)

• Hunaka = disana

• Maktab = meja

Isim Isyarah adalah kata tunjuk. Dalam bahasa arab kata tunjuk dibagi dua. Kata tunjuk untuk jarak dekat dan kata tunjuk untuk jarak jauh.

◇ Kata tunjuk jarak dekat

• Huna (disini)

• Haza (Ini --maskulin)

• Hazihi (Ini --feminin)

◇ Kata tunjuk jarak jauh

• Hunaka (disana)

• Dzalika (Itu --maskulin)

• Tilka (Itu --feminin)

هُنَاكَ مَكْتَبٌ Memiliki arti

Di sana ada Meja
