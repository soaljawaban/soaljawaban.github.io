25. tentukan beda dari barisan aritmatika jika diketahui suku-suku berikut! a. U15 =73,U20=98​


Jawab:

beda dalam barisan tersebut adalah 5

Penjelasan dengan langkah-langkah:

Dik :

U15 = 73

U20 = 98

Dit : b ?

Jawab :

Un = a + (n-1)b

U15 = a + (15-1)b

73 = a + 14b ---->Pers1

U20 = a + (20-1)b

98 = a + 19b ----->Pers2

Eliminasi a pada Pers1 dan Pers2

73 = a + 14b

98 = a + 19b

-------------------  -

-25 = -5b

25/5 = b

b = 5

semoga membantu:)

Jawaban:

• Deret Aritmatika

➖➖➖➖➖➖➖

Diket :

U15 = 73

a + (n - 1) b = 73

a + (15 - 1) b = 73

a + 14b = 73...............(1)

U20 = 98

a + (n - 1) b = 98

a + (20 - 1) b = 98

a + 19b = 98..............(2)

✔ Eliminasi a dari pers (1) dan (2)

a + 14b = 73

a + 19b = 98

------------------- --

-5b = -25

b = 5

✔Substitusi b=5 ke pers (1)

a + 14b = 73

a + 14(5) = 73

a + 70 = 73

a = 73 - 70

a = 3

Jadi, nilai beda dari barisan aritmatika tersebut adalah 5.
