10. Quick12² + 13² + 11² - 10² = Note ; Jatuh Cinta Kembali Lagi Saya ​


Jawaban:

12² + 13² + 11² - 10²

= 144 + 169 + 121 - 100

= 144 + 290 - 100

= 144 + 190

= 334

Hasil dari 12² + 13² + 11² - 10² adalah [tex] \rm \boxed{334}[/tex]

Pendahuluan :

Bilangan berpangkat adalah bilangan dengan hasil dari perkalian berulang yang memiliki bilangan sama.

Bilangan berpangkat bisa disebut juga sebagai bentuk dari sebuah bilangan yang di kalikan yaitu dengan bilangan yang sama dan dilakukan berulang ulang sebanyak pangkatnya.

Bentuk umum Bilangan Berpangkat :

[tex] \boxed{ \rm{ \underbrace{ {a}^{n} = a \times a \times a \times ... \times a}_{sebanyak \: n}}}[/tex]

[tex] \rm{Dengan \: Keterangan : }[/tex]

[tex] \rm{» \: a \: adalah \: basis \: bilangan \: berpangkat.}[/tex][tex] \rm{» \: n \: adalah \: pangkat.}[/tex]

Contoh Soal :

[tex] \rm \: {5}^{3} [/tex]

Maka,

[tex] = \rm \: 5 \times 5 \times 5[/tex]

[tex] \rm \: = 25 \times 5[/tex]

[tex] \rm \: = \boxed{ \rm \: 125}[/tex]

Jadi hasil dari [tex] {5}^{3} [/tex] adalah [tex] \boxed{ \rm \: 125}[/tex]

Pembahasan :

Beberapa sifat perpangkatan yang perlu diketahui.

Sifat - Sifat Perpangkatan ;[tex] \tt{ {a}^{0} = 1}[/tex]

[tex] \tt{ {a}^{m} \times {a}^{n} = {a}^{m + n} }[/tex]

[tex] \tt{ {a}^{m} \div {a}^{n} = {a}^{m - n} }[/tex]

[tex] \tt{ ( {a}^{m} ) {}^{n} = {a}^{mn} }[/tex]

[tex] \tt{ (a \times b) {}^{n} = {a}^{n} \times {b}^{n} }[/tex]

[tex] \tt{ (a \div b) {}^{n} = {a}^{n} \div {b}^{n} }[/tex]

[tex] \tt{ ( {a}^{m} \times {b}^{n} ) {}^{y} = {a}^{my} \times {b}^{ny} }[/tex]

[tex] \tt{ ( {a}^{m} \div {b}^{n} ){}^{y} = {a}^{my} \div {b}^{ny} }[/tex]

[tex] \tt{ {a}^{ - n} = \frac{1}{ {a}^{n} } }[/tex]

[tex] \tt{ {a}^{ \frac{x}{y} } = \sqrt[y]{ {a}^{x} } }[/tex]

Untuk Pengerjaan soal simak pembahasan dibawah ini!

Penyelesaian Soal :

[tex] \rm \: {12}^{2} + {13}^{2} + {11}^{2} - {10}^{2} [/tex]

Untuk menyelesaikan soal tersebut pertama kita tentukan hasil pangkatnya terlebih dahulu.

Menentukan hasil pangkat

[tex] \rm(12 \times 12) + (13 \times 13) + (11 \times 11) - (10 \times 10)[/tex]

[tex] \rm = 144 + 169 + 121 - 100[/tex]

Menentukan hasilnya

[tex] \rm = 144 + 169 + 121 - 100[/tex]

[tex]\rm= 313 + 121 - 100[/tex]

[tex]\rm = 434 - 100[/tex]

[tex] \rm = \boxed{334}[/tex]

Kesimpulan :

Jadi, hasil dari 12² + 13² + 11² - 10² adalah [tex] \rm \boxed{334}[/tex]

Kalimat tersebut mengandung makna untuk memperbaiki dan menyatukan kembali hubungan yang terputus atau rusak dengan cara yang penuh kasih sayang dan damai.
