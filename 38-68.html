8. tentukan suku ke 10 dari barisan 8 16 26 38 68​


Jawaban:

Untuk menentukan suku ke-10 dari barisan 8 16 26 38 68, kita dapat menggunakan rumus umum untuk barisan aritmatika.

Dalam barisan ini, perbedaan antara setiap suku berturut-turut adalah 8, 10, 12, dan seterusnya, yang menunjukkan bahwa kita memiliki suku-suku dengan selisih konstan 2.

Rumus untuk suku ke-n (dalam barisan aritmatika) adalah:

an = a1 + (n - 1) * d

di mana:

- an = suku ke-n

- a1 = suku pertama

- n = posisi suku yang diinginkan

- d = selisih antara setiap suku

Sebagai contoh, pada barisan ini, a1 = 8 dan d = 2.

Mari kita gunakan rumus tersebut untuk mencari suku ke-10 (an):

a10 = 8 + (10 - 1) * 2

= 8 + 18

= 26

Jadi, suku ke-10 dari barisan ini adalah 26.

15. A+B=68A-B=38A:B=...................


Mapel : matematika
Materi : persamaan linear
Kata kunci : persamaan linear dengan dua variabel 
Level soal : level 1, pemahaman
===================================================

                                               بِسْمِ اللهِ الرَّحْمنِ الرَّحِيمِ

→ Dari soal tersebut dapat diperoleh persamaan

A + B = 68............(1)
A - B = 38.............(2)

A - B = 38
⇒ A = 38 + B.........(3)

→ Substitusikan persamaan 3 ke 1

A + B = 68
⇒ 1(38 + B) + B = 68
⇒ 38 + B + B = 68
⇒ 38 + 2B = 68
⇒ 2B = 68 - 38
⇒ 2B = 30
⇒ B = 30 ÷ 2
⇒ B = 15

→ Masukkan nilai B = 15 ke persamaan 1

A + B = 68
⇒ A + 15 = 68
⇒ A = 68 - 15
⇒ A = 53

→ Jadi,

A = 53
B = 15

A ÷ B = 53 ÷ 15 = 3 8/15 atau 3,534
