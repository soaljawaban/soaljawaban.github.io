Jawaban =

Caei masing nilai yang dibutuhkan.

Doyeon = 12^2 = 12 x 12 = 144.

Fantagio = akar169 = 13.

Somi = 19^2 = 19 x 19 = 361.

TBL = akar144 + akar625.

TBL = 12 + 25 = 37.

XOXO = 5^2 = 5 x 5 = 25.

Maka Nilai dari:

1. Doyeon + Somi.

=> 144 + 361 = 505.

2. XOXO.

= 5 x 5 = 25.

3. Fantagio + TBL.

=> 13 + 37 = 50.
