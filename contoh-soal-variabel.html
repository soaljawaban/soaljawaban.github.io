25. contoh contoh soal sistem persamaan dua variabel


Nia membeli 5 pulpen dan 2 pensil dengan harga Rp. 23.000. Sedangkan Anggi membeli 2 pulpen dan 3 pensil dengan harga Rp. 18.000. Maka harga 1 pulpen dan 1 pensil masing masing adalah Rp. 3.000 dan Rp. 4.000.

Penyelesaian Soal :

Misalkan Harga pulpen = x

               Harga pensil = y

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Nia membeli 5 pulpen dan 2 pensil dengan harga Rp. 23.000" yaitu :

5x + 2y = 23.000   ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "Anggi membeli 2 pulpen dan 3 pensil dengan harga Rp. 18.000" yaitu :

2x + 3y = 18.000   ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai x dengan cara :

5x + 2y = 23.000   ║×3║   15x + 6y = 69.000

2x + 3y = 18.000    ║×2║   4x + 6y = 36.000

___________________________________ -

                                                   11x = 33.000

                                                      x =  [tex]\frac{33.000}{11}[/tex]

                                                      x = 3.000

LANGKAH KEEMPAT (IV)

Subtitusikan nilai x pada persamaan 2 sehingga diperoleh nilai y dengan cara :

2x + 3y = 18.000

2 (3.000) + 3y = 18.000

6.000 + 3y = 18.000

3y = 18.000 - 6.000

3y = 12.000

y =  [tex]\frac{12.000}{3}[/tex]

y = 4.000

∴ Kesimpulan harga 1 pulpen dan 1 pensil masing masing adalah Rp. 3.000 dan Rp. 4.000.

36. berikan contoh soal tiga variabel


SISTEM PERSAMAAN LINIER TIGA VARIABEL (SPLTV)

Sistem persamaan linear 3 variabel, merupakan himpunan 3 buah persamaan dengan variabel sebanyak 3. Bentuk ini satu tingkat lebih rumit dibandingkan sistem persamaan linear 2 variabel.

Metoda meyelesaikan persamaan
1. Metoda Eliminasi
2. Metoda subtitusi
3. Metoda determinan
4. Metoda matriks
5. Metoda operasi baris elementer

• Metoda Eliminasi

Contoh soal 1 :

2x + 3y – z = 20
3x + 2y + z = 20
x + 4y + 2z = 15

Jawab :

Ketiga persamaan bisa kita beri nama persamaan (1), (2), dan (3)
2x + 3y – z = 20 ………………………..(1)
3x + 2y + z = 20 ………………………..(2)
x + 4y + 2z = 15 ………………………..(3)

Sistem persamaan ini harus kita sederhanakan menjadi sistem persamaan linear 2 variabel. Untuk itu kita eliminasi variabel z

Sekarang persamaan (1) dan (2) kita jumlahkan
2x + 3y – z = 20
3x + 2y + z = 20_____ +
5x + 5y = 40
x + y = 8 ………………….(4)

Selanjutnya persamaan (2) dikali (2) dan persamaan (3) dikali (1) sehingga diperoleh
6x + 4y + 2z = 40
x + 4y + 2z = 15____ _
5x = 25
x = 5

Nilai x ini kita subtitusi ke persamaan (4) sehingga
x + y = 8
5 + y = 8
y = 3

selanjutnya nilai x dan y yang ada kita subtitusikan ke persamaan (2)
3x + 2y + z = 20
3.5 + 2.3 + z = 20
15 + 6 + z = 20
z = -1

Jadi, himpunan penyelesaiannya adalah {(5, 3, -1)}
